# std/crypto

`std/crypto` 模块为常用加密算法（包括哈希和加密）提供高层 API。

## 使用

```zc
import "std/crypto/sha256.zc"

fn main() {
    let hash = Sha256::hash("hello world");
    println "SHA-256: {hash}";
}
```

## 子模块

`std/crypto` 组织为针对特定算法的子模块：

- **std/crypto/sha256.zc**: SHA-256 哈希算法。
- **std/crypto/aes.zc**: AES (高级加密标准) 加密。
- **std/crypto/rand.zc**: 加密安全的随机数生成器。

## 类型和接口

许多加密子模块遵循通用的增量数据处理模式：

### 哈希接口 (示例: Sha256)

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `new() -> Context` | 初始化新的哈希上下文。 |
| **update** | `update(self, data: u8*, len: usize)` | 向上下文中添加数据。 |
| **finalize** | `finalize(self, out: u8*)` | 完成哈希并写入输出缓冲区。 |
