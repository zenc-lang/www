# std/set

`Set<T>` 是一种存储唯一元素的集合。它内部基于哈希表实现，可提供高效的插入、删除和成员检查操作。

## 使用

```zc
import "std/set.zc"

fn main() {
    let s = Set<int>::new();
    
    s.add(10);
    s.add(20);
    s.add(10); // 不会再次添加
    
    println "大小: {s.length()}"; // 2
    
    if (s.has(10)) {
        println "包含 10";
    }
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Set<T>::new() -> Set<T>` | 创建一个空集合。 |

### 修改

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **add** | `add(self, val: T) -> bool` | 向集合中添加一个元素。如果该值已存在，则返回 false。 |
| **remove** | `remove(self, val: T) -> bool` | 从集合中移除一个元素。 |
| **clear** | `clear(self)` | 移除所有元素。 |

### 访问与属性

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **has** | `has(self, val: T) -> bool` | 如果该值存在于集合中，则返回 true。 |
| **length** | `length(self) -> usize` | 返回集合中的元素数量。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放集合使用的内存。 |
| **Trait** | `impl Drop for Set` | 自动调用 `free()`。 |
