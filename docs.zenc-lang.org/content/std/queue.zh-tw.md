# std/queue

`Queue<T>` 是一個先進先出（FIFO）數據結構，通常用於處理等待處理的緩衝區或任務。

## 使用

```zc
import "std/queue.zc"

fn main() {
    let q = Queue<int>::new();
    
    q.push(1);
    q.push(2);
    
    match q.pop() {
        Some(val) => println "彈出: {val}", // 1
        None => println "隊列為空"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Queue<T>::new() -> Queue<T>` | 創建一個空隊列。 |
| **push** | `push(self, val: T)` | 在隊列末尾添加一個元素。 |
| **pop** | `pop(self) -> Option<T>` | 移除並返回隊列前面的第一個元素。 |
| **peek** | `peek(self) -> Option<T>` | 返回第一個元素的引用而不移除它。 |
| **length** | `length(self) -> usize` | 返回隊列中的元素數量。 |
| **clear** | `clear(self)` | 移除隊列中所有元素。 |
