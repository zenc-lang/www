# std/path

`std/path` 模組提供跨平台的工具，用於處理文件路徑字符串。

## 使用

```zc
import "std/path.zc"

fn main() {
    let p = "/usr/bin/zenc";
    
    println "基本名稱 (Base): {Path::basename(p)}"; // "zenc"
    println "目錄名稱 (Dir): {Path::dirname(p)}";  // "/usr/bin"
    
    let joined = Path::join("home", "user", "docs");
    println "拼接路徑: {joined}";
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **basename** | `Path::basename(p: char*) -> char*` | 返回路徑的最後一部分。 |
| **dirname** | `Path::dirname(p: char*) -> char*` | 返回文件或目錄所在的目錄。 |
| **extname** | `Path::extname(p: char*) -> char*` | 返回文件的擴展名 (例如 ".zc")。 |
| **is_absolute**| `Path::is_absolute(p: char*) -> bool` | 檢查路徑是否為絕對路徑。 |
| **join** | `Path::join(a: char*, b: char*) -> String`| 將兩個路徑片段拼接在一起。 |
