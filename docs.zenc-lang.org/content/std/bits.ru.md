# std/bits

Модуль `std/bits` содержит утилиты для манипуляций с битами в целых типах.

## Использование

```zc
import "std/bits.zc"

fn main() {
    let n = 0b0000_1010;
    
    // Подсчет битов и манипуляции
    let count = Bits::count_ones(n); // 2
    let reversed = Bits::reverse_bits(n);
    
    // Побитовые операции
    if (Bits::is_set(n, 3)) {
        println "Бит 3 установлен!";
    }
}
```

## Функции

### Подсчет и проверка

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **count_ones** | `count_ones(v: u32) -> int` | Возвращает количество установленных битов (`1`). |
| **count_zeros** | `count_zeros(v: u32) -> int` | Возвращает количество неустановленных битов (`0`). |
| **is_set** | `is_set(v: u32, bit: int) -> bool` | Проверяет, установлен ли указанный бит в `1`. |

### Манипуляции

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **set_bit** | `set_bit(v: u32, bit: int) -> u32` | Устанавливает указанный бит в `1`. |
| **clear_bit** | `clear_bit(v: u32, bit: int) -> u32` | Устанавливает указанный бит в `0`. |
| **toggle_bit** | `toggle_bit(v: u32, bit: int) -> u32` | Инвертирует указанный бит. |
| **reverse_bits** | `reverse_bits(v: u32) -> u32` | Разворачивает порядок битов. |

### Разное

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_power_of_two** | `is_power_of_two(v: u32) -> bool` | Возвращает true, если `v` является степенью двойки. |
