# std/bits

Модуль `std/bits` предоставляет низкоуровневые побитовые операции, включая циклические сдвиги, подсчет установленных битов и инверсию битов.

## Обзор

- **Кроссплатформенность**: Правильно реализует побитовые операции для различных архитектур.
- **Эффективность**: Использует оптимизированные алгоритмы манипулирования битами.
- **Комплексность**: Поддерживает 8, 16, 32, 64 и 128-битные целые числа без знака.
- **Утилиты порядка байтов**: Включает функции для изменения порядка байтов (перестановка порядка байтов - endianness swapping).

## Использование

```zc
import "std/bits.zc"

fn main() {
    let a: u32 = 0x80000001;
    
    // Циклический сдвиг 32-битного целого числа влево на 1
    let rotated = Bits::rotl32(a, 1); // 0x00000003
    
    // Подсчет установленных битов (population count)
    let count = Bits::popcount32(0b1011); // 3
    
    // Изменение порядка байтов (bswap)
    let swapped = Bits::bswap32(0x12345678); // 0x78563412
}
```

## Методы

### Побитовый циклический сдвиг

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **rotl[N]** | `Bits::rotl[N](n: u[N], c: u[N]) -> u[N]` | Циклический сдвиг `n` влево на `c` бит (N=8, 16, 32, 64, 128). |
| **rotr[N]** | `Bits::rotr[N](n: u[N], c: u[N]) -> u[N]` | Циклический сдвиг `n` вправо на `c` бит (N=8, 16, 32, 64, 128). |

### Подсчет битов

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **popcount[N]** | `Bits::popcount[N](n: u[N]) -> u[N]` | Возвращает количество установленных битов (единиц) в `n` (N=8, 16, 32, 64, 128). |

### Подсчет ведущих/замыкающих нулей

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **clz[N]** | `Bits::clz[N](n: u[N]) -> u[N]` | Возвращает количество ведущих нулевых битов (N=8, 16, 32, 64, 128). |
| **ctz[N]** | `Bits::ctz[N](n: u[N]) -> u[N]` | Возвращает количество замыкающих нулевых битов (N=8, 16, 32, 64, 128). |

### Перестановка байтов и инверсия битов

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **bswap[N]** | `Bits::bswap[N](n: u[N]) -> u[N]` | Изменяет порядок байтов в `n` (N=16, 32, 64, 128). |
| **reverse_bits[N]** | `Bits::reverse_bits[N](n: u[N]) -> u[N]` | Изменяет порядок битов в `n` (N=8, 16, 32, 64, 128). |
