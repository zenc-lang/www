# std/process

Модуль `std/process` предоставляет высокоуровневый API для создания дочерних процессов, выполнения системных команд и захвата их вывода.

## Обзор

- **Паттерн Builder**: Структура `Command` использует паттерн "текучего" строителя (fluid builder) для создания командных строк.
- **Захват вывода**: Легко захватывайте стандартный вывод и коды завершения завершенных процессов.
- **RAII**: И `Command`, и `Output` реализуют трейт `Drop` для автоматической очистки внутренних буферов.
- **Взаимодействие со стандартами**: Бесшовно оборачивает базовые манипуляции с процессами на системном уровне.

## Использование

```zc
import "std/process.zc"

fn main() {
    // Базовое выполнение команды
    let output = Command::new("echo")
        .arg("hello world")
        .output();
        
    if (output.exit_code == 0) {
        println "Захвачено: {output.std_out}";
        // output.std_out является String, освобождается автоматически
    } else {
        println "Команда завершилась с кодом {output.exit_code}";
    }
}
```

## Определения структур

### `Command`
Строитель для настройки и запуска процесса.
```zc
struct Command {
    program: String;
    args: Vec<String>;
}
```

### `Output`
Результат завершенного выполнения процесса.
```zc
struct Output {
    std_out: String;
    exit_code: int;
}
```

## Методы

### Методы `Command`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Command::new(program: char*) -> Command` | Создает новую команду для заданной программы. |
| **arg** | `arg(self, arg: char*) -> Command*` | Добавляет аргумент и возвращает указатель на self для цепочки вызовов. |
| **output** | `output(self) -> Output` | Выполняет команду и ожидает завершения, захватывая stdout. |
| **status** | `status(self) -> int` | Выполняет команду и возвращает код состояния завершения. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает внутренние буферы команды. |
| **Trait** | `impl Drop for Command` | Автоматически очищает буферы команды. |
| **Trait** | `impl Drop for Output` | Автоматически освобождает строку захваченного вывода. |
