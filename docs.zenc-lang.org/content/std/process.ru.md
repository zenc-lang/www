# std/process

Модуль `std/process` предоставляет высокоуровневый API для запуска дочерних процессов, выполнения системных команд и захвата их вывода.

## Обзор

- **Паттерн Builder**: Структура `Command` использует удобный "текучий" интерфейс для формирования командной строки.
- **Захват вывода**: Простое получение стандартного вывода (stdout) и кодов завершения.
- **RAII**: `Command` и `Output` реализуют трейт `Drop` для автоматической очистки внутренних буферов.
- **Стандартная интероперабельность**: Тонкая обертка над системными вызовами управления процессами.

## Использование

```zc
import "std/process.zc"

fn main() {
    // Базовое выполнение команды
    let output = Command::new("echo")
        .arg("hello world")
        .output();
        
    if (output.exit_code == 0) {
        println "Вывод: {output.std_out}";
        // output.std_out — это String, освобождается автоматически
    } else {
        println "Команда завершилась с кодом {output.exit_code}";
    }
}
```

## Определения структур

### `Command`
Builder для настройки и запуска процесса.
```zc
struct Command {
    program: String;
    args: Vec<String>;
}
```

### `Output`
Результат выполнения завершенного процесса.
```zc
struct Output {
    std_out: String;
    exit_code: int;
}
```

## Методы

### Методы `Command`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Command::new(program: char*) -> Command` | Создает новую команду для программы `program`. |
| **arg** | `arg(self, arg: char*) -> Command*` | Добавляет аргумент и возвращает указатель для цепочки вызовов. |
| **output** | `output(self) -> Output` | Выполняет команду, ждет завершения и захватывает stdout. |
| **status** | `status(self) -> int` | Выполняет команду и возвращает код завершения. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает буферы команды. |
| **Trait** | `impl Drop for Command` | Автоматически очищает буферы команды. |
| **Trait** | `impl Drop for Output` | Автоматически освобождает захваченную строку вывода. |
