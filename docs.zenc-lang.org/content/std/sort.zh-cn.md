# std/sort

`std/sort` 模块提供一系列用于高效对数组和集合进行排序的通用算法。

## 使用

```zc
import "std/sort.zc"

fn main() {
    let arr: int[5] = {5, 2, 9, 1, 5};
    
    // 对数组进行原地排序
    Sort::ints(&arr[0], 5);
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **ints** | `Sort::ints(data: int*, len: usize)` | 对整数数组执行快速排序。 |
| **doubles**| `Sort::doubles(data: double*, len: usize)` | 对双精度浮点数数组进行排序。 |
| **generic**| `Sort::generic(data: T*, len: usize, cmp: fn)` | 配合自定义比较器函数进行排序。 |
