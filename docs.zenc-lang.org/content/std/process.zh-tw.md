# std/process

`std/process` 模組提供與當前進程交互以及催生（spawning）新進程的工具。

## 使用

```zc
import "std/process.zc"

fn main() {
    // 獲取進程 ID
    let pid = Process::id();
    println "進程 ID: {pid}";
    
    // 退出程序
    Process::exit(0);
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **id** | `Process::id() -> int` | 返回當前進程的 PID。 |
| **exit** | `Process::exit(code: int)` | 以指定的狀態碼終止程序。 |
| **spawn** | `Process::spawn(cmd: char*) -> int` | 啟動一個新進程並返回其 PID。 |
| **wait** | `Process::wait(pid: int) -> int` | 等待子進程結束並返回其退出狀態碼。 |
