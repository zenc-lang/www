# std/bigint

`std/bigint` 模組支持任意大小的整數，僅受可用內存限制。這對於加密碼學或大規模科學計算至關重要。

## 使用

```zc
import "std/bigint.zc"

fn main() {
    // 創建兩個大整數
    let a = BigInt::from_string("123456789012345678901234567890");
    let b = BigInt::from_int(100);
    
    // 執行運算
    let res = a.add(b);
    
    println "結果: {res}";
    res.free();
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **from_string** | `BigInt::from_string(s: char*) -> BigInt` | 從字符串創建一個 BigInt。 |
| **from_int** | `BigInt::from_int(v: int) -> BigInt` | 從標準整數創建一個 BigInt。 |
| **add** | `add(self, other: BigInt) -> BigInt` | 將兩個 BigInt 相加並返回結果。 |
| **sub** | `sub(self, other: BigInt) -> BigInt` | 將兩個 BigInt 相減。 |
| **to_string** | `to_string(self) -> String` | 返回數值的字符串表示。 |
