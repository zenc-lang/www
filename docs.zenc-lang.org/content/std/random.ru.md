# std/random

 Модуль `std/random` предоставляет идиоматичную объектно-ориентированную оболочку для генератора псевдослучайных чисел (PRNG) над функциями POSIX `<stdlib.h>`.

## Использование

```zc
import "std/random.zc"

fn main() {
    // Автоматически инициализирует генератор текущим временем
    let rng = Random::new();

    // Генерация случайных целых чисел
    let bounded = rng.next_int_range(1, 100); // от 1 до 100 включительно
    
    println "Выпало: {bounded}";
}
```

## Определение структуры

```zc
struct Random {
    seed: U32;
}
```

## Методы

### Инициализация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Random::new() -> Random` | Создает новый генератор случайных чисел, инициализированный текущим системным временем. |
| **from_seed** | `Random::from_seed(seed: U32) -> Random` | Создает новый генератор случайных чисел, используя конкретное начальное значение (seed). |

### Генерация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **next_int** | `next_int(self) -> int` | Возвращает случайное целое число в диапазоне `[0, RAND_MAX]`. |
| **next_int_range** | `next_int_range(self, min: int, max: int) -> int` | Возвращает случайное целое число в диапазоне `[min, max]` включительно. |
| **next_double** | `next_double(self) -> double` | Возвращает случайное число с плавающей запятой в диапазоне `[0.0, 1.0)`. |
| **next_bool** | `next_bool(self) -> bool` | Возвращает случайное логическое значение. |
