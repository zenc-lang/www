# std/random

Модуль `std/random` предоставляет идиоматичную объектно-ориентированную обертку над генератором псевдослучайных чисел (PRNG) из стандартной библиотеки POSIX (`<stdlib.h>`).

## Использование

```zc
import "std/random.zc"

fn main() {
    // Автоматическая инициализация (seeding) текущим временем
    let rng = Random::new();

    // Генерация случайных целых чисел
    let limited = rng.next_int_range(1, 100); // от 1 до 100 включительно
    
    println "Результат: {limited}";
}
```

## Определение структуры

```zc
struct Random {
    seed: U32;
}
```

## Методы

### Инициализация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Random::new() -> Random` | Создает новый генератор, инициализированный текущим временем системы. |
| **from_seed** | `Random::from_seed(seed: U32) -> Random` | Создает новый генератор с использованием указанного стартового числа (seed). |

### Генерация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **next_int** | `next_int(self) -> int` | Возвращает случайное целое число в диапазоне от `0` до `RAND_MAX`. |
| **next_int_range** | `next_int_range(self, min: int, max: int) -> int` | Возвращает случайное целое число в диапазоне `[min, max]` включительно. |
| **next_double** | `next_double(self) -> double` | Возвращает случайное число с плавающей запятой в диапазоне `[0.0, 1.0)`. |
| **next_bool** | `next_bool(self) -> bool` | Возвращает случайное булево значение. |
