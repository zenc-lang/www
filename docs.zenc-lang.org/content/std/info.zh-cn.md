# std/info

`std/info` 模块提供对当前硬件和运行时环境元数据的访问。

## 使用

```zc
import "std/info.zc"

fn main() {
    println "操作系统: {Info::os_name()}";
    println "架构: {Info::arch()}";
    println "CPU 核心数: {Info::cpu_count()}";
}
```

## 方法

### 操作系统和架构

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **os_name** | `Info::os_name() -> char*` | 返回操作系统的名称（如 "linux", "macos", "windows"）。 |
| **arch** | `Info::arch() -> char*` | 返回 CPU 架构（如 "x86_64", "arm64"）。 |
| **is_debug** | `Info::is_debug() -> bool` | 如果程序是以调试模式编译的，则返回 true。 |

### 硬件资源

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **cpu_count** | `Info::cpu_count() -> int` | 返回可用的逻辑 CPU 核心总数。 |
| **page_size** | `Info::page_size() -> usize` | 返回系统的虚拟内存页面大小。 |
