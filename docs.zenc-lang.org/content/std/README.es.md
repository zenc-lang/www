# Biblioteca Estándar

La Biblioteca Estándar de Zen C proporciona una colección completa de módulos y tipos para tareas comunes, que van desde la manipulación de bits de bajo nivel hasta redes de alto nivel y procesamiento de JSON.

## Módulos

| Módulo | Descripción |
| :--- | :--- |
| **[BigFloat](./bigfloat.md)** | Aritmética de coma flotante de precisión arbitraria. |
| **[BigInt](./bigint.md)** | Aritmética de enteros de precisión arbitraria. |
| **[Bits](./bits.md)** | Operaciones bit a bit de bajo nivel (`rotl`, `rotr`). |
| **[Complex](./complex.md)** | Operaciones con números complejos. |
| **[Crypto](./crypto.md)** | Primitivas criptográficas (SHA1). |
| **[CUDA](./cuda.md)** | Operaciones e interoperabilidad de CUDA GPGPU. |
| **[Encoding](./encoding.md)** | Utilidades de codificación de datos (Base64). |
| **[Env](./env.md)** | Acceso a las variables de entorno del proceso. |
| **[FS](./fs.md)** | E/S del sistema de archivos y operaciones de directorio. |
| **[IO](./io.md)** | Entrada/Salida estándar y formateo. |
| **[Iter](./iter.md)** | Rasgos y protocolos de iteradores personalizados. |
| **[JSON](./json.md)** | Análisis y serialización JSON estilo DOM. |
| **[Map](./map.md)** | Implementación genérica de mapa hash. |
| **[Math](./math.md)** | Constantes y funciones matemáticas. |
| **[Mem](./mem.md)** | Gestión de memoria, asignadores y rasgos. |
| **[Net](./net.md)** | Análisis de TCP, UDP, HTTP, DNS y URL. |
| **[Option](./option.md)** | Envoltorio para valores opcionales (`Some`/`None`). |
| **[Path](./path.md)** | Manipulación de rutas de archivos multiplataforma. |
| **[Process](./process.md)** | Ejecución y gestión de procesos. |
| **[Result](./result.md)** | Patrón de manejo de errores (`Ok`/`Err`). |
| **[Regex](./regex.md)** | Soporte para expresiones regulares. |
| **[Random](./random.md)** | Generador de números pseudoaleatorios. |
| **[SIMD](./simd.md)** | Tipos de vectores optimizados por hardware. |
| **[Queue](./queue.md)** | Implementación de cola FIFO (Ring Buffer). |
| **[Set](./set.md)** | Implementación de conjunto hash genérico. |
| **[Slice](./slice.md)** | Vistas de matrices ligeras sin propiedad. |
| **[Sort](./sort.md)** | Motor de ordenación sin sobrecarga. |
| **[Stack](./stack.md)** | Implementación de pila LIFO. |
| **[String](./string.md)** | Tipo de cadena de caracteres dinámica asignada en el montón. |
| **[Thread](./thread.md)** | Multihilo y sincronización. |
| **[Time](./time.md)** | Utilidades de medición de tiempo y espera. |
| **[UTF-8](./utf8.md)** | Utilidades de manejo de Unicode y UTF-8. |
| **[Vector](./vec.md)** | Tipo de matriz dinámica capaz de crecer. |
