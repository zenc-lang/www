# std/complex

`std/complex` 模組提供複數（具有實部和虛部的數組）的運算。

## 使用

```zc
import "std/complex.zc"

fn main() {
    // 創建一個複數: 3.0 + 4.0i
    let c = Complex::new(3.0, 4.0);
    
    // 計算模長（絕對值）
    let m = c.abs();
    println "模長: {m}";
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Complex::new(re: double, im: double) -> Complex` | 創建一個新複數。 |
| **add** | `add(self, other: Complex) -> Complex` | 返回兩個複數的和。 |
| **sub** | `sub(self, other: Complex) -> Complex` | 返回兩個複數的差。 |
| **mul** | `mul(self, other: Complex) -> Complex` | 返回兩個複數的積。 |
| **abs** | `abs(self) -> double` | 返回複數的模。 |
| **conj** | `conj(self) -> Complex` | 返回共軛複數。 |
