# std/thread

`std/thread` 模块提供跨平台创建和管理轻量级执行线程的工具。

## 使用

```zc
import "std/thread.zc"

fn thread_work(arg: void*) -> void* {
    println "子线程正在运行";
    return null;
}

fn main() {
    // 启动一个新线程
    let t = Thread::spawn(thread_work, null);
    
    // 等待线程完成
    t.join();
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **spawn** | `Thread::spawn(func: fn(void*), arg: void*) -> Thread` | 启动一个新线程。 |
| **join** | `join(self)` | 阻塞当前线程，直到目标线程完成执行。 |
| **detach** | `detach(self)` | 让线程独立运行。 |
| **sleep** | `Thread::sleep(ms: u64)` | 挂起当前线程指定的毫秒数。 |
| **yield** | `Thread::yield()` | 告知调度器放弃当前时间片。 |
