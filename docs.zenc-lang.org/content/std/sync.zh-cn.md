# std/sync

`std/sync` 模块提供原语，用于在多线程环境中管理同步和线程安全地访问共享资源。

## 使用

```zc
import "std/sync/mutex.zc"

fn main() {
    let m = Mutex::new();
    
    m.lock();
    // 临界区
    println "受保护的操作";
    m.unlock();
}
```

## 原语

### `Mutex` (互斥锁)

用于确保一次只有一个线程可以访问一段代码或一块数据。

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Mutex::new() -> Mutex` | 初始化一个新互斥锁。 |
| **lock** | `lock(self)` | 获取锁。如果已被锁定，则阻塞当前线程。 |
| **unlock** | `unlock(self)` | 释放锁。 |
| **try_lock**| `try_lock(self) -> bool` | 尝试获取锁而不进行阻塞。 |

### `CondVar` (条件变量)

用于基于特定条件阻塞线程，直到另一个线程发出信号为止。

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **wait** | `wait(self, mutex: Mutex*)` | 等待信号，同时原子性地释放互斥锁。 |
| **signal** | `signal(self)` | 唤醒一个正在等待的线程。 |
| **broadcast**| `broadcast(self)` | 唤醒所有正在等待的线程。 |
