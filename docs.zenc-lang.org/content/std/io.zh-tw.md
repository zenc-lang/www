# std/io

`std/io` 模組提供用於輸入和輸出操作的基本語法，包括控制台打印和文件流處理。

## 使用

```zc
import "std/io.zc"

fn main() {
    // 打印到標準輸出
    Io::println("你好，世界");
    
    // 從標準輸入讀取一行
    let input = Io::read_line();
    println "你輸入了: {input}";
}
```

## 方法

### 打印

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **print** | `Io::print(s: char*)` | 向標準輸出打印字符串。 |
| **println** | `Io::println(s: char*)` | 向標準輸出打印字符串，後接換行符。 |
| **printf** | `Io::printf(fmt: char*, ...)` | 格式化打印 (POSIX 兼容)。 |

### 讀取

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **read_line** | `Io::read_line() -> String` | 從 stdin 讀取一行。 |
| **read_char** | `Io::read_char() -> char` | 從 stdin 讀取一個字符。 |

## 流 (Streams)
