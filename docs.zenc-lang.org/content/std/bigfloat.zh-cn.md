# std/bigfloat

`BigFloat` 结构体提供任意精度的浮点数算法。与标准 `f32` 或 `f64` 类型不同，`BigFloat` 可以通过动态分配内存来存储非常大或非常精确的数值。

## 使用

```zc
import "std/bigfloat.zc"

fn main() {
    let a = BigFloat::new("1.2345678901234567890123456789");
    let b = BigFloat::new("2.0");
    
    let c = a.mul(b);
    println "结果: {c}";
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `BigFloat::new(s: char*) -> BigFloat` | 从字符串表示创建 BigFloat。 |
| **from_double** | `BigFloat::from_double(v: double) -> BigFloat` | 从标准双精度浮点数创建。 |

### 算术运算

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **add** | `add(self, other: BigFloat) -> BigFloat` | 返回两个 BigFloat 的和。 |
| **sub** | `sub(self, other: BigFloat) -> BigFloat` | 返回两个 BigFloat 的差。 |
| **mul** | `mul(self, other: BigFloat) -> BigFloat` | 返回两个 BigFloat 的积。 |
| **div** | `div(self, other: BigFloat) -> BigFloat` | 返回两个 BigFloat 的商。 |

### 转换

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **to_string** | `to_string(self) -> String` | 返回 BigFloat 的字符串表示。 |
| **to_double** | `to_double(self) -> double` | 将 BigFloat 转换为双精度浮点数（可能会丢失精度）。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放内部分配的内存块。 |
| **Trait** | `impl Drop for BigFloat` | 自动调用 `free()`。 |
