# std/iter

`std/iter` 模块定义了 Zen-C 中用于顺序处理集合的通用迭代器接口。

## 概览

- **统一接口**: 多种集合（`Vec`, `Map`, `String`）都实现了兼容的迭代器。
- **内存安全**: 通过自动边界检查实现。
- **简单语法**: 与 `for..in` 循环配合使用。

## 使用

```zc
import "std/vec.zc"

fn main() {
    let v = Vec<int>::new();
    v.push(1); v.push(2); v.push(3);
    
    // 使用隐藏在底层的使用迭代器的 for..in 循环
    for item in v {
        println "{item}";
    }
}
```

## 迭代器协议

一个迭代器通常提供以下核心方法：

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **next** | `next(self) -> Option<T>` | 返回序列中的下一个元素。如果没有更多元素，则返回 `None`。 |
| **reset** | `reset(self)` | 将迭代器重置回序列开头。 |
