# std/cuda

`std/cuda` 模块为使用 NVIDIA CUDA 平台的 GPU 加速计算提供 Zen-C 接口。

## 概览

- **设备管理**: 枚举和选择 CUDA GPU。
- **内存管理**: 在主机 (CPU) 和设备 (GPU) 之间分配内存和传输数据。
- **内核启动**: 在数千个 GPU 核心上执行并行代码。

## 使用

```zc
import "std/cuda.zc"

fn main() {
    // 初始化设备
    Cuda::init(0);
    
    // 在 GPU 上分配内存
    let d_ptr = Cuda::malloc(1024);
    
    // 将数据传输到 GPU
    let h_data: float[256];
    Cuda::memcpy_to_device(d_ptr, &h_data[0], 1024);
    
    // 配置并启动内核 (伪代码)
    // kernel<<<blocks, threads>>>(d_ptr);
    
    Cuda::free(d_ptr);
}
```

## 方法

### 设备管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **init** | `Cuda::init(device_id: int) -> Result<bool>` | 为指定设备初始化 CUDA 驱动。 |
| **get_device_count**| `Cuda::get_device_count() -> int` | 返回可用的 CUDA 设备数量。 |

### 设备内存

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **malloc** | `Cuda::malloc(size: usize) -> void*` | 在设备上分配线性内存。 |
| **free** | `Cuda::free(ptr: void*)` | 释放设备上分配的内存。 |
| **memcpy_to_device** | `Cuda::memcpy_to_device(dst: void*, src: void*, size: usize)` | 从主机拷贝数据到设备。 |
| **memcpy_to_host** | `Cuda::memcpy_to_host(dst: void*, src: void*, size: usize)` | 从设备拷贝数据到主机。 |
