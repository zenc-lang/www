# std/math

`std/math` 模块提供一系列用于常用计算的数学常数和函数。

## 使用

```zc
import "std/math.zc"

fn main() {
    let x = 16.0;
    println "16 的平方根是 {Math::sqrt(x)}";
    
    let p = Math::pow(2.0, 10.0);
    println "2 的 10 次方是 {p}";
}
```

## 常数

| 键 | 值 | 描述 |
| :--- | :--- | :--- |
| **Math::PI** | 3.1415926535... | 圆周率 |
| **Math::E** | 2.7182818284... | 欧拉数 |

## 方法

### 基本函数

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **abs** | `Math::abs(x: double) -> double` | 返回绝对值。 |
| **sqrt** | `Math::sqrt(x: double) -> double` | 返回平方根。 |
| **pow** | `Math::pow(base: double, exp: double) -> double` | 将底数提升为指数次幂。 |
| **ceil** | `Math::ceil(x: double) -> double` | 向上取整。 |
| **floor** | `Math::floor(x: double) -> double` | 向下取整。 |

### 三角函数

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **sin** | `Math::sin(rad: double) -> double` | 正弦。 |
| **cos** | `Math::cos(rad: double) -> double` | 余弦。 |
| **tan** | `Math::tan(rad: double) -> double` | 正切。 |

### 对数函数

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **log** | `Math::log(x: double) -> double` | 自然对数。 |
| **log10** | `Math::log10(x: double) -> double` | 以 10 为底的对数。 |
