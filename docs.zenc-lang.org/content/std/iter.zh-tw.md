# std/iter

`std/iter` 模組提供用於處理集合序列的工具。它定義了 `Iterator` 特性（Trait），允許對數組、列表和其他數據結構進行惰性處理。

## 使用

```zc
import "std/iter.zc"

fn main() {
    let arr: int[3] = {1, 2, 3};
    let it = Iter::from_array(&arr[0], 3);
    
    match it.next() {
        Some(val) => println "值: {val}",
        None => println "結束"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **next** | `next(self) -> Option<T>` | 返回序列中的下一個項目。 |
| **count** | `count(self) -> usize` | 消耗迭代器並返回項目總數。 |
| **last** | `last(self) -> Option<T>` | 消耗迭代器並返回最後一個元素。 |
| **nth** | `nth(self, n: usize) -> Option<T>`| 返回第 n 個元素。 |

## 實作

任何實現了 `next()` 方法的類型都可以被視為一個迭代器，並與 `for-in` 循環兼容。
