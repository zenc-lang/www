# std/core

`std/core` 模組是 Zen-C 標準庫的基礎。它包含了語言運行時所需的內核定義、內置類型及其基本操作。

## 使用

```zc
import "std/core.zc"

fn main() {
    // 核心定義通常由編譯器自動使用
}
```

## 關鍵定義

| 名稱 | 描述 |
| :--- | :--- |
| **Error** | 基本錯誤枚舉。 |
| **Result**| 代表成功或失敗的泛型包裝器。 |
| **Option**| 代表一個可選值（有或無）。 |
| **Size** | 內存大小的定義（usize）。 |
