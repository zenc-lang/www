# std/core

`std/core` 模組提供了 Zen-C 程式最基本的定義和錯誤處理原語。大多數其他標準庫都會隱式地要求使用它。

## 概覽

- **標準標頭檔**：包含關鍵的 C 標頭檔，如 `stdlib.h`、`stdio.h` 和 `stdbool.h`。
- **恐慌控制（Panic Mechanism）**：提供 `panic` 宏用於不可恢復的錯誤報告。
- **行程控制**：包含基本的行程控制函數，如 `exit`。

## 方法

### 錯誤處理

| 方法/宏 | 簽名 | 說明 |
| :--- | :--- | :--- |
| **panic** | `panic(msg: char*)` | 列印格式化的錯誤訊息，包括檔案、行號和函數名稱，然後終止行程。 |

### 行程控制

| 方法 | 簽名 | 說明 |
| :--- | :--- | :--- |
| **exit** | `exit(code: int)` | 使用指定的返回代碼立即終止行程。 |
