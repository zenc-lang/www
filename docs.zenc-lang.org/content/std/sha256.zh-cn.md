# std/crypto/sha256

`std/crypto/sha256` 模块实现了 FIPS 180-4 规范定义的 SHA-256 加密哈希算法。

## 概览

- **哈希运算**：计算任意字符串或字节数据的 256 位哈希值。
- **增量哈希**：使用 `update` 和 `finalize` 分块处理数据。
- **十六进制输出**：提供便捷的 `hash` 辅助函数，返回十六进制字符串。

## 使用方法

```zc
import "std/crypto/sha256.zc"
import "std/io.zc"

fn main() {
    // 简单哈希
    let hash = Sha256::hash("hello world");
    println "SHA-256: {hash}";
    
    // 增量哈希
    let ctx = Sha256::new();
    ctx.update((u8*)"hello", 5);
    ctx.update((u8*)" world", 6);
    
    let digest: u8[32];
    ctx.finalize(&digest[0]);
}
```

## 结构体定义

```zc
struct Sha256 {
    state: u32[8];
    buffer: u8[64];
    datalen: u32;
    bitlen: u64;
}
```

## 方法

### `Sha256` 方法

| 方法 | 签名 | 说明 |
| :--- | :--- | :--- |
| **new** | `Sha256::new() -> Sha256` | 初始化一个新的 SHA-256 上下文。 |
| **update** | `update(self, data: u8*, len: usize)` | 使用一块数据更新上下文。 |
| **finalize** | `finalize(self, hash: u8*)` | 结束哈希运算并将结果写入提供的 32 字节缓冲区中。 |
| **hash** | `Sha256::hash(data: char*) -> String" | 将字符串进行哈希运算并返回其十六进制表示形式的便捷方法。 |
