# std/crypto/sha256

`std/crypto/sha256` 模块提供 SHA-256 哈希算法的实现。

## 使用

### 一次性哈希

```zc
import "std/crypto/sha256.zc"

fn main() {
    let hash = Sha256::hash("hello");
    println "哈希值: {hash}";
}
```

### 增量哈希

对于大型数据流，请使用上下文：

```zc
import "std/crypto/sha256.zc"

fn main() {
    let ctx = Sha256::new();
    ctx.update("第一部分", 4);
    ctx.update("第二部分", 4);
    
    let result: u8[32];
    ctx.finalize(&result[0]);
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **hash** | `Sha256::hash(data: char*) -> String` | 计算并返回十六进制编码的哈希。 |
| **new** | `Sha256::new() -> Sha256Context` | 创建一个新的哈希上下文。 |
| **update** | `update(self, data: u8*, len: usize)` | 向上下文中添加数据。 |
| **finalize** | `finalize(self, out: u8*)` | 完成计算并将 32 字节写入 `out`。 |
