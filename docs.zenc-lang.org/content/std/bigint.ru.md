# std/bigint

`BigInt` обеспечивает целочисленную арифметику произвольной точности для Zen-C. Это позволяет выполнять вычисления с целыми числами, превышающими емкость стандартных числовых типов, таких как `u64`.

## Обзор

- **Произвольная точность**: Числа ограничены только доступной памятью.
- **На основе десятичной системы**: В настоящее время использует простое представление по основанию 10 для простоты.
- **RAII**: Реализует трейт `Drop` для автоматического управления памятью внутреннего хранилища цифр.
- **Удобство**: Поддерживает перегрузку операторов для интуитивно понятной арифметики.

## Использование

```zc
import "std/bigint.zc"

fn main() {
    let a = BigInt::from_int(1_000_000_000_000_000);
    let b = BigInt::from_int(2_000_000_000_000_000);
    
    // Использует перегрузку операторов
    let sum = a + b; 
    
    let s = sum.to_string();
    println "Сумма: {s}";
    free(s);
} // sum, a и b освобождаются здесь автоматически
```

## Определение структуры

struct BigInt {

}

## Методы

### Конструкторы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `BigInt::new() -> BigInt` | Создает новый `BigInt`, инициализированный нулем. |
| **from_int** | `BigInt::from_int(val: u64) -> BigInt` | Создает новый `BigInt` из 64-битного целого числа. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **add_in_place** | `add_in_place(self, other: BigInt)` | Добавляет `other` к `self`, изменяя внутреннее состояние. |

### Утилиты

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **clone** | `clone(self) -> BigInt` | Возвращает глубокую копию `BigInt`. |
| **to_string** | `to_string(self) -> char*` | Возвращает строковое представление, выделенное в куче. |

## Операторы

| Оператор | Метод | Описание |
| :--- | :--- | :--- |
| `+` | **add** | Возвращает новый `BigInt`, содержащий сумму двух значений. |
| `{}` | **to_string** | Автоматически включает интерполяцию в отформатированных строках. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free_mem** | `free_mem(self)` | Вручную освобождает подлежащее хранилище `Vec` и `BigInt`. |
| **Trait** | `impl Drop for BigInt` | Автоматически вызывает `free_mem()`, когда объект выходит за пределы области видимости. |
