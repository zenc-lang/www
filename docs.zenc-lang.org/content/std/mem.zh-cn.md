# std/mem

`std/mem` 模块提供基本的内存操作工具，用于高效地操作原始字节缓冲区。

## 使用

```zc
import "std/mem.zc"

fn main() {
    let src: u8[4] = {1, 2, 3, 4};
    let dst: u8[4];
    
    // 拷贝内存
    Mem::copy(&dst[0], &src[0], 4);
    
    // 设置内存为 0 (清空)
    Mem::set(&dst[0], 0, 4);
    
    // 比较内存
    if (Mem::eq(&dst[0], &src[0], 4)) {
        println "相等";
    }
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **copy** | `Mem::copy(dst: void*, src: void*, len: usize)` | 将 `len` 字节从 `src` 拷贝到 `dst`。 |
| **move** | `Mem::move(dst: void*, src: void*, len: usize)` | 将字节从 `src` 移动到 `dst`（支持区域重叠）。 |
| **set** | `Mem::set(dst: void*, val: int, len: usize)` | 将 `dst` 的前 `len` 个字节填充为 `val`。 |
| **eq** | `Mem::eq(a: void*, b: void*, len: usize) -> bool` | 如果内存的前 `len` 个字节相等，则返回 true。 |

## 内存分配

虽然大多数 Zen-C 类型使用 `Drop` 进行自动管理，但你可以通过以下方式使用原始系统分配：

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **alloc** | `Mem::alloc(size: usize) -> void*` | 分配原始字节块。 |
| **realloc**| `Mem::realloc(ptr: void*, size: usize) -> void*` | 调整现有分配的大小。 |
| **free** | `Mem::free(ptr: void*)` | 释放先前分配的原始内存。 |
