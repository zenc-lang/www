# std/vec

`Vec<T>` — это обобщенный динамический массив, который автоматически расширяется в памяти при добавлении новых элементов. Аналог `std::vector` в C++ или `ArrayList` в Java.

## Использование

```zc
import "std/vec.zc"

fn main() {
    let v = Vec<int>::new();
    
    v.push(10);
    v.push(20);
    v.push(30);
    
    println "Размер: {v.length()}";
    println "Второй элемент: {v.get(1)}";
    
    v.remove(0); // Удаляет 10
} // v освобождается автоматически через Drop
走
```

## Детали реализации

- **Конкретные специализации**: Для эффективности `Vec` имеет оптимизированные реализации для базовых типов (например, `Vec<char>`).
- **Стратегия роста**: Емкость удваивается при заполнении, что обеспечивает амортизированную сложность **O(1)** для вставки в конец.
- **Трейт Drop**: Автоматически освобождает выделенную память кучи при выходе из области видимости.
- **Безопасность**: Методы доступа выполняют проверку границ (bounds checking).

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Vec<T>::new() -> Vec<T>` | Создает пустой вектор. |
| **with_capacity**| `Vec<T>::with_capacity(cap: usize) -> Vec<T>` | Создает пустой вектор с зарезервированной памятью. |
| **clone** | `clone(self) -> Vec<T>` | Создает глубокую копию вектора и его элементов. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **push** | `push(self, value: T)` | Добавляет элемент в конец. |
| **pop** | `pop(self) -> Option<T>` | Удаляет и возвращает последний элемент. |
| **insert** | `insert(self, idx: usize, value: T)` | Вставляет элемент по индексу (O(N)). |
| **remove** | `remove(self, idx: usize) -> T` | Удаляет элемент по индексу и возвращает его (O(N)). |
| **clear** | `clear(self)` | Удаляет все элементы, не освобождая выделенный буфер. |

### Доступ и свойства

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **get** | `get(self, idx: usize) -> T` | Возвращает значение по индексу. Паникует при выходе за границы. |
| **get_ptr** | `get_ptr(self, idx: usize) -> T*` | Возвращает указатель на элемент по индексу. |
| **length** | `length(self) -> usize` | Возвращает текущее количество элементов. |
| **capacity** | `capacity(self) -> usize` | Возвращает текущую зарезервированную емкость. |
| **as_slice** | `as_slice(self) -> Slice<T>` | Возвращает срез (`Slice`), ссылающийся на данные вектора. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает внутренний буфер в куче. |
| **Trait** | `impl Drop for Vec` | Автоматически вызывает `free()`, когда вектор выходит из области видимости. |
