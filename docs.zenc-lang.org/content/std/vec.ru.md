# std/vec

`Vec<T>` — это тип непрерывного расширяемого массива. Это стандартный динамический массив, используемый в Zen-C.

## Обзор

- **Универсальность**: Работает с любым типом `T`.
- **Динамичность**: Автоматически изменяет размер при добавлении элементов.
- **Безопасность**: Проверка границ при доступе (вызывает панику при ошибке).
- **RAII**: Автоматически освобождает память при выходе из области видимости (реализует `Drop`).

## Использование

```zc
import "std/vec.zc"

fn main() {
    let v = Vec<int>::new();
    v.push(10);
    v.push(20);
    
    // Итерация
    for x in &v {
        println "{(*x)}";
    }
} // v освобождается здесь автоматически
```

## Определение структуры

```zc
struct Vec<T> {
    data: T*;
    len: usize;
    cap: usize;
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Vec<T>::new() -> Vec<T>` | Создает новый пустой вектор. Не выделяет память до первого добавления элемента. |
| **with_capacity** | `Vec<T>::with_capacity(cap: usize) -> Vec<T>` | Создает новый вектор с начальной емкостью `cap`. Полезно для оптимизации, если количество элементов известно заранее. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **push** | `push(self, item: T)` | Добавляет элемент в конец. Вызывает панику, если выделение памяти не удалось. |
| **pop** | `pop(self) -> T` | Удаляет последний элемент и возвращает его. Вызывает панику, если вектор пуст. |
| **pop_opt** | `pop_opt(self) -> Option<T>` | Удаляет последний элемент и возвращает `Some(val)`. Возвращает `None`, если вектор пуст. Безопасное использование. |
| **insert** | `insert(self, idx: usize, item: T)` | Вставляет элемент по индексу `idx`. Сдвигает элементы вправо. Паникует, если `idx > len`. |
| **remove** | `remove(self, idx: usize) -> T` | Удаляет и возвращает элемент по индексу `idx`. Сдвигает элементы влево. Паникует, если `idx >= len`. |
| **append** | `append(self, other: Vec<T>)` | Добавляет все элементы из `other` в `self`. Потребляет `other` (семантика перемещения). |
| **clear** | `clear(self)` | Удаляет все значения. Не влияет на выделенную емкость. |
| **reverse** | `reverse(self)` | Меняет порядок элементов на обратный на месте. |

### Доступ

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **get** | `get(self, idx: usize) -> T` | Возвращает копию элемента по индексу `idx`. Паникует при выходе за границы. |
| **get_ref** | `get_ref(self, idx: usize) -> T*` | Возвращает указатель на элемент по индексу `idx`. Паникует при выходе за границы. Полезно для избежания копирования. |
| **set** | `set(self, idx: usize, item: T)` | Перезаписывает элемент по индексу `idx`. Паникует при выходе за границы. |
| **first** | `first(self) -> T` | Возвращает копию первого элемента. Паникует, если вектор пуст. |
| **last** | `last(self) -> T` | Возвращает копию последнего элемента. Паникует, если вектор пуст. |

### Утилиты

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **length** | `length(self) -> usize` | Возвращает количество элементов. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает `true`, если вектор не содержит элементов. |
| **contains** | `contains(self, item: T) -> bool` | Возвращает `true`, если вектор содержит элемент, равный `item` (побайтово). |
| **clone** | `clone(self) -> Vec<T>` | Возвращает новый вектор с глубокой копией данных. |
| **eq** | `eq(self, other: Vec<T>*) -> bool` | Возвращает `true`, если два вектора равны побайтово. Принимает указатель, чтобы избежать перемещения `other`. |

### Операторы

Zen-C поддерживает перегрузку операторов. `Vec<T>` реализует следующее:

| Оператор | Метод | Описание |
| :--- | :--- | :--- |
| `+` | **add** | `v1 + &v2`. Возвращает новый вектор (конкатенация). |
| `+=` | **add_assign** | `v1 += &v2`. Добавляет `v2` к `v1`. |
| `==` | **eq** | `v1 == &v2`. Проверка структурного равенства. |
| `!=` | **neq** | `v1 != &v2`. Проверка структурного неравенства. |
| `<<` | **shl** | `v << item`. Добавляет `item` в конец. |
| `>>` | **shr** | `v >> &item`. Извлекает последний элемент в `item`. |
| `*` | **mul** | `v * n`. Возвращает новый вектор с элементами, повторенными `n` раз. |
| `*=` | **mul_assign** | `v *= n`. Повторяет элементы на месте `n` раз. |
| `[]` | **get** / **set** | `v[i]` и `v[i] = x`. Стандартная индексация. |

### Итерация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **iterator** | `iterator(self) -> VecIter<T>` | Возвращает итератор, выдающий копии. Используется в `for x in v`. |
| **iter_ref** | `iter_ref(self) -> VecIterRef<T>` | Возвращает итератор, выдающий указатели. Используется в `for x in &v` или `for x in v.iter_ref()`. Позволяет изменять элементы на месте. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **Free** | `free(self)` | Вручную освобождает память. Безопасно вызывать несколько раз. |
| **Forget** | `forget(self)` | Отсоединяет буфер памяти от вектора (обнуляет поля). Предотвращает освобождение памяти через `Drop`. Полезно для реализации семантики перемещения или передачи владения. |
| **Trait** | `impl Drop for Vec" | Автоматически вызывает `free()`, когда `Vec` выходит из области видимости. |
