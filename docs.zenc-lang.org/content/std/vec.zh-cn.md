# std/vec

`Vec<T>` 是一种可动态调整大小的连续数组。它允许在末尾高效地添加和移除元素。

## 使用

```zc
import "std/vec.zc"

fn main() {
    // 创建一个整数向量
    let v = Vec<int>::new();
    
    v.push(10);
    v.push(20);
    v.push(30);
    
    println "元素 [1]: {v.at(1)}"; // 20
    
    v.pop(); // 移除 30
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Vec<T>::new() -> Vec<T>` | 创建一个初始容量为 0 的空向量。 |
| **with_capacity**| `Vec<T>::with_capacity(n: usize) -> Vec<T>` | 创建具有预分配内存的空向量。 |

### 修改

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **push** | `push(self, val: T)` | 在末尾添加一个元素。如果需要，会导致重新分配内存。 |
| **pop** | `pop(self) -> Option<T>` | 移除并返回最后一个元素。 |
| **insert** | `insert(self, index: usize, val: T)` | 在指定位置插入元素并移动后续元素。 |
| **remove** | `remove(self, index: usize)` | 移除并返回指定位置的元素。 |

### 访问与属性

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **at** | `at(self, index: usize) -> T` | 返回对指定索引处元素的引用。 |
| **length** | `length(self) -> usize` | 返回元素数量。 |
| **capacity** | `capacity(self) -> usize` | 返回当前分配的内存能够容纳的元素数量。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放分配的内存。 |
| **Trait** | `impl Drop for Vec` | 自动调用 `free()`。 |
