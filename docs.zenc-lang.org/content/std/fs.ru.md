# std/fs

Модуль `std/fs` предоставляет комплексный кроссплатформенный интерфейс для задач манипуляции файлами и директориями.

## Обзор

- **Обработка файлов**: Открытие, чтение, запись и закрытие файлов.
- **Управление директориями**: Создание, удаление и листинг содержимого директорий.
- **RAII**: Типы `File` и `Dir` реализуют trait `Drop`, гарантируя автоматическое закрытие дескрипторов файлов.
- **Обработка ошибок**: Предоставляет подробные коды ошибок через тип `Result`.

## Использование

```zc
import "std/fs.zc"

fn main() {
    // Чтение всего контента файла
    match File::read_to_string("app.config") {
        Ok(content) => println "Конфигурация: {content}",
        Err(e)      => println "Ошибка чтения файла: {e}"
    }

    // Запись в файл
    let f = File::create("output.txt").unwrap();
    f.write_string("Привет, Zen-C!");
} // f закрывается автоматически здесь
```

## Методы файла (`File`)

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **open** | `File::open(path: char*) -> Result<File>` | Открывает существующий файл в режиме чтения. |
| **create** | `File::create(path: char*) -> Result<File>` | Создает новый файл или обрезает (truncates) существующий. |

### Чтение и запись

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **read** | `read(self, buf: char*, size: usize) -> Result<usize>` | Читает до `size` байт в буфер. |
| **write** | `write(self, buf: char*, size: usize) -> Result<usize>` | Записывает `size` байт из буфера. |
| **write_string** | `write_string(self, s: char*) -> Result<usize>` | Записывает C-строку в файл. |
| **read_to_string**| `File::read_to_string(path: char*) -> Result<String>` | Статическая утилита для чтения всего файла целиком. |

## Методы директории (`Dir`)

### Управление

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **mkdir** | `Dir::mkdir(path: char*) -> Result<bool>` | Создает новую директорию. |
| **rmdir** | `Dir::rmdir(path: char*) -> Result<bool>` | Удаляет пустую директорию. |
| **remove** | `Dir::remove(path: char*) -> Result<bool>` | Удаляет файл. |
| **exists** | `Dir::exists(path: char*) -> bool` | Проверяет, существует ли файл или директория. |

### Листинг

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **list** | `Dir::list(path: char*) -> Result<Vec<String>>` | Возвращает имена всех элементов внутри директории. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **close** | `close(self)` | Вручную закрывает дескриптор файла или директории. |
| **Trait** | `impl Drop for File` | Автоматически закрывает файл при выходе из области видимости. |
