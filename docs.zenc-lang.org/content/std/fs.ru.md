# std/fs

Модуль `std/fs` предоставляет комплексный API для взаимодействия с файловой системой, включая ввод-вывод файлов, манипулирование директориями и получение метаданных.

## Обзор

- **Безопасные дескрипторы**: Структура `File` предоставляет безопасную оболочку над сырыми дескрипторами файлов.
- **RAII**: Дескрипторы файлов автоматически закрываются при выходе из области видимости с помощью трейта `Drop`.
- **Обработка ошибок**: Использует `Result<T>` для всех операций, которые могут завершиться ошибкой, предоставляя описательные сообщения об ошибках.
- **Удобство**: Включает статические методы для распространенных задач, таких как чтение или запись всего файла за один вызов.

## Использование

```zc
import "std/fs.zc"

fn main() {
    // Базовое чтение файла с использованием RAII
    match File::read_all("config.txt") {
        Ok(content) => println "Конфигурация: {content}",
        Err(e) => println "Ошибка чтения конфигурации: {e}"
    }
    
    // Явный дескриптор файла с автоматическим закрытием
    match File::open("data.log", "a") {
        Ok(file) => {
            file.write_string("Запись лога\n");
            // файл закрывается автоматически здесь
        }
        Err(e) => println "Не удалось открыть лог: {e}"
    }
}
```

## Определения структур

### `File`
Представляет дескриптор открытого файла.
```zc
struct File {
    handle: void*;
}
```

### `Metadata`
Метаданные файла или директории.
```zc
struct Metadata {
    size: U64;
    is_dir: bool;
    is_file: bool;
}
```

### `DirEntry`
Представляет запись в директории.
```zc
struct DirEntry {
    name: String;
    is_dir: bool;
}
```

## Методы

### Открытие / Закрытие

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **open** | `File::open(path: char*, mode: char*) -> Result<File>` | Открывает файл по пути `path` в режиме `mode`. |
| **close** | `close(self)` | Явно закрывает дескриптор файла. |

### Чтение / Запись

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **read_to_string** | `read_to_string(self) -> Result<String>` | Читает весь файл в `String`. |
| **read_all** | `File::read_all(path: char*) -> Result<String>` | Статическая утилита для полного чтения файла. |
| **read_lines** | `File::read_lines(path: char*) -> Result<Vec<String>>` | Статическая утилита для чтения файла в вектор строк. |
| **write_string** | `write_string(self, content: char*) -> Result<bool>` | Записывает строку в файл. |
| **write_lines** | `File::write_lines(path: char*, lines: Vec<String>*) -> Result<bool>` | Статическая утилита для записи вектора строк в файл. |

### Утилиты пути

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **exists** | `File::exists(path: char*) -> bool` | Возвращает true, если путь существует. |
| **current_dir** | `File::current_dir() -> Result<String>` | Возвращает абсолютный путь к текущей рабочей директории. |
| **metadata** | `File::metadata(path: char*) -> Result<Metadata>` | Получает метаданные для указанного пути. |

### Операции с файлами и директориями

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **create_dir** | `File::create_dir(path: char*) -> Result<bool>` | Создает новую директорию. |
| **remove_file** | `File::remove_file(path: char*) -> Result<bool>` | Удаляет указанный файл. |
| **remove_dir** | `File::remove_dir(path: char*) -> Result<bool>` | Удаляет указанную директорию (должна быть пустой). |
| **read_dir** | `File::read_dir(path: char*) -> Result<Vec<DirEntry>>` | Возвращает список записей в директории. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **Trait** | `impl Drop for File` | Автоматически закрывает дескриптор файла при выходе из области видимости. |
