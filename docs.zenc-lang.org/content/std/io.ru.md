# std/io

Модуль `std/io` предоставляет функции стандартного ввода-вывода, включая форматированный вывод в stdout и надежное чтение из stdin.

## Обзор

- **Форматированный вывод**: Предоставляет `print` и `println` с поддержкой спецификаторов формата в стиле C (`%s`, `%d` и т. д.).
- **Форматирование строк**: Несколько вариантов форматирования в статические, предоставленные пользователем или выделенные в куче буферы.
- **Поддержка Unicode**: Включает `read_rune` для чтения отдельных символов UTF-8 из stdin.
- **Утилиты преобразования**: Простые методы для преобразования целых чисел и рун в строки.

## Использование

```zc
import "std/io.zc"

fn main() {
    // Базовый вывод
    println("Hello, %s!", "Zen-C");
    
    // Чтение строки ввода
    print("Введите ваше имя: ");
    autofree let name = readln();
    
    if name != NULL {
        println("Приветствие, %s", name);
    }
}
```

## Методы

### Вывод

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **print** | `print(fmt: char*, ...) -> int` | Выводит форматированный вывод в stdout. |
| **println** | `println(fmt: char*, ...) -> int` | Выводит форматированный вывод в stdout, за которым следует перенос строки. |

### Ввод

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **readln** | `readln() -> char*` | Читает строку из stdin. Возвращает строку, выделенную в куче (вызывающий должен ее освободить). |
| **read_rune** | `read_rune() -> rune` | Читает один символ UTF-8 (руну) из stdin. |

### Форматирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **format** | `format(fmt: char*, ...) -> char*` | Форматирует во внутренний статический буфер. **Предупреждение**: Не потокобезопасно. |
| **format_into** | `format_into(buf: char*, size: usize, fmt: char*, ...) -> int` | Форматирует в предоставленный пользователем буфер определенного размера. |
| **format_new** | `format_new(fmt: char*, ...) -> char*` | Форматирует в новый буфер, выделенный в куче. Вызывающий должен его освободить. |

### Преобразование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **itos** | `itos(n: int) -> char*` | Преобразует `n` в строку в статическом буфере. |
| **itos_new** | `itos_new(n: int) -> char*` | Преобразует `n` в строку, выделенную в куче. |
| **utos** | `utos(n: uint) -> char*` | Преобразует беззнаковое `n` в строку в статическом буфере. |
