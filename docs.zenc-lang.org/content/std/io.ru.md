# std/io

Модуль `std/io` предоставляет базовые функции ввода-вывода, включая печать в консоль и чтение из стандартного потока ввода.

## Использование

```zc
import "std/io.zc"

fn main() {
    // Простая печать
    print "Привет, ";
    println "Мир!"; // Добавляет новую строку

    // С форматированием
    let user = "Dev";
    println "Добро пожаловать, {user}!"; 
}
```

## Функции вывода

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **print** | `print s: char*` | Печатает строку в стандартный поток вывода (stdout). |
| **println** | `println s: char*` | Печатает строку в стандартный поток вывода с символом новой строки. |
| **printf** | `printf(fmt: char*, ...args)` | Форматированный вывод в стиле C. |

## Функции ввода

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **readln** | `readln() -> String` | Читает строку из стандартного потока ввода (stdin) как `String`. |

## Потоки ошибок

Для печати в стандартный поток ошибок (stderr):

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **eprint** | `eprint s: char*` | Печатает строку в stderr. |
| **eprintln**| `eprintln s: char*` | Печатает строку в stderr с символом новой строки. |

## Другие потоки

Zen-C относится к `stdout`, `stdin` и `stderr` как к специальным файловым дескрипторам, которые можно использовать с методами из модуля `std/fs`.
