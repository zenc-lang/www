# std/queue

`Queue<T>` — это обобщенная очередь, работающая по принципу "первым пришел — первым ушел" (FIFO), реализованная как **кольцевой буфер (Ring Buffer)**.

## Использование

```zc
import "std/queue.zc"

fn main() {
    let q = Queue<int>::new();
    
    q.push(1);
    q.push(2);
    q.push(3);
    
    // pop() возвращает Option<T>
    if (q.pop().is_some()) {
        println "Извлечено: {q.pop().unwrap()}"; // 1
    }
}
```

## Детали реализации

- **Кольцевой буфер**: Использует зацикленный массив с индексами `head` и `tail`.
- **Производительность**:
    - `push`: **Амортизированная O(1)** (изменяет размер при заполнении).
    - `pop`: **O(1)** (просто сдвигает индекс head).
    - `clone`: **O(N)**.
- **Безопасность**: Безопасная обработка переполнения и перераспределения памяти.

## Определение структуры

```zc
struct Queue<T> {
    data: T*;
    cap: usize;
    head: usize;
    tail: usize;
    count: usize;
}
```

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Queue<T>::new() -> Queue<T>` | Создает новую пустую очередь. |
| **clone** | `clone(self) -> Queue<T>` | Создает глубокую копию очереди. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **push** | `push(self, value: T)` | Добавляет элемент в конец очереди. |
| **pop** | `pop(self) -> Option<T>` | Удаляет и возвращает элемент из начала. Возвращает `None`, если очередь пуста. |
| **clear** | `clear(self)` | Удаляет все элементы из очереди. |

### Доступ и свойства

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **length** | `length(self) -> usize` | Возвращает количество элементов. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает true, если очередь пуста. |
