# std/queue

`Queue<T>` - это универсальная очередь "первым пришел - первым ушел" (FIFO), реализованная в виде **кольцевого буфера (Circular Buffer)**.

## Использование

```zc
import "std/queue.zc"

fn main() {
    let q = Queue<int>::new();
    
    q.push(1);
    q.push(2);
    q.push(3);
    
    // Pop возвращает Option<T>
    if (q.pop().is_some()) {
        println "Извлечено: {q.pop().unwrap()}"; // 1
    }
}
```

## Детали реализации

- **Кольцевой буфер**: Использует кольцевой буфер с индексами `head` (голова) и `tail` (хвост).
- **Производительность**:
    - `push`: **Амортизированная O(1)** (изменяет размер при заполнении).
    - `pop`: **O(1)** (продвигает индекс головы).
    - `clone`: **O(N)**.
- **Безопасность**: Безопасная обработка зацикливания памяти и изменения размера.

## Определение структуры

```zc
struct Queue<T> {
    data: T*;
    cap: usize;
    head: usize;
    tail: usize;
    count: usize;
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Queue<T>::new() -> Queue<T>` | Создает новую пустую очередь. |
| **clone** | `clone(self) -> Queue<T>` | Создает глубокую копию очереди. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **push** | `push(self, value: T)` | Добавляет элемент в конец очереди. |
| **pop** | `pop(self) -> Option<T>` | Удаляет и возвращает элемент из начала. Возвращает `None`, если очередь пуста. |
| **clear** | `clear(self)` | Удаляет все элементы из очереди. |

### Доступ и запроры

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **length** | `length(self) -> usize` | Возвращает количество элементов. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает `true`, если очередь пуста. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Освобождает внутренний буфер. |
| **Trait** | `impl Drop for Queue` | Автоматически вызывает `free()`, когда объект выходит за пределы области видимости. |
