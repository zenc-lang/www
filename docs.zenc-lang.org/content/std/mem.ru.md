# std/mem

Модуль `std/mem` предоставляет основные утилиты управления памятью, включая функции ручного выделения, стандартные трейты жизненного цикла и реализации умных указателей.

## Обзор

- **Ручное выделение**: Оболочки для `malloc`, `calloc` и `free` с типобезопасными сигнатурами.
- **Трейты**: Определяет основные трейты жизненного цикла: `Drop` (деструкторы), `Clone` (глубокое копирование) и `Copy` (неявное копирование).
- **Умные указатели**: Включает `Box<T>` для данных, выделенных в куче, с автоматической очисткой (RAII).
- **Утилиты буфера**: Высокоуровневые функции для обмена, обнуления и копирования памяти.

## Использование

```zc
import "std/mem.zc"

fn main() {
    // Ручное выделение памяти
    let ptr = alloc<int>();
    *ptr = 42;
    free(ptr);
    
    // Автоматическая очистка с помощью Box (RAII)
    {
        let b = Box<int>::new();
        *b.get() = 100;
        // память освобождается здесь автоматически
    }
}
```

## Методы

### Выделение памяти

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **alloc\<T>**| `alloc<T>() -> T*` | Выделяет память для одного экземпляра `T`. |
| **zalloc\<T>**| `zalloc<T>() -> T*` | Выделяет обнуленную память для одного экземпляра `T`. |
| **alloc_n\<T>**| `alloc_n<T>(n: usize) -> T*` | Выделяет память для массива из `n` экземпляров `T`. |

### Операции

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **mem_zero\<T>**| `mem_zero<T>(ptr: T*, count: usize)` | Заполняет нулями память для `count` экземпляров `T`. |
| **mem_copy\<T>**| `mem_copy<T>(dst: T*, src: T*, count: usize)`| Копирует `count` экземпляров `T` из `src` в `dst`. |
| **swap\<T>** | `swap<T>(a: T*, b: T*)` | Меняет местами значения в двух местах памяти. |

## Трейты

| Трейт | Метод | Сигнатура | Описание |
| :--- | :--- | :--- | :--- |
| **Drop** | **drop** | `drop(self)` | Деструктор, вызываемый при выходе объекта из области видимости. |
| **Clone** | **clone** | `clone(self) -> Self` | Создает глубокую копию объекта. |
| **Copy** | *(Маркер)* | Н/Д | Указывает, что тип должен использовать неявное копирование вместо перемещения. |

## Определение структуры: `Box<T>`

Простой умный указатель RAII для управления памятью в куче.

```zc
struct Box<T> {
    ptr: T*;
}
```

### Методы `Box`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Box::new() -> Box<T>` | Выделяет новый управляемый в куче экземпляр. |
| **from_ptr** | `Box::from_ptr(p: T*) -> Box<T>` | Создает `Box`, который принимает владение существующим указателем. |
| **get** | `get(self) -> T*` | Возвращает внутренний сырой указатель. |
| **free** | `free(self)` | Вручную освобождает подлежащую память. |
| **Trait** | `impl Drop for Box<T>` | Автоматически вызывает `free()`, когда box выходит из области видимости. |
