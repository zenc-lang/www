# std/random

`std/random` 模組提供用於生成偽隨機數（Pseudo-Random Numbers）的工具。對於加密安全需求，請使用 `std/crypto`。

## 使用

```zc
import "std/random.zc"

fn main() {
    // 使用當前時間初始化種子
    Random::seed(Time::now());
    
    // 生成 1 到 100 之間的數字
    let n = Random::range(1, 100);
    println "隨機數: {n}";
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **seed** | `Random::seed(s: u64)` | 設置發生器的初始狀態。 |
| **next** | `Random::next() -> u32` | 返回下一個隨機 32 位整數。 |
| **range** | `Random::range(min: int, max: int) -> int` | 返回 [min, max] 範圍內的一個隨機整數。 |
| **float** | `Random::float() -> double` | 返回 0.0 到 1.0 之間的一個隨機浮點數。 |
