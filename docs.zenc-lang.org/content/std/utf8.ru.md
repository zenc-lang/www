# std/utf8

Модуль `std/utf8` предоставляет низкоуровневые утилиты для кодирования, декодирования и валидации текста в формате UTF-8.

## Обзор

- **Валидация**: Проверка, является ли массив байтов корректной UTF-8 строкой.
- **Длина символов**: Определение количества байтов, необходимых для кодирования конкретного Unicode-символа (`rune`).
- **Итерация**: Безопасный перебор Unicode-символов в строке.

## Использование

```zc
import "std/utf8.zc"

fn main() {
    let s = "Привет"; // UTF-8 строка
    
    println "Байт: {s.length()}";
    println "Символов: {Utf8::count(s)}";
    
    if (Utf8::is_valid(s)) {
        // ...
    }
}
```

## Методы

### Валидация и счет

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_valid** | `Utf8::is_valid(s: char*) -> bool` | Проверяет корректность UTF-8 последовательности. |
| **count** | `Utf8::count(s: char*) -> usize` | Возвращает количество Unicode-символов в строке. |

### Кодирование и декодирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **encode** | `Utf8::encode(r: rune, out: u8*) -> int` | Кодирует `rune` в буфер `out`. Возвращает количество записанных байтов (1-4). |
| **decode** | `Utf8::decode(s: u8*, out: rune*) -> int` | Декодирует первый символ из `s` в `out`. Возвращает количество прочитанных байтов. |

### Свойства символов

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **rune_len** | `Utf8::rune_len(r: rune) -> int` | Возвращает количество байтов, необходимых для кодирования `rune`. |
