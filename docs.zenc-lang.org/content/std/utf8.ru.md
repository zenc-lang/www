# std/utf8

Модуль `std/utf8` предоставляет утилиты для работы с кодовыми точками Unicode (тип `rune`) и кодировкой UTF-8.

## Использование

```zc
import "std/utf8.zc"

fn main() {
    let r = 'ñ';
    
    if (Utf8::is_alpha(r)) {
        println "{r} — это буква";
    }
}
```

## Методы

### Запросы и идентификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_digit** | `is_digit(r: rune) -> bool` | Возвращает true, если руна является десятичной цифрой (0-9). |
| **is_alpha** | `is_alpha(r: rune) -> bool` | Возвращает true, если руна является буквенным символом. |
| **is_whitespace** | `is_whitespace(r: rune) -> bool` | Возвращает true, если руна является пробельным символом. |
| **is_upper** | `is_upper(r: rune) -> bool` | Возвращает true, если руна является заглавной буквой. |
| **is_lower** | `is_lower(r: rune) -> bool` | Возвращает true, если руна является строчной буквой. |
| **is_valid** | `is_valid(data: char*, len: usize) -> bool` | Возвращает true, если буфер содержит валидные данные UTF-8. |

### Преобразование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **to_upper** | `to_upper(r: rune) -> rune` | Возвращает руну в верхнем регистре. |
| **to_lower** | `to_lower(r: rune) -> rune` | Возвращает руну в нижнем регистре. |

### Кодирование и декодирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **encode** | `encode(r: rune, buf: char*) -> usize` | Кодирует руну в UTF-8. Возвращает количество записанных байт (1–4). |
| **rune_len** | `rune_len(r: rune) -> usize` | Возвращает количество байт, необходимых для кодирования руны. |
| **decode** | `decode(data: char*, len: usize, consumed: usize*) -> rune" | Декодирует первую последовательность UTF-8 из данных. |
