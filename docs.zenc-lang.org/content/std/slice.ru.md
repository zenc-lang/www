# std/slice

`Slice<T>` - это легковесное, не владеющее данными "представление" (view) непрерывной последовательности элементов. В основном используется для обеспечения безопасного и удобного интерфейса при работе с массивами фиксированного размера.

## Обзор

- **Не владеющий**: Слайсы не копируют и не владеют подлежащими данными; они просто указывают на них.
- **Поддержка итерации**: Реализует метод `iterator()`, позволяя использовать слайсы напрямую в циклах `for-in`.
- **Автоматическое преобразование**: Компилятор Zen-C автоматически преобразует массивы фиксированного размера в слайсы при итерации или передаче в функции, ожидающие слайс.
- **Безопасный доступ**: Предоставляет методы `get()` и `at()` с проверкой границ, возвращающие `Option<T>`.

## Использование

```zc
import "std/slice.zc"

fn main() {
    let arr: int[5] = [10, 20, 30, 40, 50];
    
    // Явное создание слайса
    let s = Slice<int>::from_array(arr, 5);
    
    // Прямая итерация по слайсу
    for val in s {
        println "{val}";
    }
    
    // Прямая итерация по массиву (автоматически импортирует std/slice.zc)
    for val in arr {
        println "{val}";
    }
}
```

## Определение структуры

```zc
struct Slice<T> {
    data: T*;
    len: usize;
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **from_array** | `Slice<T>::from_array(ptr: T*, len: usize) -> Slice<T>` | Создает представление (слайс) над указателем на массив. |
| **new** | `Slice<T>::new(data: T*, len: usize) -> Slice<T>` | Псевдоним для `from_array`. |

### Итерация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **iterator** | `iterator(self) -> SliceIter<T>` | Возвращает итератор для использования в циклах `for-in`. |

### Доступ и запроры

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **length** | `length(self) -> usize` | Возвращает количество элементов в слайсе. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает `true`, если слайс не содержит элементов. |
| **get** | `get(self, idx: usize) -> Option<T>` | Возвращает элемент по индексу `idx` или `None`, если индекс вне границ. |
| **at** | `at(self, idx: usize) -> Option<T>` | Псевдоним для `get`. |

## Примечания

- **Автоимпорт**: `std/slice.zc` автоматически импортируется компилятором при выполнении итерации `for-in` по массивам фиксированного размера.
- **Безопасность**: Хотя `data` - это сырой указатель, структура `Slice` поощряет использование метода `get()`, учитывающего длину, для безопасного доступа.
