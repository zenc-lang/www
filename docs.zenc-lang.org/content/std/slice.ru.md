# std/slice

`Slice<T>` — это представление (view) непрерывной последовательности элементов в другой коллекции или массиве. Срез состоит из указателя на данные и длины.

## Обзор

- **Легковесность**: Срез не владеет данными, это просто ссылка на существующие данные.
- **Безопасность**: Доступ через срезы включает проверку границ (bounds checking).
- **Универсальность**: Может ссылаться на данные в куче (`Vec`), в стеке (массив) или в отображаемой памяти.

## Использование

```zc
import "std/slice.zc"
import "std/vec.zc"

fn main() {
    let v = Vec<int>::new();
    v.push(10); v.push(20); v.push(30);

    // Создание среза из первых двух элементов
    let s = v.as_slice().sub_slice(0, 2);
    
    println "Длина среза: {s.length()}"; // 2
    println "Первый элемент: {s[0]}"; // 10
}
```

## Определение структуры

```zc
struct Slice<T> {
    ptr: T*;
    len: usize;
}
```

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Slice<T>::new(ptr: T*, len: usize) -> Slice<T>` | Создает новый срез из указателя и длины. |
| **sub_slice**| `sub_slice(self, start: usize, len: usize) -> Slice<T>` | Создает новый, меньший срез на основе текущего. |

### Доступ и свойства

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **get** | `get(self, idx: usize) -> T` | Возвращает копию элемента. Паникует, если индекс вне границ. |
| **get_ptr** | `get_ptr(self, idx: usize) -> T*` | Возвращает указатель на элемент. Паникует, если индекс вне границ. |
| **length** | `length(self) -> usize` | Возвращает количество элементов в срезе. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает true, если длина равна 0. |

### Операции

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **copy_from** | `copy_from(self, src: Slice<T>)` | Копирует элементы из другого среза в текущий. Срезы должны быть одинаковой длины. |
