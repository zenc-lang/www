# std/bits

`std/bits` 模块提供用于操作位、字节和执行低级位计算的工具函数。

## 概览

- **位操作**: 设置、清除和翻转各个位。
- **计数操作**: 计算置位（设置）的位数或前导零。
- **字节序转换**: 在大端和小端表示之间转换数值。

## 使用

```zc
import "std/bits.zc"

fn main() {
    let val: u32 = 0b0000_0001;
    
    // 计数置位的位数 (popcount)
    assert(Bits::count_ones(val) == 1);
    
    // 旋转位
    let rotated = Bits::rotate_left(val, 1);
    assert(rotated == 0b0000_0010);
}
```

## 方法

### 计数和查找

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **count_ones** | `Bits::count_ones(v: u32) -> int` | 返回设置为 1 的位数。 |
| **trailing_zeros**| `Bits::trailing_zeros(v: u32) -> int` | 返回最低有效位之后连续零的数量。 |
| **leading_zeros** | `Bits::leading_zeros(v: u32) -> int` | 返回最高有效位之前连续零的数量。 |

### 位操作

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **rotate_left** | `Bits::rotate_left(v: u32, n: int) -> u32` | 将位向左循环旋转 n 位。 |
| **rotate_right** | `Bits::rotate_right(v: u32, n: int) -> u32` | 将位向右循环旋转 n 位。 |
| **reverse** | `Bits::reverse(v: u32) -> u32` | 反转位的顺序。 |

### 字节操作

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **swap_bytes** | `Bits::swap_bytes(v: u32) -> u32` | 翻转字节顺序（用于字节序转换）。 |
| **to_le** | `Bits::to_le(v: u32) -> u32` | 将数值转换为小端序。 |
| **to_be** | `Bits::to_be(v: u32) -> u32` | 将数值转换为大端序。 |
