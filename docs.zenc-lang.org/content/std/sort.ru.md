# std/sort

Модуль `std/sort` предоставляет высокооптимизированные алгоритмы сортировки. Он нативно реализует алгоритм `QuickSort` с использованием полиморфного макроса с нулевыми накладными расходами.

## Использование

```zc
import "std/sort.zc"

fn main() {
    let arr: int[5] = [52, 13, 99, 4, 42];
    sort_int((int*)arr, 5); // Станет [4, 13, 42, 52, 99]
}
```

## Функции

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **sort_int** | `sort_int(arr: int*, len: usize)` | Сортирует массив стандартных целых чисел `[i32]`. |
| **sort_long** | `sort_long(arr: long*, len: usize)` | Сортирует массив длинных целых чисел `[i64]`. |
| **sort_float** | `sort_float(arr: float*, len: usize)` | Сортирует массив чисел с плавающей запятой `[f32]`. |
| **sort_double** | `sort_double(arr: double*, len: usize)` | Сортирует массив чисел с плавающей запятой двойной точности `[f64]`. |

## Пользовательская сортировка

Если вы создаете пользовательскую структуру `struct` с перегрузкой оператора `<`, вы можете сгенерировать пользовательский алгоритм сортировки:

```zc
// Генерирует `sort_MyStruct(MyStruct* arr, usize len)`
raw { ZC_IMPL_SORT(MyStruct) } 
```
