# std/sort

Модуль `std/sort` предоставляет функции для сортировки коллекций и срезов (slices).

## Использование

```zc
import "std/sort.zc"
import "std/vec.zc"

fn main() {
    let v = Vec<int>::new();
    v.push(3); v.push(1); v.push(2);
    
    // Сортировка на месте (по умолчанию по возрастанию)
    Sort::sort_int(v.as_slice());
}
```

## Функции сортировки

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **sort_int** | `Sort::sort_int(s: Slice<int>)` | Сортирует срез целых чисел по возрастанию. |
| **sort_float**| `Sort::sort_float(s: Slice<float>)` | Сортирует срез чисел с плавающей запятой. |
| **sort_by** | `Sort::sort_by(s: Slice<T>, cmp: fn(T*, T*) -> int)` | Сортирует обобщенный срез, используя функцию сравнения. |

## Алгоритм

- В настоящее время используется алгоритм **Quicksort** с оптимизированной стратегией выбора опорного элемента (pivot), обеспечивающей хорошую производительность в среднем.
- Функция `sort_by` стабильна, если пользовательская функция сравнения реализована корректно.
