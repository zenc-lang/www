# std/vec

`Vec<T>` 是一個動態的可增長數組，是 Zen-C 中最常用的集合類型。

## 使用

```zc
import "std/vec.zc"

fn main() {
    let v = Vec<int>::new();
    
    // 添加元素
    v.push(10);
    v.push(20);
    
    // 按索引訪問
    println "第一個元素: {v.at(0)}";
    
    // 獲取大小
    println "長度: {v.length()}";
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Vec<T>::new() -> Vec<T>` | 創建一個新的空向量。 |
| **push** | `push(self, val: T)` | 在向量末尾添加一個值。 |
| **pop** | `pop(self) -> Option<T>` | 移除並返回最後一個元素。 |
| **at** | `at(self, index: usize) -> T` | 返回指定位置的元素副本。 |
| **clear** | `clear(self)` | 移除所有元素。 |
| **length** | `length(self) -> usize` | 返回當前元素數量。 |
