# std/net

`std/net` 模組提供網絡編程工具，支持 TCP 和 UDP 協議，以及客戶端和服務端的實現。

## 使用

```zc
import "std/net.zc"

fn main() {
    // 創建一個 TCP 客戶端
    match TcpStream::connect("127.0.0.1", 8080) {
        Ok(stream) => {
            stream.write("你好，服務器！");
            stream.close();
        },
        Err(e) => println "連接失敗: {e}"
    }
}
```

## TCP 服務端

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **bind** | `TcpListener::bind(port: int) -> Result<TcpListener>` | 在指定端口監聽。 |
| **accept** | `accept(self) -> Result<TcpStream>` | 等待並接受新的客戶端連接。 |

## TCP 客戶端 (TcpStream)

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **connect** | `TcpStream::connect(ip: char*, port: int) -> Result<TcpStream>` | 連接到遠程服務器。 |
| **read** | `read(self, buf: u8*, n: usize) -> usize` | 從流中讀取數據。 |
| **write** | `write(self, data: char*) -> usize` | 向流中寫入數據。 |
| **close** | `close(self)` | 關閉連接。 |
