# std/user

`std/user` 模組提供用於檢索系統用戶和組信息的工具。

## 使用

```zc
import "std/user.zc"

fn main() {
    match User::current() {
        Ok(u) => {
            println "當前用戶: {u.name}";
            println "主目錄: {u.home}";
        },
        Err(e) => println "無法獲取用戶信息: {e}"
    }
}
```

## User 結構體

| 字段 | 類型 | 描述 |
| :--- | :--- | :--- |
| **uid** | `u32` | 用戶 ID。 |
| **gid** | `u32` | 組 ID。 |
| **name** | `char*` | 用戶名。 |
| **home** | `char*` | 用戶主目錄的路徑。 |
| **shell** | `char*` | 用戶的默認 shell。 |

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **current** | `User::current() -> Result<User>` | 返回運行當前進程的用戶信息。 |
| **by_name** | `User::by_name(name: char*) -> Result<User>` | 按名稱搜索用戶。 |
| **by_id** | `User::by_id(uid: u32) -> Result<User>`| 按 UID 搜索用戶。 |
