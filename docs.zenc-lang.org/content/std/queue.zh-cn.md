# std/queue

`Queue<T>` 是一种通用的先进先出 (FIFO) 集合，实现为链表或循环缓冲区。

## 使用

```zc
import "std/queue.zc"

fn main() {
    let q = Queue<int>::new();
    
    q.enqueue(10);
    q.enqueue(20);
    
    match q.dequeue() {
        Some(val) => println "出队: {val}", // 10
        None => println "队列为空"
    }
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Queue<T>::new() -> Queue<T>` | 创建一个空队列。 |

### 修改

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **enqueue** | `enqueue(self, val: T)` | 在队列末尾添加一个元素。 |
| **dequeue** | `dequeue(self) -> Option<T>` | 移除并返回队列开头的元素。 |
| **clear** | `clear(self)` | 移除所有项。 |

### 访问与属性

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **peek** | `peek(self) -> Option<T>` | 查看开头的元素而不移除它。 |
| **length** | `length(self) -> usize` | 返回队列中项的数量。 |
| **is_empty** | `is_empty(self) -> bool` | 如果队列不含任何项，则返回 true。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放队列使用的内存。 |
| **Trait** | `impl Drop for Queue` | 自动调用 `free()`。 |
