# std/core

`std/core` 模块为 Zen-C 程序提供最基本的定义和错误 handling 原语。它是大多数其他标准库隐含需要的模块。

## 概览

- **标准头文件**：包含关键的 C 头文件，如 `stdlib.h`、`stdio.h` 和 `stdbool.h`。
- **恐慌机制**：提供 `panic` 宏，用于报告不可恢复的错误。
- **进程控制**：包含基本的进程控制函数，如 `exit`。

## 方法

### 错误处理

| 方法/宏 | 签名 | 说明 |
| :--- | :--- | :--- |
| **panic** | `panic(msg: char*)` | 打印格式化的错误消息（包括文件、行号和函数名），然后终止进程。 |

### 进程控制

| 方法 | 签名 | 说明 |
| :--- | :--- | :--- |
| **exit** | `exit(code: int)`| 使用给定的返回代码立即终止进程。 |
