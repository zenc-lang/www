# std/core

`std/core` 模块包含 Zen-C 标准库的基本构建块和基础类型。

## 概览

- **基础类型**: 定义所有其他模块使用的核心类型和接口。
- **核心宏**: 包括用于断言（assertion）和错误处理的宏。
- **平台无关性**: 提供对编译器内置功能的抽象层。

## 使用

```zc
import "std/core.zc"

fn main() {
    // assert 确保运行时条件为真
    assert(1 + 1 == 2);
}
```

## 函数和宏

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **assert** | `assert(condition: bool)` | 如果条件为假，则打印错误消息并中止执行。 |
| **panic** | `panic(message: char*) | 立即中止程序并显示指定的消息。 |
| **exit** | `exit(code: int)` | 以指定的退出代码结束进程。 |
