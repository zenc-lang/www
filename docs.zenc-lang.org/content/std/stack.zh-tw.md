# std/stack

`Stack<T>` 是一個後進先出（LIFO）數據結構，用於管理具有嵌套或遞歸特徵的數據。

## 使用

```zc
import "std/stack.zc"

fn main() {
    let s = Stack<int>::new();
    
    s.push(1);
    s.push(2);
    
    match s.pop() {
        Some(val) => println "彈出: {val}", // 2
        None => println "棧為空"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Stack<T>::new() -> Stack<T>` | 創建一個空棧。 |
| **push** | `push(self, val: T)` | 在棧頂添加一個元素。 |
| **pop** | `pop(self) -> Option<T>` | 移除並返回棧頂元素。 |
| **peek** | `peek(self) -> Option<T>` | 返回棧頂元素的引用而不移除它。 |
| **length** | `length(self) -> usize` | 返回棧中的元素數量。 |
| **clear** | `clear(self)` | 移除棧中所有元素。 |
