# std/sync

`std/sync` 模組提供用於多線程環境中互斥訪問和同步的工具，包括互斥鎖（Mutex）和條件變量。

## 使用

```zc
import "std/sync.zc"

fn main() {
    let m = Mutex::new();
    
    // 獲取鎖
    m.lock();
    
    // 臨界區
    println "受保護的操作";
    
    // 釋放鎖
    m.unlock();
}
```

## 方法

### Mutex (互斥鎖)

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Mutex::new() -> Mutex` | 創建一個新的互斥鎖。 |
| **lock** | `lock(self)` | 調用線程將阻塞，直到獲得鎖。 |
| **unlock** | `unlock(self)` | 釋放鎖，允許其他線程獲取。 |
| **try_lock** | `try_lock(self) -> bool` | 嘗試獲取鎖而不阻塞。 |

### CondVar (條件變量)

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **wait** | `wait(self, m: Mutex)` | 阻塞當前線程，直到收到通知。 |
| **notify** | `notify(self)` | 喚醒一個正在等待的線程。 |
| **notify_all** | `notify_all(self)` | 喚醒所有正在等待的線程。 |
