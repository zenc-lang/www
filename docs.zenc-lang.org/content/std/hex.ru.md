# std/hex

Модуль `std/hex` предоставляет функции для преобразования бинарных данных в шестнадцатеричные строки и наоборот.

## Использование

```zc
import "std/hex.zc"

fn main() {
    let raw: u8[3] = [0xDE, 0xAD, 0xBE];
    
    // Кодирование в hex
    let encoded = Hex::encode(&raw[0], 3);
    println "Hex: {encoded}"; // "deadbe"
    
    // Декодирование обратно
    let decoded = Hex::decode(encoded.c_str());
}
```

## Методы

### Кодирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **encode** | `Hex::encode(data: u8*, len: usize) -> String` | Кодирует бинарные данные в шестнадцатеричную строку в нижнем регистре. |
| **encode_upper** | `Hex::encode_upper(data: u8*, len: usize) -> String` | Кодирует бинарные данные в шестнадцатеричную строку в верхнем регистре. |

### Декодирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **decode** | `Hex::decode(s: char*) -> Vec<u8>` | Декодирует шестнадцатеричную строку обратно в бинарные данные. |

## Детали реализации

- **Кодирование**: Создает по 2 символа на каждый байт входных данных.
- **Декодирование**: Требует, чтобы входная строка имела четную длину и содержала только валидные шестнадцатеричные символы (`0-9`, `a-f`, `A-F`).
