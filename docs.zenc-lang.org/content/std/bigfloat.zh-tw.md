# std/bigfloat

`std/bigfloat` 模組提供任意精度的浮點數運算。它允許處理超出標準 `float` 或 `double` 類型限制的極大或極精確的數值。

## 使用

```zc
import "std/bigfloat.zc"

fn main() {
    // 創建兩個大浮點數
    let a = BigFloat::from_string("1.2345678901234567890123456789");
    let b = BigFloat::from_int(2);
    
    // 執行運算
    let res = a.mul(b);
    
    println "結果: {res}";
    res.free();
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **from_string** | `BigFloat::from_string(s: char*) -> BigFloat` | 從字符串創建一個 BigFloat。 |
| **from_int** | `BigFloat::from_int(v: int) -> BigFloat` | 從整數創建一個 BigFloat。 |
| **add** | `add(self, other: BigFloat) -> BigFloat` | 將兩個 BigFloat 相加並返回結果。 |
| **mul** | `mul(self, other: BigFloat) -> BigFloat` | 將兩個 BigFloat 相乘。 |
| **to_string** | `to_string(self) -> String` | 返回數值的字符串表示。 |
