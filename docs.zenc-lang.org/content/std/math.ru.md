# std/math

Модуль `Math` предоставляет базовый набор стандартных математических констант и функций. Он выступает в качестве Zen-C оболочки для стандартных математических операций с плавающей запятой.

## Обзор

- **Статические методы**: Все методы вызываются непосредственно у структуры `Math`.
- **Точность**: Использует `double` для высокоточной арифметики с плавающей запятой.
- **Комплексность**: Охватывает тригонометрию, экспоненты, логарифмы и округление.
- **Эффективность**: Напрямую оборачивает оптимизированные функции библиотеки C.

## Использование

```zc
import "std/math.zc"

fn main() {
    let radius = 5.0;
    let area = Math::PI() * Math::pow(radius, 2.0);
    println "Площадь круга: {area}";
}
```

## Константы

Все константы являются статическими функциями, возвращающими `double`.

| Константа | Описание |
| :--- | :--- |
| **Math::PI()** | Число Пи (приблизительно 3.14159). |
| **Math::E()** | Число Эйлера (приблизительно 2.71828). |

## Методы

### Арифметика

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **abs** | `abs(x: double) -> double` | Возвращает абсолютное значение `x`. |
| **sqrt** | `sqrt(x: double) -> double` | Возвращает квадратный корень из `x`. |
| **pow** | `pow(base: double, exp: double) -> double` | Возвращает `base`, возведенное в степень `exp`. |
| **exp** | `exp(x: double) -> double` | Возвращает `e`, возведенное в степень `x`. |
| **log** | `log(x: double) -> double` | Возвращает натуральный логарифм (по основанию e) числа `x`. |
| **log10** | `log10(x: double) -> double` | Возвращает десятичный логарифм числа `x`. |

### Тригонометрия

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **sin** | `sin(x: double) -> double` | Возвращает синус `x` (в радианах). |
| **cos** | `cos(x: double) -> double` | Возвращает косинус `x` (в радианах). |
| **tan** | `tan(x: double) -> double` | Возвращает тангенс `x` (в радианах). |
| **asin** | `asin(x: double) -> double` | Возвращает арксинус `x`. |
| **acos** | `acos(x: double) -> double` | Возвращает арккосинус `x`. |
| **atan** | `atan(x: double) -> double` | Возвращает арктангенс `x`. |
| **atan2** | `atan2(y: double, x: double) -> double` | Возвращает арктангенс отношения `y/x`. |

### Округление и остаток

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **ceil** | `ceil(x: double) -> double` | Округляет вверх до ближайшего целого. |
| **floor** | `floor(x: double) -> double` | Округляет вниз до ближайшего целого. |
| **round** | `round(x: double) -> double` | Округляет до ближайшего целого. |
| **mod** | `mod(x: double, y: double) -> double` | Вычисляет остаток от деления `x / y` для чисел с плавающей запятой. |

### Мин / Макс

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **max** | `max(a: double, b: double) -> double` | Возвращает большее из двух значений. |
| **min** | `min(a: double, b: double) -> double` | Возвращает меньшее из двух значений. |
