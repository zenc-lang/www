# std/utf8

`std/utf8` 模組提供驗證、解碼和編碼 UTF-8 文本的工具。

## 使用

```zc
import "std/utf8.zc"

fn main() {
    let s = "你好";
    
    // 獲取字符（碼點）數量，而非字節數
    let count = Utf8::count(s);
    println "字符數: {count}"; // 2
    
    // 驗證是否為有效的 UTF-8
    if (Utf8::is_valid(s)) {
        println "有效的 UTF-8";
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **is_valid** | `Utf8::is_valid(s: char*) -> bool` | 如果字符串是有效的 UTF-8，則返回 true。 |
| **count** | `Utf8::count(s: char*) -> usize` | 返回字符串中 Unicode 碼點的數量。 |
| **size** | `Utf8::size(cp: u32) -> usize` | 返回編碼一個碼點所需的字節數。 |
| **decode** | `Utf8::decode(s: char*) -> u32` | 從字符串開頭解碼一個碼點。 |
