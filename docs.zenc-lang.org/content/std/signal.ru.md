# std/sys/signal

Модуль `std/sys/signal` предоставляет примитивы для обработки системных сигналов, оборачивая функциональность POSIX `signal.h`.

## Обзор

- **Перехват сигналов**: Определение пользовательских обработчиков для таких сигналов, как `SIGINT` (Ctrl+C).
- **Плавное завершение**: Использование обработчиков сигналов для выполнения очистки перед выходом.
- **Общие константы**: Предоставляет кроссплатформенные определения для стандартных сигналов.

## Использование

```zc
import "std/sys/signal.zc"
import "std/io.zc"

fn on_interrupt(sig: int) {
    println "Получен SIGINT ({sig}). Очистка...";
    exit(0);
}

fn main() {
    Signal::set_handler(Z_SIGINT, on_interrupt);
    println "Ожидание Ctrl+C...";
    while(true) {}
}
```

## Методы

### Методы `Signal`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **set_handler** | `Signal::set_handler(sig: int, handler: fn*(int)) -> fn*(int)` | Регистрирует обработчик для заданного сигнала и возвращает предыдущий обработчик. |

## Константы

### Стандартные сигналы
- `Z_SIGINT`: Прерывание с клавиатуры (Ctrl+C).
- `Z_SIGILL`: Недопустимая инструкция.
- `Z_SIGABRT`: Сигнал прерывания (abort).
- `Z_SIGFPE`: Исключение в операции с плавающей запятой.
- `Z_SIGSEGV`: Нарушение сегментации (недопустимый доступ к памяти).
- `Z_SIGTERM`: Сигнал завершения.
