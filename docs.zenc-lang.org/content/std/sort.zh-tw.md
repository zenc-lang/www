# std/sort

`std/sort` 模組包含用於對數組和切片進行排序的算法。

## 使用

```zc
import "std/sort.zc"

fn main() {
    let arr: int[5] = {5, 2, 9, 1, 5};
    
    // 對數組進行排序
    Sort::ints(&arr[0], 5);
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **ints** | `Sort::ints(ptr: int*, n: usize)` | 對整數數組進行升序排序 (通常使用快速排序)。 |
| **doubles** | `Sort::doubles(ptr: double*, n: usize)` | 對雙精度浮點數數組進行排序。 |
| **strings** | `Sort::strings(ptr: char**, n: usize)` | 按字母順序對字符串數組進行排序。 |
| **by** | `Sort::by(ptr: T*, n: usize, cmp: fn)` | 使用自定義比較函數對泛型類型進行排序。 |
