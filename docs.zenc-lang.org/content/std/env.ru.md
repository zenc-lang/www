# std/env

Модуль `std/env` обеспечивает доступ к аргументам командной строки, переменным окружения и системной информации.

## Использование

```zc
import "std/env.zc"

fn main() {
    // Доступ к переменным окружения
    match Env::get("PATH") {
        Some(path) => println "PATH: {path}",
        None => println "PATH не установлен"
    }

    // Итерация по аргументам командной строки
    for arg in Env::args() {
        println "Arg: {arg}";
    }
}
```

## Методы

### Аргументы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **args** | `Env::args() -> Vec<String>` | Возвращает вектор аргументов, переданных процессу. |

### Переменные окружения

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **get** | `Env::get(key: char*) -> Option<String>` | Ищет переменную окружения по имени. |
| **set** | `Env::set(key: char*, value: char*)` | Устанавливает или обновляет переменную окружения. |
| **remove** | `Env::remove(key: char*)` | Удаляет переменную окружения. |

### Системная информация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **os** | `Env::os() -> char*` | Возвращает название ОС (например, "linux", "macos", "windows"). |
| **arch** | `Env::arch() -> char*` | Возвращает архитектуру процессора (например, "x86_64", "arm64"). |
