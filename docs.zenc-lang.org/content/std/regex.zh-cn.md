# std/regex

`std/regex` 模块提供强大的正则表达式搜索和匹配功能。

## 使用

```zc
import "std/regex.zc"

fn main() {
    let pattern = "[a-z]+";
    let text = "hello 123 world";
    
    // 查找模式的所有出现
    let matches = Regex::find_all(pattern, text);
    
    for m in matches {
        println "找到匹配: {m}";
    }
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **is_match** | `Regex::is_match(pattern: char*, text: char*) -> bool` | 如果文本中存在该模式，则返回 true。 |
| **find** | `Regex::find(pattern: char*, text: char*) -> Option<String>` | 返回模式的第一个匹配项。 |
| **find_all** | `Regex::find_all(pattern: char*, text: char*) -> Vec<String>` | 返回所有非重叠匹配项的列表。 |
| **replace** | `Regex::replace(pattern: char*, text: char*, repl: char*) -> String`| 将匹配的部分替换为指定的字符串。 |

## 正则表达式语法

Zen-C 的正则表达式引擎支持标准语法，包括：
- `.` (任意字符)
- `*`, `+`, `?` (量词)
- `^`, `$` (锚点)
- `[a-z]` (字符类)
- `(sub)` (捕获组)
