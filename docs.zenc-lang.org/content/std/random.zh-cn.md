# std/random

`std/random` 模块提供生成伪随机数 (PRNG) 的工具。

## 使用

```zc
import "std/random.zc"

fn main() {
    // 使用当前时间作为种子初始化
    Random::seed(0);
    
    // 生成 1 到 100 之间的随机数
    let val = Random::range(1, 101);
    println "随机值: {val}";
    
    // 生成随机浮点数 [0.0, 1.0)
    let f = Random::next_double();
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **seed** | `Random::seed(s: u64)` | 为随机数生成器设置种子。如果 s 为 0，则使用系统时钟。 |
| **next_int** | `Random::next_int() -> int` | 返回一个随机整数。 |
| **range** | `Random::range(min: int, max: int) -> int` | 返回 [min, max) 范围内的随机整数。 |
| **next_double**| `Random::next_double() -> double` | 返回 [0.0, 1.0) 范围内的随机双精度浮点数。 |
