# std/json

`std/json` 模块提供用于解析和序列化 JSON（JavaScript 对象表示法）数据的实用程序。它内部使用 `cJSON` 库。

## 使用

```zc
import "std/json.zc"

fn main() {
    let raw = "{\"name\": \"Zen\", \"version\": 1.0}";
    
    // 解析 JSON
    match Json::parse(raw) {
        Ok(json) => {
            let name = json.get_string("name").unwrap();
            println "名称: {name}";
            json.free();
        },
        Err(e) => println "解析错误: {e}"
    }
}
```

## 方法

### 解析与生成

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **parse** | `Json::parse(raw: char*) -> Result<JsonValue>` | 将字符串解析为 JSON 对象树。 |
| **stringify** | `stringify(self) -> String` | 将 JSON 对象转换回字符串。 |

### 访问器

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **get_string** | `get_string(self, key: char*) -> Option<char*>` | 获取指定键的字符串值。 |
| **get_number** | `get_number(self, key: char*) -> Option<double>` | 获取指定键的数字值。 |
| **get_bool** | `get_bool(self, key: char*) -> Option<bool>` | 获取指定键的布尔值。 |
| **get_object** | `get_object(self, key: char*) -> Option<JsonValue>` | 获取嵌套的 JSON 对象。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放 JSON 树占用的内存。必须对根对象调用。 |
| **Trait** | `impl Drop for JsonValue` | 自动释放内存。 |
