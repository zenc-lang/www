# std/option

`Option<T>` 枚举表示一个可选值：它要么包含一个值 (`Some`)，要么不包含 (`None`)。这是 Zen-C 中处理空值（null）的安全方式。

## 使用

```zc
import "std/option.zc"

fn divide(a: float, b: float) -> Option<float> {
    if (b == 0.0) {
        return None;
    }
    return Some(a / b);
}

fn main() {
    let res = divide(10.0, 2.0);
    
    if (res.is_some()) {
        println "结果: {res.unwrap()}";
    }
}
```

## 方法

### 状态检查

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **is_some** | `is_some(self) -> bool` | 如果包含一个值，则返回 true。 |
| **is_none** | `is_none(self) -> bool` | 如果不包含值，则返回 true。 |

### 值获取

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **unwrap** | `unwrap(self) -> T` | 返回包含的值。如果为 `None`，则触发运行地错误。 |
| **unwrap_or**| `unwrap_or(self, default: T) -> T` | 返回包含的值，或者如果为 `None` 则返回提供的默认值。 |

### 辅助库

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **expect** | `expect(self, msg: char*) -> T` | 类似于 `unwrap`，但在失败时提供自定义错误消息。 |
