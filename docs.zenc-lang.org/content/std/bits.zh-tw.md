# std/bits

`std/bits` 模組提供底層的位操作工具，這對於系統編程、協議實現和性能優化非常有用。

## 使用

```zc
import "std/bits.zc"

fn main() {
    let val: u32 = 0b00000001;
    
    // 定位位
    let res = Bits::set(val, 1); // 0b00000011 (3)
    
    println "具有設置位的結果: {res}";
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **set** | `Bits::set(val: T, pos: int) -> T` | 將指定位置的位設置為 1。 |
| **clear** | `Bits::clear(val: T, pos: int) -> T` | 將指定位置的位設置為 0。 |
| **toggle** | `Bits::toggle(val: T, pos: int) -> T` | 反轉指定位置的位。 |
| **check** | `Bits::check(val: T, pos: int) -> bool` | 如果指定位置的位為 1，則返回 true。 |

## 高級操作

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **count_ones** | `Bits::count_ones(val: T) -> int` | 返回設置為 1 的位數（漢明重量）。 |
| **reverse** | `Bits::reverse(val: T) -> T` | 反轉數值中的位順序。 |
| **rotate_left**| `Bits::rotate_left(val: T, n: int) -> T` | 向左循環移位。 |
