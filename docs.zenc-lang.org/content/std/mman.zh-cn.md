# std/sys/mman

`std/sys/mman` 模块提供用于虚拟内存映射的低层工具，直接包装了 POSIX `sys/mman.h`。

## 概览

- **内存映射**: 将文件或设备映射到内存中。
- **保护管理**: 设置内存区域的读取、写入和执行权限。
- **共享内存**: 创建不同进程间可见的内存区域。

## 使用

```zc
import "std/sys/mman.zc"

fn main() {
    let size = 4096;
    
    // 创建匿名的、私有的内存映射
    let ptr = Mman::mmap(
        null, 
        size, 
        Mman::PROT_READ | Mman::PROT_WRITE, 
        Mman::MAP_PRIVATE | Mman::MAP_ANONYMOUS, 
        -1, 
        0
    );

    if (ptr != Mman::MAP_FAILED) {
        // 使用内存...
        Mman::munmap(ptr, size);
    }
}
```

## 常数

| 常数 | 描述 |
| :--- | :--- |
| **Mman::PROT_READ** | 允许读取。 |
| **Mman::PROT_WRITE** | 允许写入。 |
| **Mman::PROT_EXEC** | 允许执行。 |
| **Mman::MAP_SHARED** | 共享该映射。 |
| **Mman::MAP_PRIVATE** | 创建私有的写时拷贝 (copy-on-write) 映射。 |

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **mmap** | `Mman::mmap(addr: void*, len: usize, prot: int, flags: int, fd: int, offset: i64) -> void*` | 创建新的内存映射。 |
| **munmap**| `Mman::munmap(addr: void*, len: usize) -> int` | 删除映射。 |
| **mprotect**| `Mman::mprotect(addr: void*, len: usize, prot: int) -> int` | 更改对映射区域的保护。 |
