# std/math

`std/math` 模組包含常用數學常數和基本的數學函數。

## 使用

```zc
import "std/math.zc"

fn main() {
    let x = 16.0;
    let y = Math::sqrt(x);
    println "16 的平方根是 {y}";
    
    println "Pi: {Math::PI}";
}
```

## 常數

| 名稱 | 描述 |
| :--- | :--- |
| **PI** | 圓周率常數。 |
| **E** | 自然對數的底數 (歐拉數)。 |
| **TAU** | 2 * PI。 |

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **sqrt** | `Math::sqrt(v: double) -> double` | 計算平方根。 |
| **abs** | `Math::abs(v: double) -> double` | 返回絕對值。 |
| **sin / cos** | `Math::sin(v: double) -> double` | 三角函數（以弧度為單位）。 |
| **pow** | `Math::pow(base: double, exp: double) -> double` | 計算冪運算。 |
| **floor / ceil**| `Math::floor(v: double) -> double` | 向上或向下取整。 |
