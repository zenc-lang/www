# std/crypto/sha256

Das Modul `std/crypto/sha256` implementiert den kryptographischen Hash-Algorithmus SHA-256 gemäß FIPS 180-4.

## Überblick

- **Hashing**: Berechnet 256-Bit-Hashes von beliebigen String- oder Byte-Daten.
- **Inkrementelles Hashing**: Verarbeitet Daten in Blöcken unter Verwendung von `update` und `finalize`.
- **Hex-Ausgabe**: Komfortable `hash`-Hilfsfunktion, die einen Hexadezimal-String zurückgibt.

## Verwendung

```zc
import "std/crypto/sha256.zc"
import "std/io.zc"

fn main() {
    // Einfaches Hashing
    let hash = Sha256::hash("hello world");
    println "SHA-256: {hash}";
    
    // Inkrementelles Hashing
    let ctx = Sha256::new();
    ctx.update((u8*)"hello", 5);
    ctx.update((u8*)" world", 6);
    
    let digest: u8[32];
    ctx.finalize(&digest[0]);
}
```

## Methoden

### `Sha256` Methoden

| Methode | Signatur | Beschreibung |
| :--- | :--- | :--- |
| **new** | `Sha256::new() -> Sha256` | Initialisiert einen neuen SHA-256 Kontext. |
| **update** | `update(self, data: u8*, len: usize)` | Aktualisiert den Kontext mit einem Datenblock. |
| **finalize** | `finalize(self, hash: u8*)` | Schließt den Hash ab und schreibt das Ergebnis in den bereitgestellten 32-Byte-Puffer. |
| **hash** | `Sha256::hash(data: char*) -> String` | Komfortmethode zum Hashen eines Strings und Rückgabe seiner hexadezimalen Darstellung. |
