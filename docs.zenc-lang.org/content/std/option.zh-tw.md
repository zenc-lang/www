# std/option

`Option<T>` 類型代表一個可選值：它要麼包含一個值 `Some(T)`，要麼不包含值 `None`。

## 使用

```zc
import "std/core.zc"

fn find_user(id: int) -> Option<char*> {
    if (id == 1) return Some("Alice");
    return None;
}

fn main() {
    match find_user(1) {
        Some(name) => println "找到用戶: {name}",
        None => println "未找到用戶"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **is_some** | `is_some(self) -> bool` | 如果是 Some，則返回 true。 |
| **is_none** | `is_none(self) -> bool` | 如果是 None，則返回 true。 |
| **unwrap** | `unwrap(self) -> T` | 返回 Some 中的值，如果是 None 則崩潰（panic）。 |
| **unwrap_or** | `unwrap_or(self, default: T) -> T` | 返回值或指定的默認值。 |
