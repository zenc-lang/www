# std/core

Модуль `std/core` содержит базовые строительные блоки и фундаментальные типы стандартной библиотеки Zen-C.

## Обзор

- **Фундаментальные типы**: Определяет основные типы и интерфейсы, используемые всеми остальными модулями.
- **Основные макросы**: Включает макросы для утверждений (assertions) и обработки ошибок.
- **Платформонезависимость**: Предоставляет слой абстракции над встроенными возможностями компилятора.

## Использование

```zc
import "std/core.zc"

fn main() {
    // assert гарантирует, что условие истинно во время выполнения
    assert(1 + 1 == 2);
}
```

## Функции и макросы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **assert** | `assert(condition: bool)` | Если условие ложно, выводит сообщение об ошибке и завершает выполнение. |
| **panic** | `panic(message: char*)` | Немедленно прерывает программу с указанным сообщением. |
| **exit** | `exit(code: int)` | Завершает процесс с указанным кодом выхода. |
