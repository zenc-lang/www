# std/core

Модуль `std/core` предоставляет наиболее фундаментальные определения и примитивы обработки ошибок для программ на Zen-C. Он неявно требуется большинству других стандартных библиотек.

## Обзор

- **Стандартные заголовки**: Включает критически важные C-заголовки, такие как `stdlib.h`, `stdio.h` и `stdbool.h`.
- **Механизм паники**: Предоставляет макрос `panic` для сообщения о неисправимых ошибках.
- **Управление процессами**: Включает базовые функции управления процессами, такие как `exit`.

## Методы

### Обработка ошибок

| Метод/Макрос | Сигнатура | Описание |
| :--- | :--- | :--- |
| **panic** | `panic(msg: char*)` | Выводит отформатированное сообщение об ошибке, включая файл, строку и функцию, а затем завершает процесс. |

### Управление процессами

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **exit** | `exit(code: int)` | Немедленно завершает процесс с заданным кодом возврата. |
