# std/process

`std/process` 模块提供用于生成新进程和与当前运行进程交互的工具。

## 使用

```zc
import "std/process.zc"

fn main() {
    // 获取当前进程 ID
    let pid = Process::id();
    println "PID: {pid}";

    // 执行外部命令
    let res = Process::run("ls", {"-l", "-a"}, 2);
    if (res == 0) {
        println "命令成功执行";
    }
}
```

## 方法

### 当前进程

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **id** | `Process::id() -> u32` | 返回当前进程的 PID。 |
| **exit** | `Process::exit(code: int)` | 立即终止当前进程。 |
| **abort** | `Process::abort()` | 异常终止进程（通常会导致核心转储）。 |

### 子进程

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **run** | `Process::run(cmd: char*, args: char*[], count: int) -> int` | 运行外部命令并等待其完成。返回退出状态码。 |
| **spawn** | `Process::spawn(cmd: char*, args: char*[], count: int) -> u32` | 启动子进程但不等待其结束。返回子进程 PID。 |
