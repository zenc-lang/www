# std/fs

`std/fs` 模块提供用于与文件系统交互的工具，包括文件读取、写入和目录操作。

## 使用

```zc
import "std/fs.zc"

fn main() {
    // 读取文件内容
    match File::read_to_string("hello.txt") {
        Ok(content) => println "文件内容: {content}",
        Err(e) => println "错误: {e}"
    }

    // 写入文件
    File::write_all("output.txt", "你好世界");
}
```

## 方法

### 文件操作

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **read_to_string** | `File::read_to_string(path: char*) -> Result<String>` | 将整个文件读入 String。 |
| **write_all** | `File::write_all(path: char*, data: char*) -> Result<bool>` | 将数据写入文件（如果不存在则创建，如果存在则覆盖）。 |
| **exists** | `File::exists(path: char*) -> bool` | 如果指定的路径存在则返回 true。 |
| **remove_file** | `File::remove_file(path: char*) -> Result<bool>` | 删除文件。 |

### 目录操作

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **create_dir** | `Dir::create_dir(path: char*) -> Result<bool>` | 创建一个新目录。 |
| **remove_dir** | `Dir::remove_dir(path: char*) -> Result<bool>` | 删除一个空目录。 |
| **read_dir** | `Dir::read_dir(path: char*) -> Result<Vec<String>>` | 返回目录中所有条目的名称列表。 |

## 文件结构体 (流式 IO)

对于大型文件或高级用法，请使用 `File` 结构体：

```zc
struct File {
    handle: void*; // 对应 C FILE*
}
```

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **open** | `File::open(path: char*, mode: char*) -> Result<File>` | 以指定模式打开文件。 |
| **read** | `read(self, buffer: u8*, size: usize) -> usize` | 将字节读入荷缓冲区。 |
| **write** | `write(self, data: u8*, size: usize) -> usize` | 将内存缓冲区的数据写入文件。 |
| **close** | `close(self)` | 关闭文件句柄。 |
