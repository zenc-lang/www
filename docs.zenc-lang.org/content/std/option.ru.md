# std/option

`Option<T>` представляет необязательное значение: каждый `Option` является либо `Some` и содержит значение, либо `None`. Он часто используется для обработки отсутствия значения без обращения к нулевым указателям.

## Обзор

- **Безопасность**: Поощряет явную обработку случая `None`.
- **Универсальность**: Может оборачивать любой тип `T`.
- **Нулевая стоимость**: Компилируется в простую структуру с логическим флагом.
- **Удобство**: Предоставляет множество вспомогательных методов для извлечения и преобразования значений.

## Использование

```zc
import "std/option.zc"

fn main() {
    let val = Option<int>::Some(10);
    
    if (val.is_some()) {
        println "Значение: {val.unwrap()}";
    }
    
    let empty = Option<int>::None();
    let x = empty.unwrap_or(0);
}
```

## Определение структуры

```zc
struct Option<T> {
    is_some: bool;
    val: T;
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **Some** | `Option<T>::Some(v: T) -> Option<T>` | Создает вариант `Some`, содержащий `v`. |
| **None** | `Option<T>::None() -> Option<T>` | Создает вариант `None`. |

### Запросы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_some** | `is_some(self) -> bool` | Возвращает `true`, если опция содержит `Some`. |
| **is_none** | `is_none(self) -> bool` | Возвращает `true`, если опция содержит `None`. |

### Извлечение

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **unwrap** | `unwrap(self) -> T` | Возвращает содержащееся значение. Вызывает панику, если `None`. |
| **unwrap_ref** | `unwrap_ref(self) -> T*` | Возвращает указатель на содержащееся значение. Вызывает панику, если `None`. |
| **unwrap_or** | `unwrap_or(self, def: T) -> T` | Возвращает содержащееся значение или `def`. |
| **expect** | `expect(self, msg: char*) -> T` | Возвращает значение или вызывает панику с сообщением `msg`. |
| **or_else** | `or_else(self, other: Option<T>) -> Option<T>` | Возвращает опцию, если это `Some`, иначе возвращает `other`. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **forget** | `forget(self)` | Обнуляет внутреннее значение без вызова деструкторов или освобождения памяти. |
