# std/option

`Option<T>` представляет необязательное значение: каждый `Option` является либо `Some`, содержащим значение, либо `None`. Широко используется для обработки отсутствия значения без использования нулевых указателей.

## Обзор

- **Безопасность**: Стимулирует явную обработку случая `None`.
- **Обобщенность**: Может содержать любой тип `T`.
- **Нулевая стоимость**: Компилируется в простую структуру с булевым флагом.
- **Удобство**: Предоставляет множество вспомогательных методов для извлечения и трансформации значений.

## Использование

```zc
import "std/option.zc"

fn main() {
    let val = Option<int>::Some(10);
    
    if (val.is_some()) {
        println "Значение: {val.unwrap()}";
    }
    
    let empty = Option<int>::None();
    let x = empty.unwrap_or(0);
}
```

## Определение структуры

```zc
struct Option<T> {
    is_some: bool;
    val: T;
}
```

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **Some** | `Option<T>::Some(v: T) -> Option<T>` | Создает вариант `Some`, содержащий `v`. |
| **None** | `Option<T>::None() -> Option<T>` | Создает вариант `None`. |

### Проверка

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_some** | `is_some(self) -> bool` | Возвращает `true`, если это `Some`. |
| **is_none** | `is_none(self) -> bool` | Возвращает `true`, если это `None`. |

### Извлечение

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **unwrap** | `unwrap(self) -> T` | Возвращает содержащееся значение. Вызывает панику, если это `None`. |
| **unwrap_ref** | `unwrap_ref(self) -> T*` | Возвращает указатель на содержащееся значение. Паникует, если это `None`. |
| **unwrap_or** | `unwrap_or(self, def: T) -> T` | Возвращает значение или `def`, если это `None`. |
| **expect** | `expect(self, msg: char*) -> T` | Возвращает значение или паникует с сообщением `msg`. |
| **or_else** | `or_else(self, other: Option<T>) -> Option<T>` | Возвращает текущий вариант, если это `Some`, иначе возвращает `other`. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **forget** | `forget(self)` | Очищает внутреннее значение без вызова деструкторов или освобождения памяти. |
