# std/hex

`std/hex` 模組提供用於在二進制數據和十六進制字符串表示之間進行轉換的工具。

## 使用

```zc
import "std/hex.zc"

fn main() {
    let data: u8[3] = {0x41, 0x42, 0x43}; // "ABC"
    
    // 編碼為十六進制
    let s = Hex::encode(&data[0], 3);
    println "十六進制: {s}"; // "414243"
    
    // 解碼
    let decoded = Hex::decode("414243");
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **encode** | `Hex::encode(data: u8*, n: usize) -> String` | 將字節數組轉換為小寫的十六進制字符串。 |
| **decode** | `Hex::decode(s: char*) -> Vec<u8>` | 將十六進制字符串解碼為字節。如果遇到無效字符，則會出錯。 |
