# std/result

`Result<T>` - это стандартный тип для обработки ошибок в Zen-C. Он представляет собой либо успех (`Ok`), содержащий значение типа `T`, либо неудачу (`Err`), содержащую строковое сообщение об ошибке.

## Обзор

- **Безопасность**: Принуждает к явной обработке путей успеха и неудачи.
- **Информативность**: Случаи `Err` содержат описательное сообщение об ошибке.
- **Универсальность**: Поддерживает любой тип значения успеха `T`.
- **Интегрированность**: Бесшовно работает с макросами и паттернами на основе `Result` для краткого распространения ошибок.

## Использование

```zc
import "std/result.zc"

fn divide(a: int, b: int) -> Result<int> {
    if (b == 0) {
        return Result<int>::Err("Деление на ноль");
    }
    return Result<int>::Ok(a / b);
}

fn main() {
    match divide(10, 0) {
        Ok(val) => println "Результат: {val}",
        Err(e)  => println "Ошибка: {e}"
    }
}
```

## Определение структуры

```zc
struct Result<T> {
    is_ok: bool;
    val: T;
    err: char*;
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **Ok** | `Result<T>::Ok(v: T) -> Result<T>` | Создает результат успеха, содержащий `v`. |
| **Err** | `Result<T>::Err(e: char*) -> Result<T>` | Создает результат ошибки с сообщением `e`. |

### Запросы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_ok** | `is_ok(self) -> bool` | Возвращает `true`, если результат - `Ok`. |
| **is_err** | `is_err(self) -> bool` | Возвращает `true`, если результат - `Err`. |

### Извлечение

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **unwrap** | `unwrap(self) -> T` | Возвращает значение ok. Вызывает панику с сообщением об ошибке, если это `Err`. |
| **unwrap_ref** | `unwrap_ref(self) -> T*` | Возвращает указатель на значение ok. Вызывает панику, если это `Err`. |
| **expect** | `expect(self, msg: char*) -> T` | Возвращает значение или вызывает панику с `msg` и сообщением об ошибке, если это `Err`. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **forget** | `forget(self)` | Обнуляет значение ok без вызова деструкторов или освобождения памяти. |
