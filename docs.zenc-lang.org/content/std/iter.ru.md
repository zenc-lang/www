# std/iter

Модуль `std/iter` предоставляет базовые типы и трейты для поддержки механизма итерации `for-in` в Zen-C.

## Обзор

Zen-C использует концепцию итераторов для безопасного и эффективного обхода коллекций (`Vec`, `Map`, `Queue` и т.д.) или диапазонов.

## Использование

```zc
import "std/iter.zc"

fn main() {
    // Итерация по диапазону
    for i in Range::new(0, 5) {
        println "{i}"; // 0, 1, 2, 3, 4
    }
}
```

## Структура `Range`

Представляет числовой диапазон от `start` (включительно) до `end` (исключительно).

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Range::new(start: int, end: int) -> Range` | Создает новый диапазон. |
| **iterator** | `iterator(self) -> RangeIter` | Возвращает итератор для диапазона. |

## Концепции итератора

Объект итератора должен следовать интерфейсу, который понимает цикл `for-in`:

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **next** | `next(self) -> Option<T>` | Возвращает следующий элемент, обернутый в `Option`, или `None` по окончании. |

## Трейт `Iterable`

Коллекции, которые могут быть использованы в цикле `for-in`, должны реализовывать метод:

```zc
fn iterator(self) -> IteratorType
```

Где `IteratorType` реализует метод `next`.
