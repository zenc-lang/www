# std/mman

`std/mman` 模組提供對內存映射（Memory Mapping）工具的訪問，允許程序將文件或設備映射到內存地址空間。

## 使用

```zc
import "std/mman.zc"
import "std/fs.zc"

fn main() {
    let f = File::open("data.bin");
    let size = f.size();
    
    // 映射文件到內存
    let ptr = Mman::map(null, size, Prot::Read, MapFlags::Shared, f.handle(), 0);
    
    if (ptr != null) {
        println "文件已映射到 {ptr}";
        Mman::unmap(ptr, size);
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **map** | `Mman::map(addr: void*, len: usize, prot: int, flags: int, fd: int, offset: usize) -> void*` | 創建一個新的內存映射。 |
| **unmap** | `Mman::unmap(addr: void*, len: usize) -> int` | 刪除指定範圍內的映射。 |
| **protect** | `Mman::protect(addr: void*, len: usize, prot: int) -> int` | 設置內存區域的保護權限。 |
| **lock** | `Mman::lock(addr: void*, len: usize) -> int` | 將內存區域鎖定在 RAM 中，防止被交換。 |
