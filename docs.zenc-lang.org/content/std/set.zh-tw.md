# std/set

`Set<T>` 是一個用於存儲唯一值的集合，內部基於哈希表實現。

## 使用

```zc
import "std/set.zc"

fn main() {
    let s = Set<int>::new();
    
    s.insert(10);
    s.insert(20);
    s.insert(10); // 重複值將被忽略
    
    println "大小: {s.length()}"; // 2
    
    if (s.contains(10)) {
        println "集合包含 10";
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Set<T>::new() -> Set<T>` | 創建一個新的空集合。 |
| **insert** | `insert(self, val: T)` | 將值添加到集合中（如果尚不存在）。 |
| **remove** | `remove(self, val: T)` | 從集合中移除指定值。 |
| **contains**| `contains(self, val: T) -> bool` | 如果值存在於集合中，則返回 true。 |
| **length** | `length(self) -> usize` | 返回集合中的元素數量。 |
| **clear** | `clear(self)` | 移除集合中的所有元素。 |
