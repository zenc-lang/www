# std/bigfloat

`BigFloat` обеспечивает десятичную арифметику с плавающей запятой произвольной точности для Zen-C. Он реализован как масштабируемый `BigInt`, что позволяет выполнять высокоточные вычисления без ошибок двоичного округления.

## Обзор

- **Произвольная точность**: Поддерживает десятичные числа любого размера, ограниченные только памятью.
- **Масштабируемое представление**: Использует величину `BigInt` и целое число `scale` для представления десятичных значений.
- **Управление точностью**: Легкое выравнивание масштабов для точного сложения и вычитания.
- **RAII**: Память для подлежащей величины управляется автоматически с помощью трейта `Drop`.

## Использование

```zc
import "std/bigfloat.zc"

fn main() {
    let a = BigFloat::from_int(123);
    a.scale = 2; // Представляет 1.23
    
    let b = BigFloat::from_int(4567);
    b.scale = 3; // Представляет 4.567
    
    let sum = a.add(b);
    
    let s = sum.to_string();
    println "Сумма: {s}"; // Сумма: 5.797
    free(s);
} // a, b и sum освобождаются здесь автоматически
```

## Определение структуры

```zc
struct BigFloat {
    magnitude: BigInt;
    scale: int;
}
```

## Методы

### Конструкторы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `BigFloat::new() -> BigFloat` | Создает новый `BigFloat`, инициализированный значением 0.0. |
| **from_int** | `BigFloat::from_int(val: u64) -> BigFloat` | Создает `BigFloat` из целого числа с масштабом 0. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **add** | `add(self, other: BigFloat) -> BigFloat` | Складывает два значения `BigFloat`, автоматически выравнивая их масштабы. Возвращает новый `BigFloat`. |
| **align_scale** | `align_scale(self, target_scale: int)` | Увеличивает масштаб `BigFloat` до `target_scale` путем сдвига величины. |

### Утилиты

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **clone** | `clone(self) -> BigFloat` | Возвращает глубокую копию `BigFloat`. |
| **to_string** | `to_string(self) -> char*` | Возвращает строковое представление с десятичной запятой, выделенное в куче. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free_mem** | `free_mem(self)` | Вручную освобождает подлежащую память `BigInt`. |
| **Trait** | `impl Drop for BigFloat` | Автоматически вызывает `free_mem()`, когда объект выходит за пределы области видимости. |
