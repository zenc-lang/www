# std/info

`std/info` 模組提供有關當前系統環境、架構和操作系統的元數據。

## 使用

```zc
import "std/info.zc"

fn main() {
    println "操作系統: {Info::os()}";
    println "架構: {Info::arch()}";
    
    if (Info::is_debug()) {
        println "在偵錯模式下運行";
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **os** | `Info::os() -> char*` | 返回操作系統名稱 (例如 "linux", "windows", "macos")。 |
| **arch** | `Info::arch() -> char*` | 返回 CPU 架構 (例如 "x86_64", "arm64")。 |
| **is_debug** | `Info::is_debug() -> bool` | 如果程序是使用偵錯符號編譯的，則返回 true。 |
| **is_release**| `Info::is_release() -> bool` | 如果程序是針對發佈模式編譯的，則返回 true。 |
| **version** | `Info::version() -> char*` | 返回當前 Zen-C 編譯器的版本。 |
