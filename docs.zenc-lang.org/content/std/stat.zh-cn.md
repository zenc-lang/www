# std/sys/stat

`std/sys/stat` 模块提供用于检索文件元数据（如大小、权限和修改时间）的工具。它直接包装了 POSIX `sys/stat.h`。

## 使用

```zc
import "std/sys/stat.zc"

fn main() {
    let s = Stat::new();
    
    if (Stat::path("file.txt", &s) == 0) {
        println "文件大小: {s.size} 字节";
        
        if (s.is_dir()) {
            println "这是一个目录";
        }
    }
}
```

## Stat 结构体

`Stat` 结构体包含文件的元数据：

| 字段 | 类型 | 描述 |
| :--- | :--- | :--- |
| **size** | `i64` | 总大小，以字节为单位。 |
| **mode** | `u32` | 保护模式和文件类型。 |
| **mtime** | `i64` | 最后修改时间。 |
| **atime** | `i64` | 最后访问时间。 |

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **path** | `Stat::path(path: char*, buf: Stat*) -> int` | 获取指定路径文件的统计信息。 |
| **is_dir** | `is_dir(self) -> bool` | 如果是目录，则返回 true。 |
| **is_reg** | `is_reg(self) -> bool` | 如果是常规文件，则返回 true。 |
