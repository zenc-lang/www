# std/crypto/sha256

El módulo `std/crypto/sha256` implementa el algoritmo de hash criptográfico SHA-256 según lo especificado en FIPS 180-4.

## Resumen

- **Hashing**: Calcula hashes de 256 bits de cadenas arbitrarias o datos de bytes.
- **Hashing Incremental**: Procesa los datos en trozos utilizando `update` y `finalize`.
- **Salida Hexadecimal**: Conveniente ayudante `hash` que devuelve una cadena hexadecimal.

## Uso

```zc
import "std/crypto/sha256.zc"
import "std/io.zc"

fn main() {
    // Hashing simple
    let hash = Sha256::hash("hola mundo");
    println "SHA-256: {hash}";
    
    // Hashing incremental
    let ctx = Sha256::new();
    ctx.update((u8*)"hola", 4);
    ctx.update((u8*)" mundo", 6);
    
    let digest: u8[32];
    ctx.finalize(&digest[0]);
}
```

## Métodos

### Métodos de `Sha256`

| Método | Firma | Descripción |
| :--- | :--- | :--- |
| **new** | `Sha256::new() -> Sha256` | Inicializa un nuevo contexto SHA-256. |
| **update** | `update(self, data: u8*, len: usize)` | Actualiza el contexto con un trozo de datos. |
| **finalize** | `finalize(self, hash: u8*)` | Finaliza el hash y escribe el resultado en el búfer de 32 bytes proporcionado. |
| **hash** | `Sha256::hash(data: char*) -> String` | Método de conveniencia para calcular el hash de una cadena y devolver su representación hexadecimal. |
