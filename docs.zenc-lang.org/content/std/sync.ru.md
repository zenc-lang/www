# std/sync

Модуль `std/sync` предоставляет полный набор примитивов синхронизации для управления конкурентным доступом к общим данным и координации выполнения потоков.

## Обзор

- **Стандартные примитивы**: Включает `Mutex`, `CondVar`, `RwLock`, `Once`, `Semaphore` и `Barrier`.
- **Интеграция с RAII**: Все примитивы реализуют трейт `Drop`, гарантируя автоматическое освобождение системных ресурсов (например, дескрипторов pthread).
- **Кроссплатформенность**: Безопасно абстрагирует различия платформ.
- **Эффективность**: Тонкие обертки над оптимизированными системными библиотеками синхронизации.

## Использование

```zc
import "std/sync.zc"

fn main() {
    let m = Mutex::new();
    
    // Блокировка с ограниченной областью видимости (RAII)
    {
        m.lock();
        // Критическая секция
        m.unlock();
    } // m автоматически очищается, если это последний владелец
    
    // Однократная инициализация
    let once = Once::new();
    once.call(|| {
        println "Инициализировано!";
    });
}
```

## Определения структур

### `Mutex`
Мьютекс (взаимное исключение) для защиты общих данных.

### `CondVar`
Условная переменная для передачи сигналов между потоками на основе изменения состояния.

### `RwLock`
Блокировка чтения-записи, позволяющая иметь несколько одновременных читателей, но только одного писателя.

### `Once`
Гарантирует, что определенный код инициализации будет выполнен ровно один раз.

## Методы

### Методы `Mutex`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Mutex::new() -> Mutex` | Создает новый мьютекс. |
| **lock** | `lock(self)` | Захватывает блокировку (блокирует поток). |
| **try_lock** | `try_lock(self) -> bool` | Пытается захватить блокировку без ожидания. |
| **unlock** | `unlock(self)` | Освобождает блокировку. |

### Методы `CondVar`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `CondVar::new() -> CondVar` | Создает новую условную переменную. |
| **wait** | `wait(self, mutex: Mutex*)` | Ожидает сигнала, временно освобождая мьютекс. |
| **signal** | `signal(self)` | Пробуждает один поток, ожидающий на этой переменной. |
| **broadcast**| `broadcast(self)` | Пробуждает все потоки, ожидающие на этой переменной. |
