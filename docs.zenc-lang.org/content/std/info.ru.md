# std/info

Модуль `std/info` предоставляет доступ к метаданным и информации о компиляторе Zen-C и текущей среде выполнения.

## Использование

```zc
import "std/info.zc"

fn main() {
    println "Компилятор: {Info::compiler_name()}";
    println "Версия: {Info::version()}";
    
    if (Info::is_debug()) {
        println "Запущено в отладочном режиме.";
    }
}
```

## Методы информации о компиляторе

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **version** | `Info::version() -> char*` | Возвращает текущую версию компилятора. |
| **compiler_name** | `Info::compiler_name() -> char*` | Возвращает имя компилятора (например, "zenc"). |
| **build_date** | `Info::build_date() -> char*` | Возвращает дату сборки компилятора. |

## Диагностика времени выполнения

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_debug** | `Info::is_debug() -> bool` | Возвращает true, если код скомпилирован с отладочными символами. |
| **is_release** | `Info::is_release() -> bool` | Возвращает true, если код скомпилирован с оптимизациями для релиза. |
