# std/arena

Модуль `std/arena` предоставляет быстрый "bump" аллокатор для массового выделения памяти. Вся память, выделенная внутри арены, освобождается одновременно при уничтожении или сбросе самой арены.

## Обзор

- **Производительность**: Выделение памяти происходит чрезвычайно быстро, включая лишь одно инкрементирование указателя.
- **Массовое освобождение**: Идеально подходит для локальных задач или запросов, где всё можно освободить одновременно.
- **Сохранение/Восстановление**: Поддержка "контрольных точек" для частичного освобождения памяти внутри арены.
- **RAII**: Реализует трейт `Drop` для автоматического освобождения подлежащего буфера.

## Использование

```zc
import "std/arena.zc"

struct Node {
    val: int;
}

fn main() {
    // Создание арены емкостью 1 КБ
    let a = Arena::new(1024);
    
    // Быстрое выделение памяти
    let n1 = a.alloc<Node>();
    let n2 = a.alloc<Node>();
    
    // Дублирование строки в память арены
    let s = a.dup_str("Hello World");
    
    println "Использовано арены: {a.bytes_used()} байт";
    
    // Всё освобождается автоматически, когда 'a' выходит за пределы области видимости
}
```

## Определение структуры

```zc
struct Arena {
    data: void*;
    capacity: usize;
    used: usize;
}
```

## Методы

### Конструкторы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Arena::new(cap: usize) -> Arena` | Создает новую арену указанной емкости. |

### Выделение памяти

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **alloc\<T>** | `alloc<T>(self) -> T*` | Выделяет и инициализирует нулями память для типа `T`. |
| **alloc_n\<T>**| `alloc_n<T>(self, count: usize) -> T*` | Выделяет память для массива из `count` элементов типа `T`. |
| **alloc_bytes**| `alloc_bytes(self, size: usize) -> void*` | Выделение сырых байтов (выровнено по 8 байт). |
| **dup_str** | `dup_str(self, src: char*) -> char*` | Дублирует C-строку в память арены. |

### Запросы и управление

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **bytes_used** | `bytes_used(self) -> usize` | Возвращает общее количество выделенных байт. |
| **bytes_free** | `bytes_free(self) -> usize` | Возвращает оставшуюся емкость. |
| **save** | `save(self) -> usize` | Возвращает "контрольную точку", представляющую текущее использование. |
| **restore** | `restore(self, mark: usize)` | Частично освобождает память до предыдущей контрольной точки. |
| **reset** | `reset(self)` | Освобождает все выделения, сбрасывая использование до нуля (буфер сохраняется). |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Явно освобождает подлежащий буфер арены. |
| **Trait** | `impl Drop for Arena` | Автоматически вызывает `free()`, когда арена выходит за пределы области видимости. |
