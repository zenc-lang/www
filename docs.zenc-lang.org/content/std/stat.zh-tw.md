# std/fs/stat

`std/fs/stat` 模組提供用於檢索有關文件系統對象（文件、目錄、鏈接）元數據的工具，例如大小、權限和修改時間。

## 使用

```zc
import "std/fs/stat.zc"

fn main() {
    match Stat::file("config.json") {
        Ok(info) => {
            println "文件大小: {info.size} 字節";
            if (info.is_dir()) {
                println "這是一個目錄";
            }
        },
        Err(e) => println "無法獲取狀態: {e}"
    }
}
```

## Stat 結構體

| 字段 | 類型 | 描述 |
| :--- | :--- | :--- |
| **size** | `usize` | 文件大小（以字節為單位）。 |
| **mode** | `u32` | 文件權限和類型標誌。 |
| **mtime** | `u64` | 最後修改時間（Unix 時間戳）。 |
| **ctime** | `u64` | 狀態最後更改時間。 |

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **is_file** | `is_file(self) -> bool` | 如果是對象是常規文件，返回 true。 |
| **is_dir** | `is_dir(self) -> bool` | 如果對象是目錄，返回 true。 |
| **is_link** | `is_link(self) -> bool` | 如果對象是符號鏈接，返回 true。 |
