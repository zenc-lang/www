# std/complex

Библиотека `std/complex` предоставляет структуру `Complex` и основные математические операции для работы с комплексными числами в Zen-C.

## Обзор

- **Тип значения**: Простая структура с компонентами `real` (действительная часть) и `imag` (мнимая часть).
- **Поддержка операторов**: Поддерживает `+`, `-`, `*`, `/`, `==` и `!=` через перегрузку операторов.
- **Свойства**: Предоставляет методы для вычисления модуля и фазы.
- **Интерполяция**: Может напрямую использоваться в f-строках и инструкциях print.

## Использование

```zc
import "std/complex.zc"

fn main() {
    let c1 = Complex::new(3.0, 4.0);
    let c2 = Complex::new(1.0, 2.0);
    
    let sum = c1 + c2;
    let prod = c1 * c2;
    
    println "Сумма: {sum}";       // Сумма: 4.000000 + 6.000000i
    println "Модуль: {c1.magnitude()}";
}
```

## Определение структуры

```zc
struct Complex {
    real: double;
    imag: double;
}
```

## Методы

### Конструкторы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Complex::new(r: double, i: double) -> Complex` | Создает новое комплексное число с действительной частью `r` и мнимой частью `i`. |

### Доступ и запросы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **magnitude** | `magnitude(self) -> double` | Возвращает модуль (абсолютное значение) комплексного числа. |
| **phase** | `phase(self) -> double` | Возвращает фазу (угол) в радианах. |

## Операторы

| Оператор | Метод | Описание |
| :--- | :--- | :--- |
| `+` | **add** | Складывает два комплексных числа. |
| `-` | **sub** | Вычитает одно комплексное число из другого. |
| `*` | **mul** | Перемножает два комплексных числа. |
| `/` | **div** | Делит одно комплексное число на другое. |
| `==` | **eq** | Проверяет, равны ли два комплексных числа. |
| `!=` | **neq** | Проверяет, не равны ли два комплексных числа. |
| `{}` | **to_string** | Обеспечивает прямую интерполяцию строк. |
