# std/crypto/sha256

`std/crypto/sha256` 模組提供 SHA-256 哈希算法的實現。

## 使用

```zc
import "std/crypto/sha256.zc"

fn main() {
    let data = "Zen-C";
    let hash = Sha256::hash(data);
    
    println "SHA-256 哈希值: {hash}";
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **hash** | `Sha256::hash(data: char*) -> String` | 返回字符串的十六進制哈希表示。 |
| **bytes** | `Sha256::bytes(data: u8*, n: usize) -> u8[32]`| 返回原始的 32 字節哈希。 |

## Sha256 結構體

對於大數據或流數據處理，可以使用結構體接口：

```zc
let ctx = Sha256::new();
ctx.update("第一部分");
ctx.update("第二部分");
let final_hash = ctx.finalize();
```
