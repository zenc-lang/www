# std/crypto/sha256

`std/crypto/sha256` 模組實現了 FIPS 180-4 中指定的 SHA-256 加密雜湊（hash）演算法。

## 概覽

- **雜湊（Hashing）**：計算任意字串或位元組數據的 256 位元雜湊。
- **增量雜湊**：使用 `update` 和 `finalize` 分塊處理數據。
- **十六進位輸出**：便捷的 `hash` 輔助函數，返回一個十六進位字串。

## 使用方法

```zc
import "std/crypto/sha256.zc"
import "std/io.zc"

fn main() {
    // 簡單雜湊
    let hash = Sha256::hash("hello world");
    println "SHA-256: {hash}";
    
    // 增量雜湊
    let ctx = Sha256::new();
    ctx.update((u8*)"hello", 5);
    ctx.update((u8*)" world", 6);
    
    let digest: u8[32];
    ctx.finalize(&digest[0]);
}
```

## 結構體定義

```zc
struct Sha256 {
    state: u32[8];
    buffer: u8[64];
    datalen: u32;
    bitlen: u64;
}
```

## 方法

### `Sha256` 方法

| 方法 | 簽名 | 說明 |
| :--- | :--- | :--- |
| **new** | `Sha256::new() -> Sha256` | 初始化一個新的 SHA-256 上下文。 |
| **update** | `update(self, data: u8*, len: usize)` | 使用數據塊更新上下文。 |
| **finalize** | `finalize(self, hash: u8*)` | 完成雜湊計算並將結果寫入提供的 32 位元組緩衝區中。 |
| **hash** | `Sha256::hash(data: char*) -> String` | 對字串進行雜湊處理並返回其十六進位表示的便捷方法。 |
