# std/env

`std/env` 模块提供对进程环境的访问，包括命令行参数和环境变量。

## 使用

```zc
import "std/env.zc"

fn main() {
    // 获取环境变量
    let path = Env::get("PATH");
    if (path.is_some()) {
        println "PATH 是: {path.unwrap()}";
    }

    // 处理命令行参数
    let args = Env::args();
    println "已传递 {args.length()} 个参数";
}
```

## 方法

### 环境变量

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **get** | `Env::get(name: char*) -> Option<String>` | 检索环境变量的值。 |
| **set** | `Env::set(name: char*, value: char*)` | 设置或更新环境变量。 |
| **remove**| `Env::remove(name: char*)` | 删除环境变量。 |

### 进程信息

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **args** | `Env::args() -> Vec<String>` | 返回传递给进程的命令行参数列表。 |
| **cwd** | `Env::cwd() -> String` | 返回当前工作目录。 |
| **set_cwd**| `Env::set_cwd(path: char*) -> bool` | 更改当前工作目录。 |
