# std/stack

`Stack<T>` — это обобщенная структура данных, работающая по принципу "последним пришел — первым ушел" (LIFO), реализованная как обертка над `Vec<T>`.

## Использование

```zc
import "std/stack.zc"

fn main() {
    let s = Stack<int>::new();
    
    s.push(10);
    s.push(20);
    
    println "Верхний: {s.peek().unwrap()}"; // 20
    
    let val = s.pop().unwrap(); // 20
}
```

## Определение структуры

```zc
struct Stack<T> {
    vec: Vec<T>;
}
```

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Stack<T>::new() -> Stack<T>` | Создает новый пустой стек. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **push** | `push(self, val: T)` | Добавляет элемент на вершину стека. |
| **pop** | `pop(self) -> Option<T>` | Удаляет и возвращает верхний элемент. Возвращает `None`, если стек пуст. |
| **clear** | `clear(self)` | Удаляет все элементы из стека. |

### Доступ и свойства

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **peek** | `peek(self) -> Option<T>` | Возвращает копию верхнего элемента без его удаления. |
| **peek_ref**| `peek_ref(self) -> T*` | Возвращает указатель на верхний элемент. |
| **length** | `length(self) -> usize` | Возвращает количество элементов в стеке. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает true, если стек пуст. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Освобождает память внутреннего вектора. |
| **Trait** | `impl Drop for Stack` | Автоматически вызывает `free()` при выходе из области видимости. |
