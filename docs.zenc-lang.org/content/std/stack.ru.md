# std/stack

Модуль `std/stack` предоставляет структуру данных стека LIFO (Last-In, First-Out - последним пришел, первым ушел).

## Использование

```zc
import "std/stack.zc"

fn main() {
    let s = Stack<int>::new();
    s.push(10);
    s.push(20);
    
    let top = s.pop(); // Some(20)
} // s освобождается здесь автоматически
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Stack<T>::new() -> Stack<T>` | Создает новый пустой стек. |
| **clone** | `clone(self) -> Stack<T>` | Создает глубокую копию стека. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **push** | `push(self, value: T)` | Помещает значение на вершину стека. |
| **pop** | `pop(self) -> Option<T>` | Удаляет и возвращает верхний элемент стека. Возвращает `None`, если стек пуст. |
| **clear** | `clear(self)` | Удаляет все элементы из стека. |

### Доступ и запроры

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **length** | `length(self) -> usize` | Возвращает количество элементов в стеке. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает `true`, если стек не содержит элементов. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает память стека. |
| **Trait** | `impl Drop for Stack` | Автоматически вызывает `free()`, когда стек выходит из области видимости. |
