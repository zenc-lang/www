# std/thread

`std/thread` 模組提供用於創建和管理操作系統線程的工具，支持在 Zen-C 中進行並發編程。

## 使用

```zc
import "std/thread.zc"

fn worker(arg: void*) {
    println "線程正在運行";
}

fn main() {
    // 創建一個新線程
    let t = Thread::spawn(worker, null);
    
    // 等待線程結束
    t.join();
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **spawn** | `Thread::spawn(func: fn(void*), arg: void*) -> Thread` | 啟動一個新線程並開始執行指定的函數。 |
| **join** | `join(self)` | 阻塞當前線程，直到對象線程終止。 |
| **detach** | `detach(self)` | 允許線程獨立運行，結束後自動釋放資源。 |
| **id** | `id(self) -> u64` | 返回線程的唯一標識符。 |
| **sleep** | `Thread::sleep(ms: u64)` | 使當前線程暫停指定的毫秒數。 |
