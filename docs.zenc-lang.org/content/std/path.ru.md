# std/path

Модуль `std/path` предоставляет кроссплатформенные утилиты для работы с путями файловой системы. Упрощает такие задачи, как объединение путей, извлечение расширений и поиск родительских директорий.

## Обзор

- **Кроссплатформенность**: Корректно обрабатывает прямые (/) и обратные (\) слеши.
- **Типобезопасность**: Структура `Path` инкапсулирует информацию о пути, отличая её от обычных строк.
- **Удобный парсинг**: Легко извлекайте компоненты, такие как `extension` (расширение), `file_name` (имя файла) и `parent` (родитель).
- **RAII**: Память управляется автоматически с помощью трейта `Drop`.

## Использование

```zc
import "std/path.zc"

fn main() {
    let p = Path::new("/home/user");
    let full_path = p.join("docs/file.txt");
    
    println "Полный путь: {full_path.c_str()}";
    
    match full_path.extension() {
        Some(ext) => println "Расширение: {ext}",
        None => println "Расширение не найдено"
    }
} // full_path и p освобождаются автоматически здесь
```

## Определение структуры

```zc
struct Path {
    str: String;
}
```

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Path::new(s: char*) -> Path` | Создает новый `Path` из C-строки. |
| **from_string** | `Path::from_string(s: String) -> Path` | Создает `Path`, принимая владение строкой `String`. |
| **clone** | `clone(self) -> Path` | Возвращает глубокую копию (deep copy) пути. |

### Манипуляция

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **join** | `join(self, other: char*) -> Path` | Добавляет `other` к пути, используя корректный разделитель директорий. |

### Парсинг

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **extension** | `extension(self) -> Option<String>` | Возвращает расширение файла (без точки), если оно есть. |
| **file_name** | `file_name(self) -> Option<String>` | Возвращает завершающий компонент пути. |
| **parent** | `parent(self) -> Option<Path>` | Возвращает путь к родительской директории. |

### Доступ

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **c_str** | `c_str(self) -> char*` | Возвращает строковое представление пути в стиле C. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает внутреннюю память пути. |
| **Trait** | `impl Drop for Path` | Автоматически вызывает `free()` при выходе из области видимости. |
