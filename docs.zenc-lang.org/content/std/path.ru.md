# std/path

Модуль `std/path` предоставляет кроссплатформенные утилиты для манипулирования путями файловой системы. Он упрощает общие задачи, такие как объединение путей, извлечение расширений и поиск родительских директорий.

## Обзор

- **Кроссплатформенность**: Корректно обрабатывает как прямые, так и обратные косые черты во время манипуляций.
- **Типобезопасность**: Структура `Path` инкапсулирует информацию о пути, отличая ее от обычных строк.
- **Удобный парсинг**: Легко извлекайте компоненты, такие как `extension`, `file_name` и `parent`.
- **RAII**: Память управляется автоматически с помощью трейта `Drop`.

## Использование

```zc
import "std/path.zc"

fn main() {
    let p = Path::new("/home/user");
    let full_path = p.join("docs/file.txt");
    
    println "Полный путь: {full_path.c_str()}";
    
    match full_path.extension() {
        Some(ext) => println "Расширение: {ext}",
        None => println "Расширение не найдено"
    }
} // full_path и p освобождаются здесь автоматически
```

## Определение структуры

```zc
struct Path {
    str: String;
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Path::new(s: char*) -> Path` | Создает новый `Path` из C-строки. |
| **from_string** | `Path::from_string(s: String) -> Path` | Создает `Path`, принимая владение объектом `String`. |
| **clone** | `clone(self) -> Path` | Возвращает глубокую копию `Path`. |

### Манипуляция

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **join** | `join(self, other: char*) -> Path` | Добавляет `other` к пути, используя правильный разделитель директорий. |

### Парсинг

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **extension** | `extension(self) -> Option<String>` | Возвращает расширение файла (без точки в начале), если оно есть. |
| **file_name** | `file_name(self) -> Option<String>` | Возвращает последний компонент пути. |
| **parent** | `parent(self) -> Option<Path>` | Возвращает путь к родительской директории. |

### Доступ

| Метод | Signature | Описание |
| :--- | :--- | :--- |
| **c_str** | `c_str(self) -> char*` | Возвращает внутреннее представление в виде C-строки. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает память внутренней строки пути. |
| **Trait** | `impl Drop for Path` | Автоматически вызывает `free()`, когда объект выходит за пределы области видимости. |
