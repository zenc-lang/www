# std/mem

`std/mem` 模組提供底層的內存操作工具，如複製、設置和比較內存塊。

## 使用

```zc
import "std/mem.zc"

fn main() {
    let src: u8[3] = {1, 2, 3};
    let dst: u8[3];
    
    // 複製內存
    Mem::copy(&dst[0], &src[0], 3);
    
    // 比較內存
    if (Mem::eq(&dst[0], &src[0], 3)) {
        println "內存內容一致";
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **copy** | `Mem::copy(dst: void*, src: void*, n: usize)` | 將 n 個字節從 src 複製到 dst。 |
| **set** | `Mem::set(dst: void*, val: u8, n: usize)` | 將 dst 的前 n 個字節設置為 val。 |
| **eq** | `Mem::eq(a: void*, b: void*, n: usize) -> bool` | 如果兩個內存塊的前 n 個字節相同，則返回 true。 |
| **alloc / free**| `Mem::alloc(n: usize) -> void*` | 基本的堆分配和釋放（通常由內置 `new` 調用）。 |
