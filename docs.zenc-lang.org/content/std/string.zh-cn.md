# std/string

`String` 模块提供可动态增长的 UTF-8 字符串实现，并支持自动内存管理。

## 使用

```zc
import "std/string.zc"

fn main() {
    let s = String::new("你好");
    s.append(", ")
    s.append("Zen-C");
    
    println "{s}"; // "你好, Zen-C"
    
    if (s.contains("Zen")) {
        println "找到 Zen";
    }
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `String::new(init: char*) -> String` | 从 C 字符串创建新 String。 |
走
| **with_capacity**| `String::with_capacity(n: usize) -> String` | 创建具有预分配内存的空 String。 |

### 修改

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **append** | `append(self, other: char*)` | 在末尾追加字符串。 |
| **push** | `push(self, c: char)` | 在末尾追加单个字符。 |
| **clear** | `clear(self)` | 清空内容但不释放内存。 |

### 搜索与转换

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **contains** | `contains(self, sub: char*) -> bool` | 如果包含子字符串，则返回 true。 |
| **find** | `find(self, sub: char*) -> int` | 返回子字符串的起始索引，未找到则返回 -1。 |
| **to_c_string**| `to_c_string(self) -> char*` | 返回原始 C 指针。 |
| **length** | `length(self) -> usize` | 返回字符数量。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 显式释放分配的内存。 |
| **Trait** | `impl Drop for String` | 当 String 离开作用域时由编译器自动调用 `free()`。 |
