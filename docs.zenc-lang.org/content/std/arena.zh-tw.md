# std/arena

`std/arena` 模組提供了一個區域分配器（Arena Allocator），用於高效地管理具有相同生命週期的對象集合。

## 使用

```zc
import "std/arena.zc"

fn main() {
    // 創建一個新的區域分配器
    let a = Arena::new();
    
    // 從區域分配內存
    let ptr = a.alloc(1024);
    
    // 一次性釋放所有分配的內存
    a.free();
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Arena::new() -> Arena` | 初始化一個新的區域分配器。 |
| **alloc** | `alloc(self, size: usize) -> void*` | 在區域中分配指定大小的內存。 |
| **free** | `free(self)` | 釋放與該區域關聯的所有內存。 |
| **reset** | `reset(self)` | 重置分配器以供重用，而不釋放底層內存塊。 |

## 優點

- **效能**: 分配速度極快（僅指針遞增）。
- **簡單**: 不需要手動釋放單個對象；一次性釋放整個區域。
- **無碎片**: 避免了長期運行程序中常見的堆碎片問題。
