# std/utf8

`std/utf8` 模块提供验证和操作 UTF-8 编码文本的工具。

## 使用

```zc
import "std/utf8.zc"

fn main() {
    let s = "你好, 世界";
    
    // 验证字符串
    if (Utf8::is_valid(s)) {
        println "有效的 UTF-8";
    }
    
    // 统计字符数量（码位）
    let count = Utf8::count_chars(s);
    println "字符数: {count}";
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **is_valid** | `Utf8::is_valid(s: char*) -> bool` | 如果字符串是格式良好的 UTF-8，则返回 true。 |
| **count_chars**| `Utf8::count_chars(s: char*) -> usize`| 返回字符串中的 Unicode 码位数量。 |
| **char_size** | `Utf8::char_size(byte: u8) -> usize` | 根据首字节返回一个字符预期的字节长度。 |
| **to_u32** | `Utf8::to_u32(s: char*) -> u32` | 将下一个 UTF-8 序列转换为 Unicode 码位。 |
