# std/time

`std/time` 模块提供获取当前时间、格式化日期以及测量时间间隔的工具。

## 使用

```zc
import "std/time.zc"

fn main() {
    // 获取 Unix 时间戳 (秒)
    let now = Time::now();
    println "当前时间戳: {now}";
    
    // 测量执行时间
    let start = Time::ticks();
    // 执行操作...
    let end = Time::ticks();
    
    println "耗时: {end - start} 毫秒";
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **now** | `Time::now() -> i64` | 返回自 Unix 纪元以来的当前时间（秒）。 |
| **ticks** | `Time::ticks() -> u64` | 返回高精度的单调时钟值（毫秒）。 |
| **sleep** | `Time::sleep(ms: u64)` | 挂起执行指定的毫秒数 (别名自 `Thread::sleep`)。 |
| **format**| `Time::format(t: i64, fmt: char*) -> String` | 将时间戳转换为格式化的日期字符串。 |

## Time 结构体

对于日历日期，可以使用 `DateTime` 结构体：

| 字段 | 描述 |
| :--- | :--- |
| **year** | 年份 |
| **month** | 月份 (1-12) |
| **day** | 日 (1-31) |
| **hour** | 小时 (0-23) |
