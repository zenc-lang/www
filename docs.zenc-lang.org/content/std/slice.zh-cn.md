# std/slice

`Slice<T>` 提供对数组或内存区域的视图。它包含一个指向数据的指针及其长度。

## 使用

```zc
import "std/slice.zc"

fn main() {
    let arr: int[5] = {10, 20, 30, 40, 50};
    
    // 从数组创建切片
    let s = Slice<int>::from_array(&arr[0], 5);
    
    // 对切片进行切片
    let sub = s.sub(1, 4); // {20, 30, 40}
    
    println "子切片长度: {sub.length()}";
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **from_array** | `Slice::from_array(ptr: T*, len: usize) -> Slice<T>` | 从原始指针创建切片。 |

### 访问与操作

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **at** | `at(self, index: usize) -> T` | 获取指定索引处的元素（带边界检查）。 |
| **sub** | `sub(self, start: usize, end: usize) -> Slice<T>` | 返回原始切片的一个子范围。 |
| **length** | `length(self) -> usize` | 返回切片中元素的数量。 |

### 比较

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **eq** | `eq(self, other: Slice<T>) -> bool` | 如果两个切片包含相同的元素，则返回 true。 |
