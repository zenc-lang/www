# std/regex

`std/regex` 模組提供使用正規表達式（Regular Expressions）進行模式匹配和字符串操作的工具。

## 使用

```zc
import "std/regex.zc"

fn main() {
    let pattern = "[0-9]+";
    let text = "訂單編號: 12345";
    
    // 檢查是否存在匹配
    if (Regex::test(pattern, text)) {
        println "找到數字！";
    }
    
    // 提取匹配內容
    match Regex::find(pattern, text) {
        Some(m) => println "匹配到: {m}",
        None => println "未找到匹配"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **test** | `Regex::test(re: char*, s: char*) -> bool` | 如果字符串與模式匹配，返回 true。 |
| **find** | `Regex::find(re: char*, s: char*) -> Option<String>` | 返回第一個匹配項。 |
| **replace** | `Regex::replace(re: char*, s: char*, replacement: char*) -> String` | 使用替換內容替換所有匹配項。 |
| **split** | `Regex::split(re: char*, s: char*) -> Vec<String>` | 在匹配項位置拆分字符串。 |
