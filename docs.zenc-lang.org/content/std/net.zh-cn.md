# std/net

`std/net` 模块提供用于网络编程的工具，包括 TCP 和 UDP 协议支持。

## 使用

```zc
import "std/net.zc"

fn main() {
    // 创建一个 TCP 客户端并连接到服务器
    match TcpStream::connect("127.0.0.1", 8080) {
        Ok(stream) => {
            stream.write("你好，服务器");
            stream.close();
        },
        Err(e) => println "连接失败: {e}"
    }
}
```

## TCP 协议

### `TcpListener` (服务器)

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **bind** | `TcpListener::bind(port: int) -> Result<TcpListener>` | 在指定端口启动服务器。 |
| **accept** | `accept(self) -> Result<TcpStream>` | 等待并接受新的客户端连接。 |

### `TcpStream` (客户端/连接)

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **connect** | `TcpStream::connect(ip: char*, port: int) -> Result<TcpStream>` | 连接到远程服务器。 |
| **read** | `read(self, buf: u8*, size: usize) -> usize` | 从流中读取数据。 |
| **write** | `write(self, data: char*) -> usize` | 向流中写入字符串。 |
| **close** | `close(self)` | 关闭连接。 |
