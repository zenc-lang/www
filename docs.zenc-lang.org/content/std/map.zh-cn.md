# std/map

`Map<K, V>` 是一种使用哈希表实现的键值对集合。它提供高效的项插入、检索和删除功能。

## 使用

```zc
import "std/map.zc"

fn main() {
    let scores = Map<char*, int>::new();
    
    scores.set("玩家1", 100);
    scores.set("玩家2", 150);
    
    match scores.get("玩家1") {
        Some(s) => println "得分: {s}",
        None => println "未找到"
    }
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Map<K, V>::new() -> Map<K, V>` | 创建一个新的空映射。 |

### 修改

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **set** | `set(self, key: K, value: V)` | 插入或更新键值对。 |
| **remove** | `remove(self, key: K) -> bool` | 如果键存在，则移除该项并返回 true。 |
| **clear** | `clear(self)` | 移除映射中的所有项。 |

### 访问与属性

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **get** | `get(self, key: K) -> Option<V>` | 返回与键关联的值（副本）。 |
| **has** | `has(self, key: K) -> bool` | 如果键存在，则返回 true。 |
| **length** | `length(self) -> usize` | 返回映射中项的数量。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放哈希表使用的内部分配。 |
| **Trait** | `impl Drop for Map` | 自动调用 `free()`。 |
