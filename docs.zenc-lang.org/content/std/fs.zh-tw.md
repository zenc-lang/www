# std/fs

`std/fs` 模組提供與文件系統交互的功能，包括讀取、寫入、刪除文件和操作目錄。

## 使用

```zc
import "std/fs.zc"

fn main() {
    // 讀取整個文件
    match File::read_to_string("hello.txt") {
        Ok(content) => println "文件內容: {content}",
        Err(e) => println "讀取失敗: {e}"
    }

    // 寫入文件
    File::write("output.txt", "你好，Zen-C！");
}
```

## 方法

### 文件操作

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **read_to_string**| `File::read_to_string(path: char*) -> Result<String>` | 讀取文件並將其作為 String 返回。 |
| **write** | `File::write(path: char*, content: char*) -> Result<void>` | 創建或覆蓋文件並寫入數據。 |
| **exists** | `File::exists(path: char*) -> bool` | 如果文件存在則返回 true。 |
| **remove** | `File::remove(path: char*) -> Result<void>` | 刪除指定路徑的文件。 |

### 目錄操作

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **make_dir** | `Dir::make(path: char*) -> Result<void>` | 創建一個新目錄。 |
| **remove_dir** | `Dir::remove(path: char*) -> Result<void>` | 刪除一個目錄（必須為空）。 |
| **read_dir** | `Dir::read(path: char*) -> Result<Vec<String>>` | 返回目錄中所有項目的列表。 |
