# std/simd

`std/simd` 模組提供單指令多數據 (SIMD) 指令的界面，允許在支持的架構（如 x86_64 上的 SSE/AVX）上進行向量化運算。

## 使用

```zc
import "std/simd.zc"

fn main() {
    // 創建一個 4 路 32 位整數向量
    let v1 = f32x4::new(1.0, 2.0, 3.0, 4.0);
    let v2 = f32x4::new(5.0, 6.0, 7.0, 8.0);
    
    // 同時相加所有通道
    let res = v1.add(v2);
}
```

## 類型

| 類型 | 描述 |
| :--- | :--- |
| **f32x4** | 4 個 32 位浮點數的向量。 |
| **i32x4** | 4 個 32 位整數的向量。 |
| **u8x16** | 16 個 8 位無符號整數的向量。 |

## 方法 (適用於大多數向量類型)

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **add** | `add(self, other: T) -> T` | 執行向量加法。 |
| **mul** | `mul(self, other: T) -> T` | 執行向量乘法。 |
| **shuffle**| `shuffle(self, mask: int) -> T` | 重新排列向量中的元素。 |
