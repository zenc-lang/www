# std/signal

`std/signal` 模块提供用于处理异步系统事件（信号）的工具，例如 `SIGINT` (Ctrl+C) 或 `SIGTERM`。

## 使用

```zc
import "std/signal.zc"

fn my_handler(sig: int) {
    println "接收到信号: {sig}";
}

fn main() {
    // 注册自定义处理程序
    Signal::handle(Signal::SIGINT, my_handler);
    
    // 无限循环等待信号
    while (true) {}
}
```

## 常数 (常见信号)

| 常数 | 描述 |
| :--- | :--- |
| **Signal::SIGINT** | 交互式终端中断 (通常为 Ctrl+C)。 |
| **Signal::SIGTERM** | 终止信号（请求程序停止）。 |
| **Signal::SIGILL** | 非法指令。 |
| **Signal::SIGSEGV** | 段错误（非法内存访问）。 |

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **handle** | `Signal::handle(sig: int, handler: fn(int))` | 为特定信号设置回调函数。 |
| **raise** | `Signal::raise(sig: int)` | 向当前进程发送一个信号。 |
| **ignore** | `Signal::ignore(sig: int)` | 设置进程忽略指定的信号。 |
