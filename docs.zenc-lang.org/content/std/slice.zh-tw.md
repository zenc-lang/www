# std/slice

切片（Slice）是數組或連續內存區域的輕量級視圖，由指針和長度組成。

## 使用

```zc
import "std/slice.zc"

fn main() {
    let arr: int[5] = {10, 20, 30, 40, 50};
    
    // 創建一個指向數組中間部分的切片
    let s = Slice::from_array(&arr[1], 3);
    
    println "首個元素: {s.at(0)}"; // 20
    println "長度: {s.length()}";   // 3
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **from_array**| `Slice::from_array(ptr: T*, len: usize) -> Slice<T>` | 從原始指針和長度創建切片。 |
| **at** | `at(self, index: usize) -> T` | 返回指定位置的元素。 |
| **sub_slice** | `sub_slice(self, start: usize, len: usize) -> Slice<T>` | 創建當前切片的子切片。 |
| **length** | `length(self) -> usize` | 返回切片中的元素數量。 |
| **copy_to** | `copy_to(self, dest: Slice<T>)` | 將內容複製到另一個切片。 |
