# Libreria Standard

La Libreria Standard di Zen C fornisce una raccolta completa di moduli e tipi per attività comuni, che vanno dalla manipolazione dei bit a basso livello al networking di alto livello e all'elaborazione JSON.

## Moduli

| Modulo | Descrizione |
| :--- | :--- |
| **[BigFloat](./bigfloat.md)** | Aritmetica in virgola mobile a precisione arbitraria. |
| **[BigInt](./bigint.md)** | Aritmetica intera a precisione arbitraria. |
| **[Bits](./bits.md)** | Operazioni bitwise a basso livello (`rotl`, `rotr`). |
| **[Complex](./complex.md)** | Operazioni con numeri complessi. |
| **[Crypto](./crypto.md)** | Primitive crittografiche (SHA1). |
| **[CUDA](./cuda.md)** | Operazioni GPGPU CUDA e interoperabilità. |
| **[Encoding](./encoding.md)** | Utilità di codifica dei dati (Base64). |
| **[Env](./env.md)** | Accesso alle variabili d'ambiente del processo. |
| **[FS](./fs.md)** | I/O del file system e operazioni sulle directory. |
| **[IO](./io.md)** | Input/Output standard e formattazione. |
| **[Iter](./iter.md)** | Trait e protocolli per iteratori personalizzati. |
| **[JSON](./json.md)** | Parsing e serializzazione JSON in stile DOM. |
| **[Map](./map.md)** | Implementazione di una tabella hash generica. |
| **[Math](./math.md)** | Costanti e funzioni matematiche. |
| **[Mem](./mem.md)** | Gestione della memoria, allocatori e trait. |
| **[Net](./net.md)** | TCP, UDP, HTTP, DNS e parsing URL. |
| **[Option](./option.md)** | Wrapper per valori opzionali (`Some`/`None`). |
| **[Path](./path.md)** | Manipolazione di percorsi file cross-platform. |
| **[Process](./process.md)** | Esecuzione e gestione dei processi. |
| **[Result](./result.md)** | Pattern di gestione degli errori (`Ok`/`Err`). |
| **[Regex](./regex.md)** | Supporto per le espressioni regolari. |
| **[Random](./random.md)** | Generatore di numeri pseudo-casuali. |
| **[SIMD](./simd.md)** | Tipi vettoriali ottimizzati per l'hardware. |
| **[Queue](./queue.md)** | Implementazione di coda FIFO (Ring Buffer). |
| **[Set](./set.md)** | Implementazione di un insieme (hash set) generico. |
| **[Slice](./slice.md)** | Viste di array leggere senza proprietà. |
| **[Sort](./sort.md)** | Motore di ordinamento a zero-overhead. |
| **[Stack](./stack.md)** | Implementazione di pila LIFO. |
| **[String](./string.md)** | Tipo di stringa espandibile, allocata nell'heap. |
| **[Thread](./thread.md)** | Multithreading e sincronizzazione. |
| **[Time](./time.md)** | Utilità di misurazione del tempo e sospensione. |
| **[UTF-8](./utf8.md)** | Utilità di gestione Unicode e UTF-8. |
| **[Vector](./vec.md)** | Tipo di array dinamico espandibile. |
