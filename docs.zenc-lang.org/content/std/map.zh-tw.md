# std/map

`Map<K, V>` 是一個基於哈希表的鍵值對集合，它提供快速的查找、插入和刪除操作。

## 使用

```zc
import "std/map.zc"

fn main() {
    let m = Map<char*, int>::new();
    
    m.insert("apple", 5);
    m.insert("banana", 10);
    
    match m.get("apple") {
        Some(v) => println "蘋果的數量: {v}",
        None => println "未找到蘋果"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Map<K, V>::new() -> Map<K, V>` | 創建一個新的空映射。 |
| **insert** | `insert(self, key: K, value: V)` | 關聯一個鍵和一個值。 |
| **get** | `get(self, key: K) -> Option<V>` | 返回與鍵關聯的值。 |
| **remove** | `remove(self, key: K)` | 移除指定的鍵及其關聯值。 |
| **contains** | `contains(self, key: K) -> bool` | 如果鍵存在，則返回 true。 |
| **length** | `length(self) -> usize` | 返回鍵值對的數量。 |
