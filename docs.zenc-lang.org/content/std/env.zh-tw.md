# std/env

`std/env` 模組提供與程序運行環境（環境變量和命令行參數）交互的工具。

## 使用

```zc
import "std/env.zc"

fn main() {
    // 讀取環境變量
    match Env::get("PATH") {
        Some(val) => println "PATH: {val}",
        None => println "未設置 PATH"
    }
    
    // 獲取命令行參數
    let args = Env::args();
    println "程序名稱: {args.at(0)}";
}
```

## 方法

### 環境變量

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **get** | `Env::get(key: char*) -> Option<String>` | 檢索一個環境變量的值。 |
| **set** | `Env::set(key: char*, value: char*)` | 設置或更新一個環境變量。 |
| **remove** | `Env::remove(key: char*)` | 刪除一個環境變量。 |

### 命令行參數

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **args** | `Env::args() -> Vec<String>` | 返回用於啟動程序的參數列表。 |

### 程序信息

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **current_dir** | `Env::current_dir() -> String` | 返回當前工作目錄。 |
| **set_current_dir**| `Env::set_current_dir(path: char*)` | 更改當前工作目錄。 |
