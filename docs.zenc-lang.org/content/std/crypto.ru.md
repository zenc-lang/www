# std/crypto

Модуль `std/crypto` предоставляет высокоуровневый API для распространенных криптографических алгоритмов, включая хеширование и шифрование.

## Использование

```zc
import "std/crypto/sha256.zc"

fn main() {
    let hash = Sha256::hash("hello world");
    println "SHA-256: {hash}";
}
```

## Подмодули

`std/crypto` организован в виде подмодулей для конкретных алгоритмов:

- **std/crypto/sha256.zc**: Алгоритм хеширования SHA-256.
- **std/crypto/aes.zc**: Шифрование AES (Advanced Encryption Standard).
- **std/crypto/rand.zc**: Криптографически стойкий генератор случайных чисел.

## Типы и интерфейсы

Многие подмодули криптографии следуют общему шаблону для инкрементальной обработки данных:

### Интерфейс хеширования (Пример: Sha256)

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `new() -> Context` | Инициализирует новый контекст хеширования. |
| **update** | `update(self, data: u8*, len: usize)` | Добавляет данные в контекст. |
| **finalize** | `finalize(self, out: u8*)` | Завершает хеширование и записывает результат в выходной буфер. |

## Безопасность

> [!WARNING]
> Всегда используйте рекомендованные размеры ключей и режимы работы (например, AES-GCM) для приложений, чувствительных к безопасности.
