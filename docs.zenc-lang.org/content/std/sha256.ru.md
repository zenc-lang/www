# std/crypto/sha256

Модуль `std/crypto/sha256` предоставляет реализацию алгоритма хеширования SHA-256.

## Использование

```zc
import "std/crypto/sha256.zc"

fn main() {
    let data = "Zen-C";
    let hash = Sha256::hash(data);
    
    println "Хеш SHA-256: {hash}";
}
```

## Методы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **hash** | `Sha256::hash(data: char*) -> String` | Возвращает шестнадцатеричное строковое представление хеша. |
| **bytes** | `Sha256::bytes(data: u8*, n: usize) -> u8[32]`| Возвращает необработанный 32-байтовый хеш. |

## Структура Sha256

Для обработки больших данных или потоков используйте структурный интерфейс:

```zc
let ctx = Sha256::new();
ctx.update("часть 1");
ctx.update("часть 2");
let final_hash = ctx.finalize();
```
