# std/crypto/sha256

Модуль `std/crypto/sha256` реализует алгоритм криптографического хеширования SHA-256, как указано в FIPS 180-4.

## Обзор

- **Хеширование**: Вычисление 256-битных хешей произвольных строковых или байтовых данных.
- **Инкрементальное хеширование**: Обработка данных частями с использованием `update` и `finalize`.
- **Вывод в Hex**: Удобный помощник `hash`, возвращающий шестнадцатеричную строку.

## Использование

```zc
import "std/crypto/sha256.zc"
import "std/io.zc"

fn main() {
    // Простое хеширование
    let hash = Sha256::hash("hello world");
    println "SHA-256: {hash}";
    
    // Инкрементальное хеширование
    let ctx = Sha256::new();
    ctx.update((u8*)"hello", 5);
    ctx.update((u8*)" world", 6);
    
    let digest: u8[32];
    ctx.finalize(&digest[0]);
}
```

## Определение структуры

## Методы

### Методы `Sha256`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Sha256::new() -> Sha256` | Инициализирует новый контекст SHA-256. |
| **update** | `update(self, data: u8*, len: usize)` | Обновляет контекст порцией данных. |
| **finalize** | `finalize(self, hash: u8*)` | Завершает хеширование и записывает результат в предоставленный 32-байтовый буфер. |
| **hash** | `Sha256::hash(data: char*) -> String` | Удобный метод для хеширования строки и возврата ее шестнадцатеричного представления. |
