# std/bigint

`BigInt` 结构体提供任意精度的整数算法。它允许存储和操作远超 `u64` 或 `i64` 限制的数值，根据需要增长内存。

## 使用

```zc
import "std/bigint.zc"

fn main() {
    let a = BigInt::new("123456789012345678901234567890");
    let b = BigInt::new("2");
    
    let c = a.mul(b);
    println "结果: {c}";
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `BigInt::new(s: char*) -> BigInt` | 从字符串表示创建 BigInt。 |
| **from_int** | `BigInt::from_int(v: int) -> BigInt` | 从标准整数创建。 |

### 算术运算

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **add** | `add(self, other: BigInt) -> BigInt` | 返回两个 BigInt 的和。 |
| **sub** | `sub(self, other: BigInt) -> BigInt` | 返回两个 BigInt 的差。 |
| **mul** | `mul(self, other: BigInt) -> BigInt` | 返回两个 BigInt 的积。 |
| **div** | `div(self, other: BigInt) -> BigInt` | 返回两个 BigInt 的商。 |
| **mod** | `mod(self, other: BigInt) -> BigInt` | 返回取模结果。 |

### 转换

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **to_string** | `to_string(self) -> String` | 返回 BigInt 的字符串表示。 |
| **to_int** | `to_int(self) -> int` | 将 BigInt 转换为标准整数（可能会溢出）。 |
