# std/simd

Zen-C предоставляет нативные векторные типы SIMD (Single Instruction, Multiple Data - одна инструкция, много данных), которые компилируются непосредственно в оптимизированные аппаратные векторные инструкции (SSE, AVX, NEON и т. д.), поддерживаемые целевым бекендом.

## Обзор

- **Нативная производительность**: Использует векторные расширения LLVM/GCC для максимальной эффективности.
- **Неявная переносимость**: Типы вроде `f32x4` сопоставляются с лучшим доступным 128-битным оборудованием, специфичным для архитектуры.
- **Поэлементная арифметика**: Стандартные операторы (`+`, `-`, `*`, `/`) применяются ко всем каналам (lanes) одновременно.
- **Вещание (Broadcasting)**: Инициализация одним значением распространяет его на все каналы.

## Использование

```zc
import "std/simd.zc"

fn main() {
    // Инициализация (явное указание каналов)
    let a = f32x4 { 1.0, 2.0, 3.0, 4.0 };
    
    // Вещание (одно значение для всех каналов)
    let b = f32x4 { v: 2.0 };
    
    // Поэлементное сложение
    let c = a + b;   // Результат: { 3.0, 4.0, 5.0, 6.0 }
    
    // Доступ к каналу
    let first = c[0];
}
```

## Векторные типы

Стандартная библиотека определяет несколько 128- и 256-битных векторных типов. Вы также можете определить пользовательские типы, используя атрибут `@vector(N)`.

### 128-битные векторы (SSE / NEON)

| Тип | Тип элемента | Каналы | Размер в байтах |
| :--- | :--- | :--- | :--- |
| `f32x4` | `f32` | 4 | 16 |
| `f64x2` | `f64` | 2 | 16 |
| `i32x4` | `i32` | 4 | 16 |
| `u32x4` | `u32` | 4 | 16 |
| `i64x2` | `i64` | 2 | 16 |
| `u64x2` | `u64` | 2 | 16 |
| `i16x8` | `i16` | 8 | 16 |
| `u16x8` | `u16` | 8 | 16 |
| `i8x16` | `i8` | 16 | 16 |
| `u8x16` | `u8` | 16 | 16 |

### 256-битные векторы (AVX / AVX2)

| Тип | Тип элемента | Каналы | Размер в байтах |
| :--- | :--- | :--- | :--- |
| `f32x8` | `f32` | 8 | 32 |
| `f64x4` | `f64` | 4 | 32 |
| `i32x8` | `i32` | 8 | 32 |
| `u32x8` | `u32` | 8 | 32 |
| `i64x4` | `i64` | 4 | 32 |
| `u64x4` | `u64` | 4 | 32 |
| `i16x16` | `i16` | 16 | 32 |
| `u16x16` | `u16` | 16 | 32 |
| `i8x32` | `i8` | 32 | 32 |
| `u8x32" | `u8" | 32 | 32 |

## Операции

| Категория | Оператор | Описание |
| :--- | :--- | :--- |
| **Арифметика** | `+`, `-`, `*`, `/` | Стандартное поэлементное сложение, вычитание, умножение и деление. |
| **Побитовые** | `&`, `|`, `^`, `~` | Побитовые И, ИЛИ, Исключающее ИЛИ и НЕ для всех каналов. |
| **Индексация** | `[i]` | Доступ или изменение отдельных каналов по индексу. |
| **Сравнение** | `==`, `!=`, `<`, `>` | Возвращает вектор логической маски (результаты зависят от бекенда). |
