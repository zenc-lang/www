# std/simd

Модуль `std/simd` предоставляет переносимые (portable) SIMD-типы и операции (Single Instruction, Multiple Data) для параллелизма на уровне данных.

## Обзор

- **Переносимые векторы**: Векторные типы, отображаемые на инструкции SSE, AVX или NEON в зависимости от аппаратного обеспечения.
- **Высокая производительность**: Выполнение одной операции над несколькими элементами данных одновременно.
- **Автоматическое выравнивание**: Гарантирует выравнивание памяти, необходимое для эффективных SIMD-операций.

## Использование

```zc
import "std/simd.zc"

fn main() {
    let a = f32x4::new(1.0, 2.0, 3.0, 4.0);
    let b = f32x4::new(5.0, 6.0, 7.0, 8.0);
    
    // Все 4 сложения выполняются одной инструкцией процессора
    let c = a.add(b);
}
```

## Векторные типы

### Векторы с плавающей запятой
- `f32x4`: 4 числа float (32 бит), всего 128 бит.
- `f32x8`: 8 чисел float (32 бит), всего 256 бит.
- `f64x2`: 2 числа double (64 бит), всего 128 бит.
- `f64x4`: 4 числа double (64 бит), всего 256 бит.

### Целочисленные векторы
- `i32x4`, `u32x4`, `i32x8`, `u32x8`.
- `i64x2`, `u64x2`, `i64x4`, `u64x4`.

## Методы (общие для всех типов)

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Type::new(...args) -> Vector` | Создает вектор из отдельных значений. |
| **splat**| `Type::splat(val) -> Vector` | Создает вектор, заполненный значением `val`. |
| **load** | `Type::load(ptr: T*) -> Vector` | Загружает вектор из памяти (указатель должен быть выровнен). |

### Арифметика

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **add** | `add(self, other: Vector) -> Vector` | Поэлементное сложение. |
| **sub** | `sub(self, other: Vector) -> Vector` | Поэлементное вычитание. |
| **mul** | `mul(self, other: Vector) -> Vector` | Поэлементное умножение. |
| **div** | `div(self, other: Vector) -> Vector` | Поэлементное деление. |

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **sum** | `sum(self) -> T` | Возвращает сумму всех элементов вектора. |
