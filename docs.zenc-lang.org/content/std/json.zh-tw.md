# std/json

`std/json` 模組提供用於解析和生成 JSON（JavaScript Object Notation）數據的工具。

## 使用

```zc
import "std/json.zc"

fn main() {
    let raw = "{\"name\": \"Zen-C\", \"version\": 1}";
    
    // 解析 JSON 字符串
    match Json::parse(raw) {
        Ok(obj) => {
            println "名稱: {obj.get(\"name\").as_string()}";
            obj.free();
        },
        Err(e) => println "解析失敗: {e}"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **parse** | `Json::parse(s: char*) -> Result<JsonValue>` | 將 JSON 字符串解析為數值結構。 |
| **stringify** | `Json::stringify(val: JsonValue) -> String` | 將 JSON 數值轉換為字符串。 |

## JsonValue 類型

| 類型 | 描述 |
| :--- | :--- |
| **Object** | 鍵值對的映射。 |
| **Array** | 數值的有序列表。 |
| **String** | 文本。 |
| **Number** | 雙精度浮點數。 |
| **Bool** | true 或 false。 |
| **Null** | 代表無值。 |
