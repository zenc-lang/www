# std/sys/user

Модуль `std/sys/user` предоставляет доступ к идентификационной информации пользователя и группы, являясь оберткой для POSIX `unistd.h`.

## Обзор

- **Идентификация пользователя**: Получение реальных и эффективных ID пользователя (UID) и ID группы (GID).
- **Контекст процесса**: Полезно для проверки привилегий и управления разрешениями в системных утилитах.

## Использование

```zc
import "std/sys/user.zc"
import "std/io.zc"

fn main() {
    println "Текущий UID: {User::get_uid()}";
    println "Текущий GID: {User::get_gid()}";
    
    if (User::get_euid() == 0) {
        println "Запущено с привилегиями root.";
    }
}
```

struct User {}

## Методы

### Методы `User`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **get_uid** | `User::get_uid() -> u32` | Возвращает реальный ID пользователя текущего процесса. |
| **get_gid** | `User::get_gid() -> u32` | Возвращает реальный ID группы текущего процесса. |
| **get_euid** | `User::get_euid() -> u32` | Возвращает эффективный ID пользователя. |
| **get_egid** | `User::get_egid() -> u32` | Возвращает эффективный ID группы. |
