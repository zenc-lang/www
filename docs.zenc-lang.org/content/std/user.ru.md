# std/sys/user

Модуль `std/sys/user` предоставляет функции для получения информации о текущем пользователе системы и группах, являясь оберткой над POSIX `pwd.h` и `grp.h`.

## Использование

```zc
import "std/sys/user.zc"
import "std/io.zc"

fn main() {
    // Получение информации о текущем пользователе
    match User::current() {
        Some(u) => {
            println "Пользователь: {u.name}";
            println "UID: {u.uid}";
            println "Домашняя директория: {u.home}";
        },
        None => println "Не удалось получить информацию о пользователе"
    }
}
```

## Определение структуры

### `User`
```zc
struct User {
    name: String;
    uid: u32;
    gid: u32;
    home: String;
    shell: String;
}
```

## Методы

### Методы `User`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **current** | `User::current() -> Option<User>` | Возвращает информацию о текущем залогиненном пользователе. |
| **by_name** | `User::by_name(name: char*) -> Option<User>` | Ищет пользователя по имени. |
| **by_uid** | `User::by_uid(uid: u32) -> Option<User>` | Ищет пользователя по его числовому ID (UID). |
