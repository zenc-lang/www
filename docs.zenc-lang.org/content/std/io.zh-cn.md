# std/io

`std/io` 模块提供基本的输入/输出功能，包括打印到控制台。

## 使用

```zc
import "std/io.zc"

fn main() {
    // 带有换行的打印
    println "你好，世界";
    
    // 格式化打印 (插值)
    let name = "Zen";
    println "你好，{name}";
    
    // 不带换行的打印
    print "处理中...";
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **print** | `print(msg: char*)` | 将字符串打印到标准输出 (`stdout`)。 |
| **println** | `println(msg: char*)` | 打印并随后输出一个换行符。 |
| **scanf** | `scanf(fmt: char*, ...)` | 从标准输入读取格式化输入（直接包装 C `scanf`）。 |

## 格式化输出

Zen-C 的 `println` 支持通过内嵌大括号进行字符串插值：
- 使用 `{var}` 自动格式化大多数内置类型。
- 底层实现利用 `std/string` 进行动态构建。
