# std/stack

`Stack<T>` 提供后进先出 (LIFO) 的数据存储。

## 使用

```zc
import "std/stack.zc"

fn main() {
    let s = Stack<int>::new();
    
    s.push(10);
    s.push(20);
    
    match s.pop() {
        Some(val) => println "弹出: {val}", // 20
        None => println "栈为空"
    }
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Stack<T>::new() -> Stack<T>` | 创建一个空栈。 |

### 修改

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **push** | `push(self, val: T)` | 将一个元素推入栈顶。 |
| **pop** | `pop(self) -> Option<T>` | 移除并返回栈顶元素。 |
| **clear** | `clear(self)` | 移除所有元素。 |

### 访问与属性

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **peek** | `peek(self) -> Option<T>` | 查看栈顶元素而不移除它。 |
| **length** | `length(self) -> usize` | 返回栈中的元素数量。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放栈使用的内存。 |
| **Trait** | `impl Drop for Stack` | 自动调用 `free()`。 |
