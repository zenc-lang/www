# std/signal

`std/signal` 模組提供處理 POSIX 信號（如 `SIGINT`, `SIGTERM`）工具，允許程序優雅地響應外部事件或系統中斷。

## 使用

```zc
import "std/signal.zc"

fn handle_sigint(sig: int) {
    println "收到 SIGINT，正在退出...";
    Process::exit(0);
}

fn main() {
    // 註冊信號處理函數
    Signal::handle(Signal::INT, handle_sigint);
    
    // 無限循環等待信號
    while (true) {}
}
```

## 常量 (信號)

| 名稱 | 描述 |
| :--- | :--- |
| **INT** | 中斷信號（通常是 Ctrl+C）。 |
| **TERM** | 終止信號。 |
| **KILL** | 強制殺死進程。 |
| **SEGV** | 分段錯誤（非法內存訪問）。 |

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **handle** | `Signal::handle(sig: int, handler: fn(int))` | 為指定信號設置處置函數。 |
| **raise** | `Signal::raise(sig: int)` | 向當前進程發送一個信號。 |
