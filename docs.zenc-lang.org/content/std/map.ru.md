# std/map

`Map<V>` - это реализация универсальной хеш-таблицы, сопоставляющей строковые ключи со значениями типа `V`.

## Использование

```zc
import "std/map.zc"

fn main() {
    let m = Map<int>::new();
    
    m.put("one", 1);
    m.put("two", 2);
    
    if (m.contains("one")) {
        let val = m.get("one");
        println "{val.unwrap()}";
    }
    
    m.remove("two");
} // m освобождается здесь автоматически
```

## Определение структуры

```zc
struct Map<V> {
    keys: char**;
    vals: V*;
    // ... внутренние поля
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Map<V>::new() -> Map<V>` | Создает новую пустую карту. |

### Итерация

Вы можете итерировать по парам ключ-значение карты, используя цикл `for`.

```zc
let m = Map<int>::new();
m.put("a", 1);

for entry in m {
    println "Ключ: {entry.key}, Знач: {entry.val}";
}
```

Итератор выдает структуру `MapEntry<V>`:
```zc
struct MapEntry<V> {
    key: char*;
    val: V;
}
```

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **put** | `put(self, key: char*, val: V)` | Вставляет или обновляет пару ключ-значение. |
| **remove** | `remove(self, key: char*)` | Удаляет ключ и его значение из карты. |

### Доступ и запроры

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **get** | `get(self, key: char*) -> Option<V>` | Извлекает значение, связанное с ключом. |
| **contains** | `contains(self, key: char*) -> bool` | Возвращает true, если ключ существует. |
| **length** | `length(self) -> usize` | Возвращает количество элементов в карте. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает true, если карта пуста. |
| **capacity** | `capacity(self) -> usize` | Возвращает текущую емкость карты. |

### Вспомогательные методы итерации

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_slot_occupied** | `is_slot_occupied(self, idx: usize) -> bool` | Проверяет, занят ли необработанный индекс слота. |
| **key_at** | `key_at(self, idx: usize) -> char*` | Получает ключ по необработанному индексу слота. |
| **val_at** | `val_at(self, idx: usize) -> V` | Получает значение по необработанному индексу слота. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Освобождает внутреннее хранилище карты. **Примечание**: Это не освобождает значения, если они являются указателями или объектами. |
| **Trait** | `impl Drop for Map` | Автоматически вызывает `free()`, когда объект выходит за пределы области видимости. |
