# std/map

`Map<K, V>` — это обобщенная (generic) хеш-таблица для хранения пар ключ-значение. Она использует открытую адресацию с линейным пробированием.

## Использование

```zc
import "std/map.zc"

fn main() {
    let m = Map<char*, int>::new();
    
    m.insert("яблоки", 5);
    m.insert("бананы", 10);
    
    match m.get("яблоки") {
        Some(val) => println "Количество: {val}",
        None => println "Ключ не найден"
    }
    
    m.remove("бананы");
} // m освобождается автоматически здесь
```

## Детали реализации

- **Линейное пробирование**: Эффективное использование кэша процессора.
- **Автоматическое расширение**: Таблица пересобирается (rehash), когда коэффициент заполнения (load factor) превышает 0.7.
- **Трейт Drop**: Реализует автоматическое управление памятью для самой таблицы.

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Map<K, V>::new() -> Map<K, V>` | Создает пустую хеш-таблицу. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **insert** | `insert(self, key: K, val: V)` | Добавляет или обновляет пару ключ-значение. |
| **remove** | `remove(self, key: K) -> bool` | Удаляет ключ. Возвращает true, если ключ был найден. |
| **clear** | `clear(self)` | Удаляет все элементы, не освобождая выделенную память. |

### Доступ и проверка

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **get** | `get(self, key: K) -> Option<V>` | Возвращает значение по ключу. |
| **get_ptr** | `get_ptr(self, key: K) -> V*` | Возвращает указатель на значение (для прямой модификации). |
| **contains** | `contains(self, key: K) -> bool` | Проверяет наличие ключа. |
| **length** | `length(self) -> usize` | Возвращает количество элементов. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает внутренние буферы. |
| **Trait** | `impl Drop for Map` | Автоматически очищает ресурсы при выходе из области видимости. |
