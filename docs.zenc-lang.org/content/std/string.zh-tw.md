# std/string

`String` 類型是一個動態可變的大小的 UTF-8 字元序列。它是處理文本數據的主要方式。

## 使用

```zc
import "std/string.zc"

fn main() {
    // 創建一個新字符串
    let s = String::from("你好");
    
    // 拼接字符串
    s.push_str(", Zen-C!");
    
    println "{s}"; // "你好, Zen-C!"
    
    // 獲取長度
    let len = s.length();
}
```

## 方法

### 創建與修改

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **from** | `String::from(s: char*) -> String` | 從 C 字符串創建一個 String。 |
| **push_str** | `push_str(self, other: char*)` | 在末尾追加字符串。 |
| **clear** | `clear(self)` | 將字符串清空。 |

### 查詢與搜索

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **length** | `length(self) -> usize` | 返回字節長度。 |
| **is_empty** | `is_empty(self) -> bool` | 如果長度為 0，則返回 true。 |
| **contains** | `contains(self, sub: char*) -> bool` | 如果包含子字符串，則返回 true。 |
| **starts_with**| `starts_with(self, pre: char*) -> bool` | 檢查是否以指定前綴開始。 |
