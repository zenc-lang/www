# std/crypto/sha256

O módulo `std/crypto/sha256` implementa o algoritmo de hash criptográfico SHA-256 conforme especificado em FIPS 180-4.

## Visão Geral

- **Hashing**: Calcula hashes de 256 bits de dados arbitrários em string ou bytes.
- **Hashing Incremental**: Processa dados em pedaços (chunks) utilizando `update` e `finalize`.
- **Saída Hexadecimal**: Auxiliar `hash` conveniente que retorna uma string hexadecimal.

## Uso

```zc
import "std/crypto/sha256.zc"
import "std/io.zc"

fn main() {
    // Hashing simples
    let hash = Sha256::hash("olá mundo");
    println "SHA-256: {hash}";
    
    // Hashing incremental
    let ctx = Sha256::new();
    ctx.update((u8*)"olá", 3);
    ctx.update((u8*)" mundo", 6);
    
    let digest: u8[32];
    ctx.finalize(&digest[0]);
}
```

## Definição da Estrutura

struct Sha256 {

## Métodos

### Métodos de `Sha256`

| Método | Assinatura | Descrição |
| :--- | :--- | :--- |
| **new** | `Sha256::new() -> Sha256` | Inicializa um novo contexto SHA-256. |
| **update** | `update(self, data: u8*, len: usize)` | Atualiza o contexto com um pedaço de dados. |
| **finalize** | `finalize(self, hash: u8*)` | Finaliza o hash e escreve o resultado no buffer de 32 bytes fornecido. |
| **hash** | `Sha256::hash(data: char*) -> String` | Método de conveniência para calcular o hash de uma string e retornar a sua representação hexadecimal. |
