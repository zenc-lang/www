# std/complex

`Complex` 结构体提供复数运算，支持实部和虚部。

## 使用

```zc
import "std/complex.zc"

fn main() {
    let a = Complex::new(1.0, 2.0); // 1 + 2i
    let b = Complex::new(3.0, 4.0); // 3 + 4i
    
    let c = a.add(b);
    println "结果: {c.re} + {c.im}i";
}
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Complex::new(re: double, im: double) -> Complex` | 创建一个新的复数。 |

### 算术运算

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **add** | `add(self, other: Complex) -> Complex` | 复数加法。 |
| **sub** | `sub(self, other: Complex) -> Complex` | 复数减法。 |
| **mul** | `mul(self, other: Complex) -> Complex` | 复数乘法。 |
| **div** | `div(self, other: Complex) -> Complex` | 复数除法。 |

### 复数特定操作

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **abs** | `abs(self) -> double` | 返回复数的模（绝对值）。 |
| **conj** | `conj(self) -> Complex` | 返回共轭复数。 |
| **arg** | `arg(self) -> double` | 返回复数的辐角。 |
