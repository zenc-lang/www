# std/result

`Result<T, E>` 類型用於錯誤處理。它代表一個成功的值 `Ok(T)` 或一個錯誤 `Err(E)`。

## 使用

```zc
import "std/core.zc"

fn divide(a: int, b: int) -> Result<int, char*> {
    if (b == 0) return Err("除數不能為零");
    return Ok(a / b);
}

fn main() {
    match divide(10, 2) {
        Ok(val) => println "結果: {val}",
        Err(e) => println "錯誤: {e}"
    }
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **is_ok** | `is_ok(self) -> bool` | 如果結果是成功，則返回 true。 |
| **is_err** | `is_err(self) -> bool` | 如果結果是錯誤，則返回 true。 |
| **unwrap** | `unwrap(self) -> T` | 返回內部的值，如果是 Err 則崩潰。 |
| **unwrap_or** | `unwrap_or(self, default: T) -> T`| 返回值或指定的默認值。 |
