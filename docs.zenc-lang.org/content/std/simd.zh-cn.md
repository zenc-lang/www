# std/simd

Zen-C 提供原生 SIMD（单指令多数据）向量类型，这些类型直接编译为目标后端支持的硬件优化向量指令（如 SSE、AVX、NEON 等）。

## 概览

- **原生性能**：利用 LLVM/GCC 向量扩展实现最高效率。
- **隐式可移植性**：`f32x4` 等类型会自动映射到相应架构上可用的最佳 128 位硬件指令。
- **逐元素算术运算**：标准运算符（`+`、`-`、`*`、`/`）会同时应用于所有通道（lane）。
- **广播（Broadcasting）**：使用单个值进行初始化会将其广播到所有通道。

## 使用方法

```zc
import "std/simd.zc"

fn main() {
    // 初始化（显式指定通道值）
    let a = f32x4 { 1.0, 2.0, 3.0, 4.0 };
    
    // 广播（将单个值应用到所有通道）
    let b = f32x4 { v: 2.0 };
    
    // 逐元素加法
    let c = a + b;   // 结果: { 3.0, 4.0, 5.0, 6.0 }
    
    // 通道访问
    let first = c[0];
}
```

## 向量类型

标准库定义了多种 128 位和 256 位向量类型。您还可以使用 `@vector(N)` 属性定义自定义向量类型。

### 128 位向量 (SSE / NEON)

| 类型 | 元素类型 | 通道数 | 字节大小 |
| :--- | :--- | :--- | :--- |
| `f32x4` | `f32` | 4 | 16 |
| `f64x2` | `f64` | 2 | 16 |
| `i32x4` | `i32` | 4 | 16 |
| `u32x4` | `u32` | 4 | 16 |
| `i64x2` | `i64` | 2 | 16 |
| `u64x2` | `u64` | 2 | 16 |
| `i16x8` | `i16` | 8 | 16 |
| `u16x8` | `u16` | 8 | 16 |
| `i8x16` | `i8` | 16 | 16 |
| `u8x16` | `u8` | 16 | 16 |

### 256 位向量 (AVX / AVX2)

| 类型 | 元素类型 | 通道数 | 字节大小 |
| :--- | :--- | :--- | :--- |
| `f32x8` | `f32` | 8 | 32 |
| `f64x4` | `f64` | 4 | 32 |
| `i32x8` | `i32` | 8 | 32 |
| `u32x8` | `u32` | 8 | 32 |
| `i64x4` | `i64` | 4 | 32 |
| `u64x4` | `u64` | 4 | 32 |
| `i16x16` | `i16` | 16 | 32 |
| `u16x16` | `u16` | 16 | 32 |
| `i8x32` | `i8` | 32 | 32 |
| `u8x32" | `u8` | 32 | 32 |

## 操作

| 类别 | 运算符 | 说明 |
| :--- | :--- | :--- |
| **算术运算** | `+`, `-`, `*`, `/` | 标准的逐元素加、减、乘、除。 |
| **按位运算** | `&`, `|", `^", `~` | 跨所有通道的按位与、或、异或和取反。 |
| **索引** | `[i]` | 通过索引访问或修改单个通道。 |
| **比较** | `==`, `!=`, `<`, `>` | 返回布尔掩码向量（结果因后端而异）。 |
