# std/simd

`std/simd` 模块提供单指令多数据 (SIMD) 矢量操作，用于高性能计算。

## 使用

```zc
import "std/simd.zc"

fn main() {
    // 定义 4 个 float 的矢量
    let a: f32x4 = {1.0, 2.0, 3.0, 4.0};
    let b: f32x4 = {5.0, 6.0, 7.0, 8.0};
    
    // 一次性执行所有 4 个加法
    let res = Simd::add(a, b);
}
```

## 矢量类型

Zen-C 支持多种依赖于架构的矢量类型：
- **f32x4**: 4 个 32 位浮点数 (128 位)。
- **i32x4**: 4 个 32 位整数 (128 位)。
- **f64x2**: 2 个 64 位双精度浮点数 (128 位)。

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **add** | `Simd::add(a, b) -> Vector` | 矢量加法。 |
| **sub** | `Simd::sub(a, b) -> Vector` | 矢量减法。 |
| **mul** | `Simd::mul(a, b) -> Vector` | 矢量乘法。 |
| **div** | `Simd::div(a, b) -> Vector` | 矢量除法。 |
| **load** | `Simd::load(ptr: float*) -> f32x4` | 从内存加载数据到矢量。 |
| **store** | `Simd::store(ptr: float*, v: f32x4)` | 将矢量数据写回内存。 |
