# std/result

`Result<T, E>` 枚举表示一个操作的成功 (`Ok`) 或失败 (`Err`)。它是 Zen-C 中推荐的错误处理机制。

## 使用

```zc
import "std/result.zc"

fn perform_action() -> Result<int, char*> {
    if (false) {
        return Err("出错了");
    }
    return Ok(42);
}

fn main() {
    match perform_action() {
        Ok(val) => println "成功: {val}",
        Err(e) => println "错误: {e}"
    }
}
```

## 方法

### 状态检查

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **is_ok** | `is_ok(self) -> bool` | 如果结果为 `Ok`，则返回 true。 |
| **is_err** | `is_err(self) -> bool` | 如果结果为 `Err`，则返回 true。 |

### 值获取

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **unwrap** | `unwrap(self) -> T` | 返回 OK 的值。如果为 `Err`，则触发运行地错误。 |
| **unwrap_err**| `unwrap_err(self) -> E` | 返回错误值。如果结果为 `Ok`，则触发崩溃。 |
| **unwrap_or** | `unwrap_or(self, default: T) -> T` | 返回 OK 的值，或者提供的默认值。 |

### 辅助库

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **expect** | `expect(self, msg: char*) -> T` | 类似于 `unwrap`，但在失败时提供自定义消息。 |
