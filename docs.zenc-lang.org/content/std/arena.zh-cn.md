# std/arena

`Arena` 分配器是一种高效的内存管理机制，它预先分配一大块内存，并允许从中通过简单的递增指针进行快速分配。当 Arena 被销毁时，所有分配的内存会一次性释放。

## 概览

- **快速分配**: 分配只需移动一个指针。
- **批量释放**: 销毁 Arena 会立即释放所有相关内存。
- **无碎片**: 内存按顺序分配。
- **生命周期**: 非常适合具有明确开始和结束的处理任务（例如解析或编译器传递）。

## 使用

```zc
import "std/arena.zc"

fn main() {
    // 创建一个初始大小为 1KB 的 Arena
    let a = Arena::new(1024);
    
    // 从 Arena 中分配内存
    let ptr1 = a.alloc(128);
    let ptr2 = a.alloc(256);
    
    // 无需手动释放 ptr1 或 ptr2
} // a 在此处销毁时，ptr1 和 ptr2 会被一并释放
```

## 方法

### 初始化

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **new** | `Arena::new(initial_size: usize) -> Arena` | 创建一个新的 Arena。 |

### 分配

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **alloc** | `alloc(self, size: usize) -> void*` | 从 Arena 分配空间。如果空间不足，Arena 将自动调整大小或链接新块。 |
| **reset** | `reset(self)` | 重置分配指针。之前分配的所有内存仍然有效直到 `free()`。 |

### 辅助库

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **total_allocated**| `total_allocated(self) -> usize` | 返回当前从此 Arena 分配的总字节数。 |

## 内存管理

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **free** | `free(self)` | 释放所有由 Arena 控制的内存块。 |
| **Trait** | `impl Drop for Arena` | 离开作用域时自动调用 `free()`。 |
