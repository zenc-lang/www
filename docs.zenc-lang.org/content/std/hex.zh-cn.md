# std/hex

`std/hex` 模块提供用于在二进制数据与其十六进制字符串表示之间进行编码和解码的函数。

## 使用

```zc
import "std/hex.zc"

fn main() {
    let data: u8[] = { 0xDE, 0xAD, 0xBE, 0xEF };
    
    // 编码为十六进制字符串
    let encoded = Hex::encode(data, 4);
    println "十六进制: {encoded}"; // "deadbeef"
    
    // 解码回二进制
    let decoded = Hex::decode("48656c6c6f");
    println "解密: {decoded}"; // "Hello"
}
```

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **encode** | `Hex::encode(data: u8*, len: usize) -> String` | 将原始字节数组转换为小写的十六进制字符串。 |
| **decode** | `Hex::decode(hex_str: char*) -> String` | 将十六进制字符串转换回其原始字节（以 String 形式返回）。 |
