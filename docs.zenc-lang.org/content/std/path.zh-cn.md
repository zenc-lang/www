# std/path

`std/path` 模块提供用于跨平台处理文件系统路径的工具。

## 使用

```zc
import "std/path.zc"

fn main() {
    let p = "/usr/bin/zenc";
    
    println "文件名: {Path::basename(p)}"; // "zenc"
    println "目录: {Path::dirname(p)}"; // "/usr/bin"
    
    let joined = Path::join("home", "user", "docs");
    println "拼接路径: {joined}"; // "home/user/docs"
}
```

## 方法

### 路径分解

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **basename** | `Path::basename(p: char*) -> String` | 返回路径的最后一部分。 |
| **dirname** | `Path::dirname(p: char*) -> String` | 返回父目录路径。 |
| **extension**| `Path::extension(p: char*) -> String` | 返回文件的扩展名（例如 ".zc"）。 |

### 路径合成

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **join** | `Path::join(a: char*, b: char*) -> String` | 将两个路径片段合并，并根据操作系统自动添加分隔符 (`/` 或 `\`)。 |
| **absolute** | `Path::absolute(p: char*) -> String` | 将相对路径转换为绝对路径。 |

### 属性检查

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **is_absolute**| `Path::is_absolute(p: char*) -> bool` | 如果路径是绝对的，则返回 true。 |

## 常数

| 常数 | 描述 |
| :--- | :--- |
| **Path::SEP** | 当前平台的路径分隔符（`char`）。 |
