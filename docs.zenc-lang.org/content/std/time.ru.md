# std/time

Модуль `std/time` предоставляет функции для работы с системным временем, измерения длительности и форматирования дат.

## Использование

```zc
import "std/time.zc"

fn main() {
    // Получение текущего Unix-времени (в секундах)
    let now = Time::now();
    println "Текущее время: {now}";

    // Измерение времени выполнения
    let start = Time::ticks_ms();
    // ... выполнение кода ...
    let end = Time::ticks_ms();
    println "Заняло: {end - start} мс";
}
```

## Методы

### Системное время

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **now** | `Time::now() -> i64` | Возвращает текущее Unix-время (секунды с 1 января 1970 года). |
| **ticks_ms**| `Time::ticks_ms() -> u64` | Возвращает монотонно возрастающее количество миллисекунд (полезно для профилирования). |
| **sleep** | `Time::sleep(ms: int)` | Приостанавливает выполнение текущего потока на `ms` миллисекунд. |

### Форматирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **format** | `Time::format(t: i64, fmt: char*) -> String` | Форматирует время согласно строке формата (использует `strftime`). |
