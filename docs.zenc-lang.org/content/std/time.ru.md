# std/time

Модуль `std/time` предоставляет утилиты для высокоточного измерения времени и приостановки потоков.

## Обзор

- **Миллисекундная точность**: `Time::now()` возвращает текущее системное время в миллисекундах.
- **Тип Duration**: Структура `Duration` позволяет интуитивно вычислять временные интервалы.
- **Простое засыпание**: Простые в использовании функции для приостановки выполнения.
- **Легковесность**: Минимальные накладные расходы, оболочка над стандартными системными функциями времени.

## Использование

```zc
import "std/time.zc"

fn main() {
    let start = Time::now();
    
    // Сон на 1.5 секунды
    Time::sleep(Duration::from_ms(1500));
    
    let end = Time::now();
    println "Прошло: {end - start} мс";
}
```

## Определения структур

### `Duration`
Представляет интервал времени.
```zc
struct Duration {
    millis: U64;
}
```

### `Time`
Статическая вспомогательная структура для операций с системным временем.
```zc
struct Time {}
```

## Методы

### Методы `Duration`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **from_ms** | `Duration::from_ms(ms: U64) -> Duration` | Создает `Duration` из количества миллисекунд. |
| **from_secs** | `Duration::from_secs(s: U64) -> Duration" | Создает `Duration` из количества секунд. |

### Статические методы `Time`

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **now** | `Time::now() -> U64` | Возвращает текущее системное время в миллисекундах с начала эпохи. |
| **sleep** | `Time::sleep(d: Duration)` | Приостанавливает текущий поток на указанную длительность `Duration`. |
| **sleep_ms** | `Time::sleep_ms(ms: U64)` | Приостанавливает текущий поток на указанное количество миллисекунд. |
