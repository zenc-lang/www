# std/time

`std/time` 模組提供用於測量時間、獲取當前系統時間以及處理時間延遲的工具。

## 使用

```zc
import "std/time.zc"

fn main() {
    // 獲取當前 Unix 時間戳
    let now = Time::now();
    println "當前時間: {now}";
    
    // 使線程休眠 1 秒
    Time::sleep(1000);
}
```

## 方法

| 方法 | 簽名 | 描述 |
| :--- | :--- | :--- |
| **now** | `Time::now() -> u64` | 返回自 Unix 紀元以來的秒數。 |
| **ms** | `Time::ms() -> u64` | 返回自 Unix 紀元以來的毫秒數。 |
| **sleep** | `Time::sleep(ms: u64)` | 暫停當前線程執行指定的毫秒數。 |
| **format** | `Time::format(t: u64, fmt: char*) -> String` | 根據格式字符串格式化時間戳。 |
