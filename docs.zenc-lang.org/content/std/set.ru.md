# std/set

`Set<T>` — это обобщенная реализация множества (хеш-набора) для хранения уникальных значений типа `T`. Использует хеш-таблицу с открытой адресацией и линейным пробированием.

## Обзор

- **Обобщенность**: Хранит любой тип `T`.
- **Уникальность**: Автоматически обрабатывает дубликаты; добавление существующего элемента возвращает `false`.
- **Скорость**: Средняя сложность O(1) для добавления, удаления и поиска.
- **RAII**: Реализует трейт `Drop` для автоматического управления памятью.

## Использование

```zc
import "std/set.zc"

fn main() {
    let s = Set<int>::new();
    
    s.add(10);
    s.add(20);
    s.add(10); // Дубликат, вернет false
    
    if (s.contains(10)) {
        println "Множество содержит 10";
    }
    
    s.remove(20);
    println "Размер: {s.length()}";
} // s освобождается автоматически здесь
```

## Определение структуры

```zc
struct Set<T> {
    data: T*;
    len: usize;
    cap: usize;
    // ... внутренние поля
}
```

## Методы

### Создание

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Set<T>::new() -> Set<T>` | Создает новое пустое множество. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **add** | `add(self, val: T) -> bool` | Добавляет значение. Возвращает `true`, если добавлено, `false`, если уже есть. |
| **remove** | `remove(self, val: T) -> bool` | Удаляет значение. Возвращает `true`, если оно было найдено. |
| **clear** | `clear(self)` | Удаляет все элементы, не освобождая выделенный буфер. |

### Доступ и проверка

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **contains** | `contains(self, val: T) -> bool` | Проверяет наличие значения в множестве. |
| **length** | `length(self) -> usize` | Возвращает количество уникальных элементов. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает true, если множество пусто. |
| **capacity** | `capacity(self) -> usize` | Возвращает текущую внутреннюю емкость. |

### Утилиты

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_slot_occupied** | `is_slot_occupied(self, idx: usize) -> bool` | Проверяет, занят ли конкретный внутренний слот хеш-таблицы. |
| **val_at** | `val_at(self, idx: usize) -> Option<T>` | Возвращает значение из конкретного слота, если оно есть. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает буферы множества. |
| **Trait** | `impl Drop for Set` | Автоматически вызывает `free()`, когда множество выходит из области видимости. |
