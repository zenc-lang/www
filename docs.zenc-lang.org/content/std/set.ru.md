# std/set

`Set<T>` - это реализация универсального множества (hash set) для хранения уникальных значений типа `T`. Она использует хеш-таблицу с открытой адресацией и линейным пробированием.

## Обзор

- **Универсальность**: Хранит любой тип `T`.
- **Уникальность**: Автоматически обрабатывает дубликаты; добавление существующего элемента возвращает `false`.
- **Скорость**: Средняя временная сложность O(1) для добавления, удаления и поиска.
- **RAII**: Реализует трейт `Drop` для автоматического управления памятью.

## Использование

```zc
import "std/set.zc"

fn main() {
    let s = Set<int>::new();
    
    s.add(10);
    s.add(20);
    s.add(10); // Дубликат, возвращает false
    
    if (s.contains(10)) {
        println "Множество содержит 10";
    }
    
    s.remove(20);
    println "Длина: {s.length()}";
} // s освобождается здесь автоматически
```

## Определение структуры

```zc
struct Set<T> {
    data: T*;
    len: usize;
    cap: usize;
    // ... внутренние поля
}
```

## Методы

### Конструирование

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **new** | `Set<T>::new() -> Set<T>` | Создает новое пустое множество. |

### Модификация

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **add** | `add(self, val: T) -> bool` | Добавляет значение в множество. Возвращает `true`, если добавлено, `false`, если уже присутствует. |
| **remove** | `remove(self, val: T) -> bool` | Удаляет значение из множества. Возвращает `true`, если оно присутствовало и было удалено. |
| **clear** | `clear(self)` | Удаляет все элементы из множества без освобождения выделенной памяти. |

### Доступ и запросы

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **contains** | `contains(self, val: T) -> bool` | Возвращает `true`, если значение существует в множестве. |
| **length** | `length(self) -> usize` | Возвращает количество уникальных элементов. |
| **is_empty** | `is_empty(self) -> bool` | Возвращает `true`, если в множестве нет элементов. |
| **capacity** | `capacity(self) -> usize` | Возвращает текущую внутреннюю емкость. |

### Утилиты

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **is_slot_occupied** | `is_slot_occupied(self, idx: usize) -> bool` | Проверяет, занять ли конкретный внутренний хеш-слот. |
| **val_at** | `val_at(self, idx: usize) -> Option<T>` | Возвращает значение в конкретном внутреннем слоте, если оно есть. |

## Управление памятью

| Метод | Сигнатура | Описание |
| :--- | :--- | :--- |
| **free** | `free(self)` | Вручную освобождает внутренние буферы множества. |
| **Trait** | `impl Drop for Set` | Автоматически вызывает `free()`, когда множество выходит из области видимости. |
