# std/sys/user

`std/sys/user` 模块提供用于检索有关系统用户和组的信息的工具。

## 使用

```zc
import "std/sys/user.zc"

fn main() {
    // 获取当前用户 ID
    let uid = User::get_uid();
    println "当前 UID: {uid}";
    
    // 按名称查找用户
    match User::from_name("root") {
        Some(u) => println "找到 root，家目录: {u.home}",
        None => println "未找到用户"
    }
}
```

## User 结构体

| 字段 | 类型 | 描述 |
| :--- | :--- | :--- |
| **uid** | `u32` | 用户 ID。 |
| **gid** | `u32` | 主要组 ID。 |
| **name** | `String` | 用户名。 |
| **home** | `String` | 家目录路径。 |
| **shell** | `String` | 默认 Shell 路径。 |

## 方法

| 方法 | 签名 | 描述 |
| :--- | :--- | :--- |
| **get_uid** | `User::get_uid() -> u32` | 返回当前进程的实际用户 ID。 |
| **from_uid** | `User::from_uid(uid: u32) -> Option<User>` | 根据 UID 查找用户信息。 |
| **from_name**| `User::from_name(name: char*) -> Option<User>` | 根据用户名查找用户信息。 |
