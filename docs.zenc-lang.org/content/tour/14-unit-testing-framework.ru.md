# 14. Библиотека для юнит-тестирования


В Zen C есть встроенная библиотека для тестирования, которая позволяет писать юнит-тесты прямо в исходных файлах с помощью ключевого слова `test`.

#### Синтаксис
Блок `test` содержит описательное имя и тело кода для выполнения. Тесты не требуют функции `main` для запуска.

```zc
test "unittest1" {
    "Это юнит-тест";

    let a = 3;
    assert(a > 0, "a должно быть положительным целым числом");

    "unittest1 пройден.";
}
```

#### Запуск тестов
Чтобы запустить все тесты в файле, используйте команду `run`. Компилятор автоматически обнаружит и выполнит все блоки `test` верхнего уровня.

```bash
zc run my_file.zc
```

#### Утверждения (Assertions)
Используйте встроенную функцию `assert(condition, message)` для проверки ожиданий. Если условие ложно, тест не будет пройден и будет выведено указанное сообщение.
